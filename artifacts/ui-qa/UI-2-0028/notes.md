# Visual QA Notes: UI-2-0028 Icon Candidate

## 基本資料

- 候選稿：`unitinfo_type_icon_spear_v1.png`
- 比對時間：2026-04-01
- 比對來源：`docs/UI品質參考圖/` 裁切區塊
- 狀態：🟡 NEEDS REFINEMENT

## 主要差異

| 項目 | 結果 | 說明 |
|---|---|---|
| 材質層次 | 🟡 偏弱 | 候選稿表面太平，少了參考圖常見的金屬、琺瑯、舊化層次。 |
| 飽和度與亮部 | 🟡 偏乾淨 | 藍色與金色都太均勻，缺少暖色偏移、磨耗、邊緣髒污。 |
| carrier 語言 | 🟡 不夠厚重 | 外框圓盤過於簡潔，和參考圖中帶裝飾、厚邊、浮雕感的載體差距明顯。 |
| 小尺寸可讀性 | 🟡 有風險 | 縮到 `32x32` 時，交叉槍柄與紅點中心不夠強，遠看會糊成一團。 |

## 為何這次判定不夠

1. 這張圖已經有「功能 icon 可量產」的基線，但還沒有到參考圖那種正式商業化 UI 的質感。
2. 參考圖的 icon 不是只有符號本體，而是連外框、材質、光澤、厚度一起構成辨識語言。
3. BattleScene 真正上畫面後常常會縮小，所以 `32x32` 的辨識度比單看 `128x128` 更重要。

## 下一步建議

1. 做 `UI-2-0032` 的 `v2` refinement。
2. 優先加強外框厚度、材質深度、舊化與亮暗對比。
3. 主 glyph 要更粗、更集中，避免縮小後辨識度掉太多。
