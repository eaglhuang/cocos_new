# 血脈命鏡 v3 評估與可行性

## 結論

`突破頁 v3` 已達到可正式回寫規格的門檻。

它不是日常資訊頁，而是非常適合收斂為：

1. `血脈命鏡 Loading Screen`
2. `英靈升星 / 二轉突破頁`
3. `未持有名將展示頁`

## 本輪成立的點

1. 左右雙面命格已成立。
2. 中央命鏡已明顯像古老命格之門，而非純 fantasy portal。
3. 定心者位置與尺寸合理，已具儀式核心感。
4. 下方 5 格故事帶已能讀出 `人生 -> 危機 -> 覺醒 -> 延續` 的節奏。
5. 整體具備國際可轉版潛力，因骨架是 universal 的雙面命格結構。

## 相較 v2 的進步

### 右側英靈更專屬

v2 仍偏 generic 靈魂武將，v3 已更接近「趙雲本人之血脈面」。

### 中央命鏡更收斂

v3 中央視覺比 v2 更像古鏡裂隙，少了一點 generic 魔法門味。

### 故事帶更適合 loading

v3 的 5 格節奏清楚，放入 Loading / 覺醒場景都更自然。

## 後續 v4 仍可修的點

1. 右側英靈仍可再增加少量命紋辨識，不必大改。
2. 中央門內紋理還能再往 `古鏡 / 祖紋 / 靈門裂縫` 靠一步。
3. 底部第 4 格可再更明顯像「血脈接觸點」。

## 元件與實作可行性

### 目前好做

- 靜態雙人物
- 中央門主圖
- 定心者圖層
- 5 格故事縮圖
- CTA / loading 文案 / 名稱列

### 需要中度前端處理

- 中央門淡光
- 定心者 aura
- 右側英靈透明與外發光

### 不建議現在硬做

- 重型粒子門
- 故事帶長距離自動平移
- 角色或英靈複雜動畫

## AI 生產管線意義

這張 proof 非常適合當成：

1. `結構線稿圖母型`
2. `slot 標註母型`
3. `AI 角色替換模板`
4. `loading / 覺醒共用 layout 骨架`

原因是它具備：

- 主視覺位置穩定
- slot 邊界清楚
- 文字與故事區塊獨立
- 可分離為多個固定節點
