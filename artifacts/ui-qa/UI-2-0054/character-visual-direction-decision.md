# 角色視覺方向決策 v1

## 1. 一句話定案

本專案角色視覺應採「半寫實三國國風 + 歷史奇幻英雄化」方向；戰場以 `3D 模型` 承擔主表現，`2D 立繪` 則承擔辨識、情緒、收藏與商業化畫面的主語言。

## 2. 為什麼不是戰場直接用 2D 立繪

現有 [BattleScene.ts](C:\Users\User\3KLife\assets\scripts\battle\views\BattleScene.ts) 與 [UnitRenderer.ts](C:\Users\User\3KLife\assets\scripts\battle\views\UnitRenderer.ts) 已經是完整的 3D 戰場思維，包含：

- 棋盤世界空間、斜角相機、單位 tween、投影與 world-space label
- hero / troop prefab 載入與陣營、兵種、站位綁定
- 真戰場 HUD 與場景背景的疊合

在這個架構下，若把敵我武將直接改成 2D 立繪主表現，會出現三個問題：

1. 戰場空間感會被削弱，整體觀感容易退回「UI 貼圖疊在棋盤上」。
2. 移動、衝鋒、受擊、朝向與距離判讀會變差。
3. 已有 3D 資產與之後的 capture QA 流程價值會被吃掉。

結論不是「不要做立繪」，而是「不要讓立繪取代戰場本體」。

## 3. 戰場與 UI 的主從關係

在進一步定方向前，要先承認目前的製作現實：

- 沒有 2D Spine 人物動畫量能
- 沒有 3D 角色動畫量能
- 可以穩定生產的是 `2D 靜態繪圖` 與 `3D 靜態 AI 建模`
- 特效目前只有一套原生 Cocos 3D 粒子較完整，其餘多數仍停在 Unity 免費素材轉換中，且品質尚未達標

這代表我們的畫面策略不能建立在「大量角色動態表演」或「高密度精緻特效」之上，而要建立在 `靜態輪廓、材質、構圖、光影與 UI 敘事`。

### 3.1 戰場主語言

- 主戰場敵我武將：`3D 靜態 hero model`
- 主戰場兵種單位：`3D 靜態 troop formation`
- 戰場情緒高點：優先靠 `3D 輪廓 + lighting + 鏡頭構圖 + world-space UI`，特效只做點綴，不拿來當主支柱

### 3.2 2D 立繪主語言

- `BattleHUD` 小裁片頭像
- `GeneralQuickView` 與 `GeneralDetail`
- `TigerTally` / support card / 收藏卡
- `Shop / Gacha / 活動 Banner`
- 劇情 cut-in、招募結果、角色獲得頁

Unity 對照：戰場 3D 模型在目前階段更接近「高質感靜態 Prefab 擺位 + 鏡頭演出」，而不是完整角色動畫控制器；2D 立繪則是角色資料頁、抽卡頁與 HUD avatar 的高精度 key art。兩者不是二選一，而是不同層級的資產分工。

## 4. 人物立繪的正式風格

### 4.1 視覺北極星

- 類型：半寫實、英雄化、非 Q 版、非日系校園動畫感
- 世界觀：三國歷史基底 + 國風戰場奇幻感
- 質感：布料、皮革、金屬、毛髮都要看得出材質，不走平塗塑膠感
- 氣質：成熟、威儀、銳氣、帶故事感，不走偶像式甜美濾鏡
- 受眾平衡：主體仍服務 `30~55 歲男性三國情懷族群`，但要用較乾淨、較俊朗、較現代的五官與輪廓，避免整體過老
- 懷舊參考：要保有早期三國歷史遊戲那種一眼可辨的懷舊武將感，但不能停留在舊世代厚塗的距離感

### 4.1.1 本輪樣本判讀

以 `C:\Users\User\Pictures\cocos專案\人物\` 的目前樣本來看：

- 趙雲立繪方向最接近建議北極星：有歷史武將輪廓，也有乾淨俐落的動漫化整理
- 關羽 / 張飛部分版本更接近傳統史詩寫實感，懷舊味足，但有些圖的電影感與動態特效偏重，不適合直接當量產母型
- 呂布目前樣本的輪廓、甲冑與黑紅金配色很適合做高辨識敵將，但渲染語言略偏乾淨卡面，需避免與較寫實的關羽系樣本分裂成兩個世界

因此正式風格應收斂為：

- `懷舊三國歷史感 70%`
- `現代動漫化修飾 30%`

不是把角色變成強二次元，而是把老派三國武將圖做得更好入口、更有親和力。

### 4.2 造型規則

- 輪廓要先能一眼分辨人物，再談細節。
- 每位武將必須有明確主武器、頭肩輪廓、披掛節奏與陣營辨識。
- 不追求過度寫實到接近照片；保留可量產的英雄化筆觸與戲劇光影。
- 臉部五官可精緻，但不能過度年輕化、白淨化或網紅化。
- 男性主將可更俊朗，但鬍鬚、年齡感、威嚴感不能被完全磨掉。
- 女性角色可更精緻，但不能直接滑向純仙俠美女卡或偶像卡。

### 4.3 色彩規則

- 以低到中飽和為主，重要角色可用一個高辨識 accent 色。
- 金屬與飾件可亮，但服裝大面積不應螢光化。
- 陣營色要藏在披風、內襯、飾條、旗幟感元素中，不要整套大面積純色制服。

### 4.4 光影規則

- 優先使用斜側主光，讓臉、胸甲、武器和肩部形成英雄輪廓。
- 背景需留足裁切空間，不能把頭肩後方塞滿複雜裝飾。
- 不做過強逆光導致小尺寸裁片只剩黑影。
- 不要長期依賴大面積戰場塵爆、刀光弧線或電影式特效去撐氣勢，因為這會和目前低特效依賴策略相衝突。

- 構圖：頭肩到胸上，五官清楚，武器只露局部

- 核心：強化角色魅力、身份與材質層次

### Diorama / card art

- 用途：`TigerTally`、活動卡、支援卡
- 構圖：角色與場景關係、事件感、戰術感
- 核心：比單純半身像更有故事與商品價值

## 6. 現階段禁止方向

- 不要把戰場將領退回成 2D 大頭貼站在地板上。
- 不要把立繪做成高飽和、年輕偶像感過重的日系卡牌風。
- 不要讓 HUD 小頭像直接拿全身立繪硬裁，導致五官糊掉。
- 不要讓 General 詳情、BattleHUD、TigerTally 三處各自採不同畫風。

## 7. 對現有專案的直接決策

1. `assets/resources/units/troops/heroes/` 與既有 troop prefab 應繼續作為戰場主資產線。

3. 因為沒有角色動畫量能，戰場角色品質要優先靠 silhouette、材質、比例、站姿、鏡頭與 UI overlay 補強，而不是假設後續會靠動畫救畫面。

5. `UI-2-0039` 應以 `A4/A5` 的 card art 視角補強 TigerTally，而不是回頭把戰場角色改成立繪。
6. 特效目前屬於短板，戰場與主畫面設計應先採「低特效依賴」策略，等 Cocos 粒子管線穩定後再做第二階段加分。

## 8. 專案長短板判斷

### 目前長板

- 2D 靜態繪圖可作為角色魅力與商品感的主要來源
- 3D AI 建模可穩定供應戰場人物與建築靜態表現
- 3D 靜態模型可搭配較強的 AI shader，補出材質、輪廓、描邊與展示感
- UI 規格、screen contract、QA artifact 正在形成正式流程

### 目前短板

- 角色動態演出不足，無法靠 Spine 或 3D 動畫堆疊情緒
- 特效轉換鏈尚未成熟，Unity 粒子搬運到 Cocos 後精緻度下滑
- 戰場高光時刻目前不能依賴大量技能演出或華麗粒子撐場

### 因此的策略

- 角色魅力主要靠立繪、卡面、裁片與 UI 呈現建立
- 戰場精緻感主要靠靜態模型質感、shader、場景構圖、地面陰影、鏡頭與資訊層級建立
- 特效只先保留必要命中感、buff feedback、重點技能點綴，避免全面鋪開

## 10. 女武將風格決策

女武將不需要像男武將那樣把歷史厚重感壓到最滿，但也不能直接滑向純仙俠偶像卡。她們的訴求確實不同，賣點更偏向外型、氣質、辨識與收藏慾，但仍必須站在同一個三國世界裡。

### 10.1 正式方向

- 基底：`古典國風 60%`
- 修飾：`二次元美感 40%`
- 核心：比男武將更柔、更美、更好入口，但不能脫離三國懷舊感

### 10.1.1 這條商業路線能不能成立

可以成立，但要守住兩個底線：

1. 女角即使更偏男性向，也必須仍是「這個世界裡的人」，不是現代偶像穿古裝。
2. 對外呈現上，最好把女角定位成後宮權力核心、女教官、女官、謀士、名門女性、舞姬等有功能與身份的人，而不是只剩戰利品描述。

也就是說，`男武將吃歷史戰場感`、`女武將吃現代男性審美` 這件事本身是可行的；真正要避免的是讓兩者像兩個不同產品硬拼。

### 10.2 造型原則

- 臉部可以比男武將更精緻、更理想化，但不要變成純現代網紅臉
- 服裝可更飄逸、更華美，但必須保有漢末 / 三國感的布料、甲片、飾件語言
- 可以適度性感，但性感應來自氣質、剪裁與姿態，不應只靠過度暴露
- 女武將之間也要分 archetype：貴氣型、舞姬型、女將型、謀士型，不能全部都長成同一張臉

### 10.2.1 現代妝容要怎麼放進去

你提的「現代妝容，例如韓式眼妝 + 古代服裝 + 女曲線鎧甲」是有市場潛力的，但要這樣收：

- 臉：`65% 現代審美`
- 服裝輪廓：`70% 古代 / 國風`
- 材質與飾件：`80% 三國世界語言`

可以用的現代化元素：

- 更乾淨的眼妝與眼尾線條
- 更講究的睫毛、臥蠶、唇色與膚質
- 更精修、但不過頭的五官比例

不能過頭的地方：

- 不要變成現代美妝廣告臉
- 不要讓眉、唇、鼻影、打亮一眼就像現代棚拍模特
- 不要讓髮型完全失去古裝與階級感

也就是說，現代審美應該藏在「修飾方式」，而不是直接蓋掉世界觀。

### 10.2.2 女曲線鎧甲怎麼做才不會掉價

女曲線鎧甲可以做，而且很適合你這條路線，但要把它做成：

- `儀式型鎧甲`
- `展示型鎧甲`
- `輕戰型女將鎧甲`

不建議：

- bikini armor
- 只有胸腰腿、沒有階級、材質與結構
- 現代性感服飾直接套古代花紋

建議：

- 胸甲、肩甲、裙甲、腰封仍有結構與身份語言
- 曲線感主要來自收腰、開衩、布料層次與姿態
- 華麗感更多靠金屬雕紋、珠玉、披帛、髮飾與色票，不只靠裸露
