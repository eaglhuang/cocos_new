# General Detail V3 Evaluation And Feasibility

## 評估結論

這張 `日常人物頁 v3` 已達到可正式回寫規格的門檻，可作為目前 `GeneralDetail` 的視覺母型。

本次 proof 的關鍵成立點：

1. `人物仍是主角`
2. `血脈區已從附加資訊升級為深層人格區`
3. `六格故事條成立`
4. `UI 具有國際化彈性`

## 與 v2 相比的正式提升

1. 右側 bloodline 區不再像另一張外掛卡。
2. 命紋徽記已比單純獸首 icon 更接近系統化符號。
3. 故事條重要性被抬升，符合《三國傳承》角色內容層定位。

## 尚可留到 v4 的精修點

1. 祖獸符印可再從單一圓徽擴成更完整的命紋家徽系統。
2. 故事第 5、6 格仍可再更強化血脈真相與命運延續。
3. 右側血脈區的背景命紋還能再更細緻，避免看起來過於平面。

## 依目前條件的實作可行性

### 易於落地

1. 左人物 / 右資訊 / 下故事帶版型，與現有 JSON UI pipeline 高度相容。
2. 六格故事帶是固定 slot thumbnail 結構，不依賴高成本動畫。
3. 命紋徽記可先以靜態 sprite / badge family 落地。

### 中度成本

1. 角色身上的血脈裝飾語言，需要與角色生圖流程同步收斂。
2. 血脈區的 background motif / crest family 仍需補齊整套資產。

### 暫不建議混入日常頁

1. 長駐血脈虛影
2. 大量發光動效
3. 自動捲動式故事跑馬燈

## 總結

`日常人物頁 v3` 已可作為正式規格母型。

以你們目前條件來看，這一頁是 `好拆、好填、好量產` 的，真正要慢慢補強的是命紋徽記 family、六格故事圖 production pipeline 與血脈區視覺一致性。
