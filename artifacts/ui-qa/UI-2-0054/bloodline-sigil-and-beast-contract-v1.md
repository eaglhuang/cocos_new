# 祖紋命篆與命紋靈獸視覺契約 v1

## 定位

本文件定義《三國傳承》血脈世界觀的第一套正式視覺母語，供以下模組共用：

- 血統 UI
- 日常人物頁
- 血脈命鏡 / 覺醒頁
- 血脈卡 / 英靈虎符
- 虎符特種軍隊
- 角色立繪服裝裝飾

核心原則是：
`懷舊只是入口，祖紋命篆才是主體。`

## 核心原則

### 1. 不走科技基因

- 不使用實驗室、DNA 雙股、HUD、掃描線、科幻數據層作為主視覺語言。
- 基因感應被翻譯為 `古老命紋 / 血脈印輪 / 祖靈痕跡`。

### 2. 不走重宗教與恐怖

- 不使用邪神、惡魔、腐化裂臉、血腥寄生。
- 命紋面應是 `古老、神秘、宿命、貴氣`，不是恐怖污染。

### 3. 不做可愛寵物

- 命紋靈獸不是萌系吉祥物，也不是 sidekick。
- 它是 `血脈象徵物 / 祖靈獸印 / 命格化身`。

### 4. 先有 family，再有武將專屬變體

- 先建立共用的靈獸 family 與因子紋樣規則。
- 再針對趙雲、甄姬、孫尚香等名將做專屬細化，避免每人都從零發明一套世界。

## 六色因子視覺母語

### 綠系

- 語意：生機、仁厚、延續、守護、軍勢根性
- 紋樣：葉脈、蔓生、玉石裂脈、弧形生長線
- 材質：溫玉、青石、舊銅綠
- 靈獸傾向：鹿、麟、木蛇、長角草獸

### 藍系

- 語意：智略、觀測、推演、冷靜、戰法
- 紋樣：水紋、旋脈、流線、鏡面星點
- 材質：青玉、冷銀、霧玻璃感礦石
- 靈獸傾向：龍、鶴、魚龍、鏡鰭靈獸

### 紅系

- 語意：血性、體魄、爆發、烈性、武威
- 紋樣：斷裂火紋、斜向裂脈、獠牙線
- 材質：赤金、焦銅、熔鐵、獸骨光澤
- 靈獸傾向：虎、狼、焰獅、角獸

### 白系

- 語意：天賦、純度、靈性、祝福、上限
- 紋樣：環紋、羽紋、薄霧光圈、鏡白印記
- 材質：銀白、白玉、月石、冰金
- 靈獸傾向：白狐、雪鶴、羽鹿、鏡獸

### 粉系

- 語意：魅力、連結、親和、情緒感應、群體影響
- 紋樣：花瓣流線、珠母紋、柔弧纏繞線
- 材質：珠玉、柔金、絲緞、母貝
- 靈獸傾向：蝶鹿、花狐、羽雀、柔角靈獸

### 紫系

- 語意：宿命、異能、禁忌、稀有覺醒、因果牽引
- 紋樣：雙環裂篆、鏡裂紋、陰影交纏、殘筆命輪
- 材質：紫銅、古鏡、夜玉、暗曜石
- 靈獸傾向：夜龍、影狼、鏡蛇、冥鹿

## 命紋靈獸 Family v1

### 1. 龍系

- 用途：智略、名將、覺醒、貴種血統
- 代表氣質：高階、統攝、命運主導

### 2. 虎系

- 用途：武威、軍魂、強攻、震懾
- 代表氣質：壓迫、衝陣、血勇

### 3. 鹿 / 麟系

- 用途：仁德、國運、延續、護佑
- 代表氣質：穩定、祥瑞、承繼

### 4. 鶴 / 羽獸系

- 用途：靈性、智性、純度、天賦
- 代表氣質：清冷、遠觀、上乘氣質

### 5. 狼 / 焰獸系

- 用途：體魄、野性、獵性、極限爆發
- 代表氣質：危險、機動、侵略

### 6. 鏡獸 / 蛇系

- 用途：宿命、異能、因果、命格投影
- 代表氣質：神秘、內向、潛伏、雙面性

## 祖紋命篆規則

- 字形方向接近 `金文 / 篆意 / 帛書殘筆 / 玉刻磨損`
- 要有高級神秘感，但不能直接照抄佛經梵字或宗教真字符
- 可作為：
  - 胸甲核心命紋
  - 腰封印記
  - UI badge
  - 命鏡裂隙周圍符號
  - 血統樹節點印記
  - 虎符兵牌烙印

### 禁止事項

- 科技 HUD glyph
- 霓虹魔法陣
- 太密太碎的小字牆
- 看起來像現成宗教經文或召喚陣

## 血脈 Crest 規則

一個正式血脈徽記由三層組成：

1. `靈獸輪廓抽象化`
2. `對應因子紋樣融合`
3. `中心命篆核`

這代表徽記不能只是單顆獸頭，也不能只是單一圓章。

## 畫面濃度對應

### 日常人物頁

- 低濃度
- 以 `胸甲 / 腰封 / badge / 背景淡紋` 為主

### 血脈命鏡 / 覺醒頁

- 高濃度
- 可出現 `命鏡裂隙 / 血脈虛影 / 大型 crest / 命篆浮現`

### 血統 UI

- 中濃度
- 以 `命盤 / 家譜印輪 / 祖紋連線 / 節點徽記` 為主

## 國際版相容性

本契約的骨架不是「中國古風專用」，而是以下幾個全球可讀概念：

- mirror self
- inherited self
- ancient sigil
- beast crest
- lineage motif

未來若要做國際版，可替換：

- 色盤
- 邊框材質
- 人物畫風
- 局部紋樣語言

但不需重做整個版型與世界觀骨架。
