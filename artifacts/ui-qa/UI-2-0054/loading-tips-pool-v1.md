# Bloodline Mirror Loading Tips Pool v1

## 目標

建立 `血脈命鏡 Loading` 可直接量產使用的 tips 文案池，讓過場圖不只好看，也能持續：

- 教玩法
- 補世界觀
- 刺激抽卡慾望
- 強化角色收藏感

## 使用原則

1. 每次 loading 顯示 `1~2` 條 tips。
2. 同一畫面不混入超過兩種語氣。
3. tips 文案長度以 `18~34` 字為優先。
4. 若畫面主角為未持有武將，可提高 `角色誘惑類` 權重。

## 分類

### A. 系統提示

1. `同名名將可選擇復生重練，或灌注為更強的英靈血脈。`
2. `血脈命鏡不只顯示傳承，也預示武將未來的覺醒方向。`
3. `英靈虎符越強，對應的特種軍隊在戰場上的存在感越高。`
4. `六色因子不只影響數值，也決定祖紋命篆與靈獸傾向。`
5. `角色頁中的血脈性格，會影響後代的成長傾向與劇情氛圍。`
6. `未持有武將仍可在命鏡中現影，先看見命格，再決定是否招募。`

### B. 歷史趣聞

1. `趙雲的七進七出，不只是膽識，也是血脈面對危局時的本能覺醒。`
2. `諸葛亮六出祁山的執念，在血脈傳聞裡被解讀為宿命未竟。`
3. `劉備的魅力不只來自仁德，也來自能牽引人心的稀有血統。`
4. `孫尚香的任性與英氣並存，常被視為吳地烈脈與王族傲性的混合。`
5. `甄姬的冷艷與疏離，在古老傳聞中被稱為洛水血脈的殘響。`
6. `關羽的威儀並非單靠武勇，更多人相信那是祖魂守節的外顯。`

### C. 血脈傳聞

1. `每位名將心中都同時住著表象面與血脈面，只有定心者能看清兩者。`
2. `命紋靈獸不是寵物，而是古老血脈以象徵形態留下的伴靈。`
3. `血脈卡不是殘骸，而是名將一生意志凝成的可繼承之魂。`
4. `當命鏡兩側開始互相滲透，代表武將已接近真正的覺醒門檻。`
5. `英靈不死，只是褪去人身，以血脈與軍勢延續在後世。`
6. `越是高階的英靈虎符，越能喚回更完整的血脈面與軍勢殘響。`

### D. 角色誘惑

1. `尚未結識的名將，往往最先在命鏡中留下令人難忘的輪廓。`
2. `有些武將不必先擁有，只看命鏡中的一眼，就足以讓人想抽。`
3. `未納入麾下的英靈，只能在命鏡中現影；真正的力量仍待你招募。`
4. `當一位名將的血脈面開始清晰，通常意味著他值得被長期培養。`
5. `命鏡中最安靜的那道身影，往往是最不該錯過的血脈核心。`
6. `還沒得到的武將，也會先在命鏡裡告訴你他是怎樣的人。`

## 推薦權重

### 已持有角色 Loading

- 系統提示：`35%`
- 歷史趣聞：`20%`
- 血脈傳聞：`35%`
- 角色誘惑：`10%`

### 未持有角色 Loading

- 系統提示：`20%`
- 歷史趣聞：`15%`
- 血脈傳聞：`25%`
- 角色誘惑：`40%`

## 配對建議

### 人物頁相關 loading

- 系統提示 + 血脈傳聞

### 覺醒 / 升星 loading

- 血脈傳聞 + 歷史趣聞

### 未持有輪播 loading

- 角色誘惑 + 血脈傳聞
