# 全專案美術品質優先序 v1

## 1. 核心判斷

如果目標是讓整個專案的觀感「大大加分」，現在最有槓桿的不是再補零散 icon，而是先把角色、戰場、卡面與主畫面的高辨識視覺語言鎖定。角色是玩家最容易記住的東西，戰場則是最容易暴露品質天花板的地方。

但這個優先序必須建立在目前真實產能上：

- 有：`2D 靜態畫圖`
- 有：`3D AI 靜態建模`
- 有：`3D 靜態模型可搭配強 shader 提升質感`
- 沒有：`2D Spine 人物動畫`
- 沒有：`3D 角色動畫`
- 弱項：`Cocos 特效管線與 Unity 粒子轉換品質`

所以我們的加分策略要偏向 `靜態高質感`，而不是假設後面能靠大量動態演出補回來。

## 2. 優先序總表

### P0. 角色視覺北極星與 family 統一

- 影響面：BattleHUD、GeneralDetail、QuickView、招募、商店、卡片、活動
- 為什麼最優先：角色畫風一旦漂移，整個專案會立刻像拼裝包
- 本輪決策：已由 `UI-2-0054` 先定下 `懷舊三國歷史感 70% + 現代動漫化修飾 30%`，並維持 `3D 戰場 / 2D UI` 分工

### P1. BattleScene 真場景品質升級

- 內容：世界空間 UI、3D 英雄 silhouette、地面陰影、鏡頭、場景色調、低量但準確的關鍵特效
- 為什麼高優先：戰場是專案最像正式遊戲、也最容易被一眼評分的地方
- 直接依賴：`UI-2-0046` 補齊真場景 capture 後，才能做 placement QA 與反覆校正

### P1.5 主城 / 家園 / 國家治理地圖的 3D 景物語言

- 內容：主城建物、國家地圖節點、景物、城牆、地標、資源點與環境 shader 語言
- 為什麼高優先：這類系統對靜態 3D 建模非常友善，而且正好命中你們目前的長板
- 建議：把這塊視為未來中期很有機會拉開差距的系統，不一定要等動畫成熟才做

### P2. Portrait pipeline 正式量產

- 內容：A2 HUD portrait、A5 hero panel、裁切規則、badge 安全區、proof board
- 為什麼高優先：這會同時提升戰場 HUD、武將頁、收藏與商業化畫面
- 直接相關卡：`UI-2-0038`、`UI-2-0043`、`UI-2-0029`

### P3. TigerTally / 支援卡 / 招募卡面的商品感

- 內容：A4 diorama、A5 hero panel、rarity、badge、材質節奏
- 為什麼高優先：這是最接近「有商品價值」與「玩家想收藏」的地方
- 直接相關卡：`UI-2-0039`、`UI-2-0030`

### P4. 主畫面 style profile 全覆蓋

- 內容：Lobby、General、Shop、Gacha、Battle 的專屬 profile
- 為什麼高優先：能避免所有主畫面共用同一套 tone，導致缺少場景辨識
- 直接相關卡：`UI-2-0053`

### P5. Reward / bundle / chest / material pack 的高級感

- 內容：容器、箱體、卷軸、禮包、素材箱的 family kit
- 為什麼高優先：這些是商城與獎勵回饋的情緒出口，很影響「值不值得拿」
- 直接相關卡：`UI-2-0040`

### P6. 陣營色、環境色與 lighting 一致性

- 內容：蜀 / 魏 / 群等勢力的 accent、戰場時段、背景與 UI 的互相讓位
- 為什麼高優先：可以把同一張畫面從「湊在一起」推到「像同一世界」
- 目前狀態：已有 BattleScene style profile 雛形，但還未橫向覆蓋全專案

### P7. 特效管線補強，但列為第二階段

- 內容：Cocos 粒子語言、Unity 粒子轉換品質、常用技能與 hit feedback 的 Cocos 化
- 為什麼不是更前面：目前是短板，但短時間內很難變成主賣點，先把畫面骨架做好更有感
- 策略：先做「少量關鍵特效可用」，不要一開始就追求全量華麗還原

## 3. 你現在最該先做什麼

### 第一批，最有感

1. 定案人物立繪風格，不再讓後續 portrait proof 漂移。
2. 確認戰場繼續以 `3D 靜態英雄 / 小兵` 為主，不回頭改成 2D 站樁。
3. 補出 BattleScene 真場景 capture，開始做真正的 placement QA。

### 第二批，最能讓玩家覺得精緻

1. 做 A2 HUD portrait proof。
2. 做 A5 hero panel proof。
3. 做 TigerTally / 招募卡面 proof。
4. 做小兵 family compare board，先鎖「可愛半寫實」的正式比例與質感。

### 第三批，最能拉開商業感

1. 做 reward container / bundle props。
2. 做 Shop / Gacha 專屬 style profile 與 banner 語言。
3. 做稀有度與卡面材質節奏統一。
4. 補一套真正能在 Cocos 內穩定使用的關鍵粒子特效 baseline。
5. 規劃主城 / 家園 / 國家治理地圖的 3D 建物與景物語言。

## 4. 戰場 3D 與立繪的投資比例建議

- 戰場主體：`55%` 投資在 3D 靜態模型、鏡頭、lighting、world-space readability、必要特效
- 角色魅力與變現畫面：`45%` 投資在 portrait、hero panel、card art、招募呈現

這個比例的意思不是資源平均分，而是先承認戰場是基礎體感，立繪是情緒與商業化放大器。因為目前沒有角色動畫量能，所以立繪權重要比原先再高一點，用來補足角色魅力與高光畫面。

## 5. 若只能選三件事先衝

1. `BattleScene` 真場景 capture + placement QA
2. `A2 / A5` 角色立繪 proof 與 portrait pipeline
3. `TigerTally / 招募 / 收藏` 的 card art 商品化

## 6. 目前不建議優先投入的項目

- 先大量做次要 icon 微修，因為提升有限
- 先做過多零散 props，而主角色與戰場語言還沒定
- 先把每個畫面都塞滿特效，會掩蓋真正的材質與層級問題
- 先賭後面一定能把 Unity 粒子完美轉進 Cocos，這風險太高
- 先讓主將和小兵各走各的畫風，會直接削弱世界觀一致性

## 7. 對 Agent1 / Agent2 的拆工建議

- Agent2：持續扮演美術總監，負責 portrait / card / style profile / QA board 規格與 review-round
- Agent1：承接 runtime、capture、proof 接線、preview host、screen-context placement 驗證，以及關鍵粒子 baseline 的可用化

Unity 對照：Agent2 比較像在做 Art Bible 與 QA gate；Agent1 比較像在做 Prefab 落地、場景串接與實機驗證。

## 8. 市場 benchmark 與差異化建議

### 8.1 目前市場最紅三國手游立繪的共通點

從這批 benchmark 圖來看，主流市場已經高度收斂到以下語言：

- `半寫實 + 仙俠化國風` 的安全組合
- 男角大量採用俊美臉、亮面甲、金邊裝飾、斜向動態 pose
- 女角大量採用白皙皮膚、柔光臉、飄帶、花飾、胸腰腿強調與收藏卡感
- 卡面多半用高飽和暖底、紅橙金 rarity、濃厚發光框與數值養成標記堆滿
- UI 與角色之間的關係偏「卡牌抽收集」，而不是「世界觀真實存在的人」

### 8.2 這些圖為什麼會紅

- 第一眼很爽：角色都夠帥、夠美、夠華麗
- 手游感明確：稀有度、戰力、養成、抽卡爽感都很直接
- 成本效率高：這套風格容易量產，也容易維持商業包裝感

### 8.3 這套市場主流的問題

- 同質化非常嚴重，很多作品一眼看起來像同一家美術 outsourcing 團隊做的
- 男角容易全部變成「同一種帥」，只差衣服顏色
- 女角容易全部變成「同一種美」，只差髮型與道具
- 三國歷史記憶點會被仙俠 / 手游卡牌慣性吃掉，最後只剩「國風 fantasy 角色」

### 8.4 我們不應該怎麼做

- 不要直接抄成同一套亮面仙俠卡風
- 不要把所有角色都磨成手游標準俊男美女模板
- 不要把稀有度框、發光、火焰底圖當成主要辨識來源

### 8.5 我們可以怎麼拉開市場區隔

#### A. 用「懷舊三國辨識」拉開，而不是用更誇張的仙俠感硬拼

- 讓人第一眼就想到老派三國人物，而不是 generic 國風手游角色
- 保留光榮系、歷史系、名將系的影子，但把五官、材質與乾淨度做得更現代
- 這會讓你的核心 TA 一眼有情懷，同時又不會覺得太舊

#### B. 用「角色賣點分工」拉開，而不是人人都同一種帥

- 趙雲賣俊朗正氣
- 關羽賣威儀忠義
- 張飛賣猛與壓迫
- 呂布賣危險華麗
- 曹操賣權勢與統御

主流市場常把角色都收成單一模板，你反而應該讓每個角色更像一個被記得住的人。

#### C. 用「主將 / 女將 / 小兵 / 3D 場景」四條清楚分線拉開

- 主將：懷舊三國英雄化
- 女將：古典國風 + 男性向收藏感
- 小兵：可愛半寫實量產化
- 場景 / 建物：3D 靜態模型 + shader 質感

多數競品強在卡面，弱在世界；你們反而可以把世界與角色一起成立。

#### D. 用「3D 靜態世界質感」拉開

- 主城、家園、國家治理地圖、建物、地標非常適合成為你們的差異化強項
- 因為市場上很多作品 2D 卡面很強，但一進場景世界就很普通
- 你們有 3D 靜態模型與 shader 的長板，這是很值得用來拉開差距的點

#### E. 用「少而準的特效」拉開，而不是用廉價滿版特效

- 現階段不要跟市場拼滿屏火光、刀氣、粒子雨
- 反而該讓材質、姿態、輪廓與 UI 對位先站住
- 當畫面骨架更穩，少量關鍵特效反而更顯高級

### 8.6 最終差異化公式

如果要濃縮成一句話，我會建議你們的市場區隔是：

`不是更仙俠、更浮誇的三國手游，而是更有懷舊辨識、更有角色分工、且世界質感更完整的現代三國產品。`

這條線比直接跟市場主流拼「誰更亮、更美、更發光」更適合你們，也更有機會留下記憶點。
