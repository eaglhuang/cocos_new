# 英靈虎符與特種軍隊視覺契約 v1

## 定位

本文件將以下四件事視為同一套世界觀鏈條：

1. 名將本體
2. 血脈卡 / 英靈卡
3. 英靈虎符
4. 虎符特種軍隊

核心原則是：

`人死不滅，留下的是可被再喚起的血脈面。`

因此名將不是單純退場，而是轉為：

- 可插入的英靈卡
- 可裝備的英靈虎符
- 可召出的特種軍勢

## 正式世界觀鏈條

### 1. 名將本體

- 平時以 `表象面` 存在。
- 可在人物頁與命鏡頁看見 `血脈面` 痕跡。

### 2. 血脈卡 / 英靈卡

- 名將壽命盡後，保留下來的是 `英靈化的血脈卡`。
- 它不是一般素材，而是該名將靈魂與命紋的濃縮體。

### 3. 英靈虎符

- 血脈卡可以凝為 `英靈虎符`。
- 虎符不是普通軍令，而是名將軍魂的權柄載體。

### 4. 虎符特種軍隊

- 每張英靈虎符都能喚出對應的特種軍勢。
- 軍隊不只是兵種數值，而是該名將一生軍勢、戰法、血脈、命紋的殘響。

## 視覺對應規則

### 名將本體

- 角色身上必須已有對應的 `胸甲核 / 腰封核 / 肩扣 / 玉佩 / 武器掛飾`。
- 這些 slot 是英靈化後的視覺母本。

### 英靈卡 / 血脈卡

- 應保留該名將的 `人臉辨識 + 命紋面`。
- 卡面不是另一個陌生角色，而是名將被提煉後的血脈面形象。
- 可使用：
  - 半靈化
  - 全靈化
  - 雙面同框

### 英靈虎符

- 以 `血脈 crest + 靈獸輪廓 + 命篆核` 為主體。
- 不應只是一般虎符外型換個顏色。
- 虎符上要能一眼看出屬於誰、屬於哪條血脈、喚的是哪支軍勢。

### 特種軍隊

- 軍勢 banner、兵牌、盾徽、武器束帶、頭盔飾件，都要回收虎符 crest 語言。
- 軍隊不是 generic 特效兵，而是 `某名將留下的軍魂編制`。

## 視覺濃度分級

### 日常 UI

- 低濃度
- 顯示：
  - 虎符小 badge
  - 特種軍隊名稱
  - 血脈 crest 縮略圖

### 虎符 / 特種軍隊詳情

- 中濃度
- 顯示：
  - 英靈虎符主圖
  - 專屬 crest
  - 軍勢稱號
  - 名將來源

### 戰鬥插卡 / 覺醒演出

- 高濃度
- 顯示：
  - 名將血脈面短暫浮現
  - 命紋裂隙
  - 對應軍勢召出

## 視覺組成公式

每個英靈虎符 / 特種軍隊 family 應包含：

1. `來源名將辨識`
2. `命紋靈獸 family`
3. `血脈 crest`
4. `軍勢標記`
5. `稀有度 aura`

## 命名與敘事規則

### 卡面敘事

- 名將本體：現世名號
- 英靈卡：血脈面稱號
- 虎符：軍勢稱號
- 特種軍隊：兵團名 / 軍勢名

### 例

- 趙雲
- 銀龍血脈卡
- 銀龍英靈虎符
- 銀龍義從

## 稀有度與升階

### 星等表現

- 1~3 星：以數值增幅與 icon / aura 強化為主
- 4~5 星：開始允許英靈面與命紋面更明顯露出

### 覺醒方向

- 低星：看軍勢
- 高星：看血脈面與英靈面

這表示：

- 低階虎符可以先偏軍牌感
- 高階虎符則應偏英靈 relic 感

## 與 UI 的對應 slot

必須同步存在於：

- 人物頁血脈 crest 區
- 血脈命鏡 badge / marker
- 虎符詳情卡主 badge
- 軍隊詳情中的旗紋 / 兵牌 icon
- 戰鬥插卡顯示的 spirit crest

## Prompt / 產圖規則

### 英靈卡

- `same hero in bloodline-spirit form`
- `ancestral crest integrated into chest core`
- `spirit-tally relic framing`
- `not generic fantasy ghost`

### 英靈虎符

- `premium ancient military tally relic`
- `bloodline crest carved into the tally core`
- `ancestral beast motif fused with command insignia`
- `clearly linked to the hero's bloodline`

### 特種軍隊

- `elite troop banner using the same bloodline crest family`
- `unit armor and weapon details echo the hero's sigil slots`
- `not generic soldiers, but a named spirit troop lineage`

## 禁止事項

- 虎符只是普通道具 icon
- 特種軍隊與名將來源沒有任何視覺關聯
- 英靈卡只是人物立繪加藍色發光
- 一切只靠文字說明，沒有 crest / family / slot 對應
