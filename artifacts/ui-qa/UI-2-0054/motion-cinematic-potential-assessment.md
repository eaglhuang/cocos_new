# 動態演出潛力評估

## 1. 前提

目前專案的動態演出能力並不穩定，但也不是完全沒有牌可打。

現況可分成三種：

- 沒有穩定的 `2D Spine` 人物動畫
- 沒有穩定的 `3D 角色動畫`
- 有一套不穩定但可生成短片的影片生成器，可產出 `5~10 秒` 左右的技能 / 奧義影片

另外，還有一條非常有潛力的補線：

- `漫畫劇情式插圖 / 分鏡插畫`

這兩條其實很適合拿來填補你們目前沒有角色動畫的缺口。

這個時長非常重要，因為它代表它們更像：

- 奧義 `cut-in / 全屏技能短片`
- 招募高光
- 劇情節點過場

而不像完整戰鬥中會頻繁重播的長演出。

## 3. 全屏奧義影片的潛力

### 3.1 為什麼它很有潛力

- 這可能是你們目前唯一真正能做出「會動」而且有戲劇感的資產
- 它不依賴即時角色動畫系統，對現有 pipeline 比較友善
- 如果只在 `奧義 / 大招 / 角色獲得 / 高光時刻` 出現，玩家會覺得質感很高
- 它非常符合手游語境，尤其對男性向三國題材來說，奧義 full-screen 是合理且有爽感的

Unity 對照：這很像用預先渲染好的 skill cut-in movie，取代即時骨架動畫與複雜 timeline 演出。雖然不是真正的即時動畫，但在成本受限時，情緒回報很高。

### 3.2 它最適合用在哪裡

- 每位主將的 `奧義發動`
- 首次獲得角色
- 章節 Boss 技能
- 劇情高潮節點

### 3.3 不適合怎麼用

- 不要每個普通技能都播
- 不要每回合反覆看同一支，會很快疲勞
- 不要把它當作戰場本體演出的替代品

### 3.4 最佳使用策略

- 每位重點武將只配 `1 支代表性奧義影片`
- 長度控制在 `3~6 秒` 最理想，`10 秒` 已偏長
- 可支援「首次完整播放，之後可跳過 / 縮短」
- 角色與技能要高度綁定，讓玩家一看到畫面就知道是誰的招

## 4. 漫畫劇情式插圖串接的潛力

### 4.1 為什麼它其實非常強

這條線我認為比「不穩定影片生成器」更值得長期投資，原因是：

- 你們本來就有 `2D 靜態繪圖` 長板
- 漫畫式分鏡不需要角色骨架動畫
- 很適合補三國題材最需要的「故事感、名場面、人物關係」
- 也很容易做成一致風格，不像影片生成器可能會有較大波動

### 4.2 它最適合用在哪裡

- 劇情章節推進
- 武將列傳
- 招募 / 覺醒 / 關係事件
- 奧義發動前的漫畫式切頁
- 任務完成、遭遇戰開場、章節回顧

### 4.3 它能補什麼缺

- 補你們沒有角色動畫的敘事缺口
- 補你們戰場特效短板，因為情緒可以用畫面分鏡來撐
- 補角色之間的戲劇張力，不只靠數值與卡面

### 4.4 它的商業價值

- 非常適合做武將收藏、圖鑑、章節演出與社群素材
- 能把「老三國情懷」做得比市場主流更強
- 也容易形成自家辨識度，不會像 generic 仙俠卡面那麼容易跟別人撞

## 5. 兩者怎麼搭最好

最建議的不是二選一，而是做成一條混合演出線：

### 建議流程 A：奧義演出

1. 漫畫式定格 / 大字卡 / 分鏡插圖先進場
2. 接 `3~5 秒` 奧義全屏影片
3. 回到戰場 HUD 與傷害結果

這樣的好處是：

- 前段用穩定的 2D 圖像定角色與情緒
- 中段用短片拉爆點
- 後段回戰場，不會拖太久

### 建議流程 B：劇情推進

1. 漫畫分鏡頁
2. 關鍵角色 close-up
3. 重要事件才插短影片或動態鏡頭

這條線比全片都靠 AI 影片更穩，也更容易維持世界觀一致性。

## 6. 哪個更值得先投資

如果只能先押一條，我會這樣排：

### 第一優先：漫畫劇情式插圖串接

- 穩定
- 可量產
- 命中你們長板
- 最能建立敘事辨識

### 第二優先：少量高價值奧義短片

- 當作角色高光武器
- 不要鋪滿整個遊戲
- 僅給重點武將與重點招式

## 7. 風險提醒

### 影片生成器風險

- 視覺穩定度可能不一致
- 同一角色不同影片可能長相飄移
- 時長太長會打斷戰鬥節奏
- 若影片質感忽高忽低，反而會拉低整體品質

### 漫畫插圖風險

- 若分鏡語言不成熟，會變成只是靜態插畫輪播
- 若文字與畫面節奏沒有設計，戲劇感會弱

## 8. 最終判斷

### 奧義影片

有潛力，而且值得做，但應定位為：

- `高光演出資產`
- `少量精品`
- `每位主將 1 支代表作`

不要把它當作全面替代動畫系統的主解。

### 漫畫劇情式插圖

潛力非常高，而且更符合你們目前的產能結構。它不只是備案，而是可以成為你們三國題材差異化的正式武器。

## 9. 建議下一步

1. 先挑 `2 位男將 + 1 位女將`，各做一版「漫畫定格 + 奧義短片」混合 demo。
2. 先把奧義影片目標長度壓到 `3~6 秒`，避免 10 秒過長。
3. 建立一份 `奧義影片 QA checklist`：
   - 角色長相是否穩定
   - 招式辨識是否清楚
   - 是否像同一個世界
   - 是否值得玩家反覆看
4. 建立一份 `漫畫分鏡 QA checklist`：
   - 情緒點是否明確
   - 是否比單張立繪更有故事感
   - 是否能補足沒有骨架動畫的缺口
