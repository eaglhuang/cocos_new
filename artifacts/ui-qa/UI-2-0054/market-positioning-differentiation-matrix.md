# 市場定位差異化矩陣

## 1. 目標

這份矩陣不是要否定市場主流，而是要回答一個更務實的問題：

如果目前最紅的三國手游立繪都已經長這樣，我們到底要學什麼、不要學什麼，以及要靠哪幾個點把自己拉出來。

## 2. 主流市場共通語言

| 面向 | 主流市場常見做法 | 為什麼有效 | 為什麼容易同質化 |
|---|---|---|---|
| 男武將 | 半寫實俊美臉、亮面甲、誇張披風、標準英雄 pose | 第一眼夠帥、夠爽 | 角色常變成同一種帥，只差配色 |
| 女武將 | 柔光臉、飄帶、花飾、收藏卡姿態、偏仙俠性感 | 男性向吸引力強 | 容易變成同一種美，失去世界觀差異 |
| 卡面 | 發光外框、紅橙金 rarity、滿版數值與養成符號 | 手游感明確、商業化成熟 | 很多產品視覺節奏幾乎一模一樣 |
| 世界觀 | 多以角色卡為主，場景只是襯底 | 成本低、聚焦角色 | 世界感薄，角色像不活在同一個世界 |
| 三國辨識 | 借三國名字與象徵色，但偏 fantasy 國風手游模板 | 大眾易懂 | 歷史情懷被沖淡，最後只剩 generic 國風 |

## 3. 我們該學什麼

### 可學習

- 角色第一眼辨識要強
- 卡面商品感要夠
- 女角要有男性向吸引力
- 稀有度、收藏感、角色入手爽感要直接

### 不該照抄

- 不要全部角色都用同一種俊男美女模板
- 不要全靠亮面、發光與 rarity 框撐高級感
- 不要把三國畫到只剩 generic 仙俠手游味
- 不要讓世界場景永遠只是 UI 背板

## 4. 我們的差異化主軸

### 主軸 A：懷舊三國辨識

市場主流常把三國人物畫成「國風 fantasy 帥哥美女」；我們要保留更強的三國辨識。

做法：

- 男武將保留歷史名將氣質與身份感
- 女武將保留古典底子，不直接滑向仙俠偶像卡
- 配色、甲冑、髮冠、武器與姿態要能一眼聯想到三國名將 archetype

### 主軸 B：角色賣點分工

市場主流常把所有角色收成同一套審美模板；我們要讓每個角色的主賣點不同。

做法：

- 趙雲：俊朗正氣
- 關羽：威儀忠義
- 張飛：壓迫爆發
- 呂布：危險華麗
- 曹操：權勢統御
- 女武將：通用古典母型 / 威儀型 / 商業高光型三條分支

### 主軸 C：世界與角色一起成立

市場主流很多是卡面強、世界弱；我們要用靜態 3D 世界質感補這個洞。

做法：

- 戰場：3D 靜態角色 + shader + UI 疊合
- 主城 / 家園 / 國家治理：精緻 3D 建物與景物
- 小兵：可愛半寫實，但要與主將同世界

### 主軸 C.1：男女分工可以不同，但世界語言不能裂開

你想走的路線可以成立：

- 男武將吃歷史、戰場、輸出與奧義爽感
- 女武將吃男性向、現代審美、收藏與劇情魅力

但前提是：

`賣點可以不同，世界語言不能不同。`

也就是：

- 男角更硬、更戰場、更歷史
- 女角更美、更精緻、更現代

但兩邊都要共用：

- 布料語言
- 金屬語言
- 階級配色
- 飾件與髮冠邏輯
- 城池 / 宮廷 / 戰場的世界觀基底

### 主軸 D：少而準的高級感

市場主流容易用大量光效與發光框營造昂貴感；我們的策略是把高級感放在材質、輪廓、光影與構圖。

做法：

- 角色甲冑與布料質感先站住
- UI 發光與 rarity 特效節制使用
- 特效先做關鍵命中與高光，不做廉價滿屏鋪量

## 5. 逐面向差異化矩陣

| 面向 | 市場主流 | 我們建議 | 拉開差距的原因 |
|---|---|---|---|
| 男武將 | 同一種仙俠化俊美模板 | 懷舊三國底子 + 現代入口感 + 賣點分工 | 讓核心 TA 有情懷，也讓 roster 更有記憶點 |
| 女武將 | 高收藏感、偏 fantasy 美人卡 | 古典國風底子 + 男性向美感 + 分支化 archetype | 保留吸引力，同時不脫離世界觀 |
| 男女分工 | 常不是一起仙俠化、就是一起保守化 | 男吃歷史戰場、女吃現代男性向，但共用世界材質語言 | 能同時滿足商業性與世界一致性 |
| 小兵 | 常被弱化成廉價陪襯 | 可愛半寫實 family，與主將同世界 | 讓整體戰場更完整，不只主將好看 |
| 戰場 | 常靠 2D 卡感或特效堆疊 | 3D 靜態模型 + shader + world-space readability | 命中你們的製作長板 |
| 主城 / 地圖 | 常普通、功能化 | 精緻 3D 建物 / 景物 / 地標語言 | 很容易成為中期差異化優勢 |
| 卡面 | 靠發光框與養成 UI 疊滿 | 靠人物 archetype、構圖、材質與稀有度節奏 | 比較不容易落入廉價卡牌感 |
| 特效 | 大量、華麗、常同質 | 少量關鍵、高辨識、先求可用再求華麗 | 避免以短板硬拼主流 |

## 6. 最終市場定位句

### 對外感受版

`這不是又一款長得一樣的仙俠化三國手游，而是一款更有老三國味、更有角色辨識、世界也更完整的現代三國作品。`

### 對內製作版

`角色靠懷舊辨識與 archetype 分工建立記憶點，世界靠 3D 靜態模型與 shader 建立質感，特效只做高價值補強，不做廉價鋪量。`

## 7. 風格防呆清單

之後每次審圖時，都可以先問這幾題：

1. 如果遮掉名字，這張還像三國人物，還是 generic 國風角色？
2. 這個角色的賣點，和 roster 其他人有明確差異嗎？
3. 這張圖的高級感，是靠材質與構圖，還是只靠發光與裝飾？
4. 如果放進戰場、GeneralDetail、TigerTally，會不會像三個不同遊戲？
5. 這張圖有沒有把我們往「市場主流模板複製品」推過去？
6. 這張女角是「現代審美的三國女性」，還是「現代偶像穿古裝」？

## 8. 建議下一步

1. 做一張 `市場主流 vs 我們` 的 compare board，至少放男主將、女主將、小兵各 1 組。
2. 做一張 `世界感` compare board，把主城 / 建物 / 地圖節點的 3D 靜態方向先定調。
3. 把這份矩陣濃縮成後續生圖 prompt 與 QA checklist 的固定欄位。
