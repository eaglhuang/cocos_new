# General Detail Bloodline Concept Prompt v3

## 用途

- 日常人物頁 `v3`
- 角色本人是主角
- 血脈層是深層人格區，不是第二主角
- 下方故事條是產品特色內容，不是附贈漫畫

## Gemini Prompt

```text
Please generate a high-quality game UI concept illustration for a Three Kingdoms hero detail screen.

This is a normal daily character profile interface, not an awakening screen and not a poster.

Use Zhao Yun as the example hero.

The layout should be clear, practical, elegant, and readable in a 16:9 horizontal format:
- left side: premium near full-body character portrait from head to calf
- right side: refined readable information panels
- bottom: a wide connected 6-panel story strip that reads from left to right as one complete life-and-bloodline story

Main priority:
The hero must remain the true visual center of the page.
This is still a character profile screen, not a bloodline spectacle page.

Left side:
Show Zhao Yun as a premium heroic character portrait, noble, elegant, handsome, and clear.
His design should feel collectible and readable.
Integrate subtle bloodline motifs directly into armor, belt ornament, shoulder trim, jade accessory, or fabric pattern, so the bloodline language is already part of the character design.
Do not make the portrait feel overloaded.

Right side:
Show structured character information in a premium game UI style:
- name
- title
- rarity
- five core attributes
- combat role
- combat traits
- bloodline summary
- awakening tendency
- bloodline personality note
- ancestral beast symbol badge

Important:
The bloodline area should feel like the hero's deeper identity layer, not like a separate side card pasted onto the page.
Blend it into the same interface system with shared framing, shared material language, and subtle bloodline patterns in the panel background.

Bloodline style:
Use subtle ancient sigils, elegant carved symbols, quiet ancestral traces, refined mystical motifs, and a more abstract bloodline crest system.
Avoid using an overly literal animal-head emblem as the only symbol.
The ancestral-beast mark should feel like a premium crest, seal, relic emblem, or bloodline insignia.

Bottom story strip:
Create one connected 6-panel illustrated story strip across the entire bottom of the interface.
It should read continuously from left to right.
Panel rhythm:
1. early training or youth
2. entry into history
3. mortal achievement or famous deed
4. turning point where destiny begins to interfere
5. bloodline truth, ancestral encounter, or inner calling
6. the hero carrying inherited destiny into the future

The strip should feel important, premium, and memorable.
It should look like a signature content layer of the product, not a tiny decorative extra.

Visual style:
Premium ancient strategy game UI with strong readability and elegant international adaptability.
Use parchment, jade-white, muted teal, soft gold, dark ink accents, and refined ornamental framing.
Avoid overly rigid traditional Chinese symbolism, heavy calligraphy, sci-fi HUD, cyberpunk, horror, or western fantasy.
The interface should feel like a real premium game character screen with a distinct identity.

Overall feeling:
A premium character profile screen where the hero is the main focus, but the viewer can clearly sense that a deeper inherited bloodline destiny is quietly shaping the hero from within.
```

## 精修重點

1. 血脈區要更像同一頁的深層人格區，不像外掛第二張卡。
2. 圖騰要從「直白狼頭」提升成抽象、可系統化的命紋徽記。
3. 六格故事要更清楚讀出「人生 -> 轉折 -> 血脈真相 -> 延續」。
