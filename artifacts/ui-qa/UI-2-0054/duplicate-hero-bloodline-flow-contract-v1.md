# 重複卡分流與血脈灌注契約 v1

## 定位

本文件定義重複抽到同名英雄後，不再走傳統碎片路線，而是分成兩條正式世界觀分流：

1. `復生重練`
2. `血脈灌注`

## 核心原則

### 1. 第一次抽到是得人

- 玩家第一次拿到的是活著的名將本體。

### 2. 之後抽到是留人，還是養魂

- 重複抽到同名卡，不應只是數值碎片。
- 玩家必須做選擇：
  - 讓他再活一世
  - 或讓他的英靈更完整

## 正式分流

### 復生重練

- 讓同名名將再次以人身歸來
- 舊有 `英靈卡 / 英靈虎符 / 血脈卡` 可重新嵌回新身上
- 定義上是：
  - 帶著前世英靈重練
  - 同一血脈再次入世

### 血脈灌注

- 不生成新身體
- 把這次抽到的同名卡直接灌進既有英靈體系
- 用途：
  - 提升血脈等級
  - 提升英靈濃度
  - 強化英靈虎符
  - 強化特種軍隊

## 系統影響

### 角色

- 復生重練：角色本體再入 roster
- 血脈灌注：角色本體不增加，但靈魂層變強

### 英靈卡

- 復生重練：保留舊卡，可重新鑲嵌
- 血脈灌注：直接升階 / 累積 essence

### 英靈虎符

- 血脈灌注後同步升級
- 可解鎖更高階血脈面、命紋層、軍勢 aura

### 特種軍隊

- 跟隨虎符與英靈層級同步成長
- 不只是 attack 數值提升，也應有稱號 / badge / crest 進化

## UI 決策卡片

### 重複抽卡後彈窗必備欄位

- 英雄名
- 已持有狀態
- 當前血脈等級
- 當前英靈虎符等級
- 兩個選項：
  - `復生重練`
  - `血脈灌注`

### 選項說明文

#### 復生重練

- 適合想重新培養本體的玩家
- 對 roster / 人身 / 裝備 / 養成更有感

#### 血脈灌注

- 適合想快速養魂、強化英靈面與虎符軍勢的玩家
- 對命鏡、英靈卡、虎符、特種軍隊更有感
