# 血脈逸聞與命紋靈獸劇本樣本

## 定位

這個功能不是一般角色傳記，也不是可愛寵物陪伴系統，而是把《三國傳承》的核心玩法 `基因 / 血脈 / 傳承` 轉成可閱讀、可收藏、可商品化的角色內容層。

建議正式命名：

- 功能名：`血脈逸聞`
- 角色伴生物名：`命紋靈獸`

若 UI 或文案需要更白話，對外可以暫時寫成「血脈逸聞：某武將與其命紋靈獸的逸事」。

## UI 用法建議

- 入口位置：角色主立繪下方資訊層，點擊人物後緩慢浮出。
- 呈現方式：`3~5 格` 靜態連環漫畫，每格搭配 `1~2 句` 短文。
- 解鎖節奏：取得武將後先開第一則，升星、覺醒、婚配或血脈培養後逐步解鎖後續逸聞。
- 美術語言：不是萌系寵物，而是古老圖騰、魂影、獸紋、青銅器紋樣、殘墨靈體。
- 功能目的：讓玩家感受到這名武將的獨特基因，不只存在於數值，而是會投射成一種命格、一種伴生靈獸、一段可記住的故事。

## 角色設計硬規則

血脈逸聞若要成立，命紋靈獸不能只存在於故事裡，也必須回寫到人物主立繪本身。

- 每位名將身上都要有 `1~3 個` 明確可辨的血脈符號 slot。
- 符號優先落點：`胸甲 / 腰封 / 髮飾 / 披肩扣件 / 玉佩 / 虎符 / 武器掛飾`。
- 命紋靈獸不只是一個小 logo，而應該同時帶出：
  - 獸種輪廓
  - 對應基因紋樣
  - 專屬材質語言
  - 色系微光或刻痕
- 男武將可偏向 `甲紋 / 槍紋 / 腰牌 / 披風扣具`。
- 女武將可偏向 `髮冠 / 垂飾 / 胸甲弧紋 / 腰鏈 / 裙甲紋線`。
- 小尺寸閱讀時，玩家至少要能看出「這個人身上有一種固定血脈徽記」，不能只剩細碎花紋。
- 若故事中的靈獸與角色立繪沒有對應符號，這條線就會變成兩套系統，必須避免。

## 角色符號植入模板

- `輪廓層`
  - 例如角、羽、鱗、螭尾、麟額、獸瞳弧線，轉譯成胸甲或髮冠輪廓。
- `紋樣層`
  - 把靈獸身上的命紋變成布料暗紋、金屬刻痕、玉飾鏤空或披風邊紋。
- `材質層`
  - 白麟偏霜銀裂紋、月螭偏月白水光、赤翎偏赤金火漆，不同靈獸必須有不同材質記憶點。
- `點亮層`
  - 覺醒、共鳴、婚配、子嗣預測時，原本藏在服裝裡的符號可以短暫點亮或浮現魂影。

## 雙面角色概念

這套世界觀最有力的地方，不是單純讓角色「很帥 / 很美」，而是讓每位名將都同時擁有兩面：

- `表象面`
  - 玩家第一眼看到的英雄形象、身份、魅力、威儀、戰力。
- `血脈面`
  - 角色被古老因果與祖靈命紋牽引的一面，是命格、慾望、執念、宿命與後代傾向的來源。

這兩面不應被做成黑化、恐怖或怪物化，而是：

- 表象面看起來仍然是正常名將
- 血脈面則透過 `命紋靈獸虛影 + 古老命篆符號 + 局部裝飾點亮` 被短暫揭露

也就是說，玩家點角色時，不是看到另一個可怕人格，而是看到「他真正被什麼古老血脈牽著走」。

## 點擊立繪演出節奏

建議角色主介面的互動順序固定化：

1. 點擊人物主立繪
2. 角色上方浮出對應 `命紋靈獸虛影`
3. 角色身上既有的血脈符號 slot 短暫亮起
4. 背後或肩後浮現 `祖紋命篆`
5. 下方資訊層滑出 `血脈逸聞` 靜態連環漫畫

這個演出要偏：

- 靜
- 儀式感
- 古老
- 高級

不要偏：

- 科技 UI 掃描
- 恐怖附身
- 高奇幻爆光特效

## 古老命紋符號語言

這套符號不應該像現代基因圖譜，也不應該像純宗教符咒照抄，而應該做成你們自己的 `祖紋命篆`。

### 視覺來源建議

- 甲骨殘筆
- 金文 / 青銅器銘紋
- 小篆流線
- 帛書殘缺筆意
- 玉璧刻痕
- 漢代石刻裂紋
- 類梵字的神秘秩序感，但不要直接使用現成宗教文字

### 應有特徵

- 中心有一枚 `核心命核`
- 外圍有不完整封輪或斷裂圓環
- 內部筆劃有向外生長的因子支線
- 每種靈獸對應不同尾端筆勢
  - 白麟：角狀、裂霜、向上收尖
  - 月螭：水弧、纏行、細長迴游
  - 赤翎：羽枝、火舌、外放分叉
- 符號不求可讀，但必須 `可辨`

### 材質與亮法

- 不是 neon glow，而是：
  - 玉光
  - 舊金微亮
  - 月白水光
  - 火漆燙紋
  - 殘墨滲光
- 亮起方式像古器受靈，不像 UI 掃描線。

## 圖示與品牌應用

這套雙面概念甚至可以往品牌識別推，尤其適合遊戲 icon。

### 遊戲 icon 構圖方向

- 主角建議可先用趙雲 proof。
- 構圖不是一半正常、一半怪物，而是：
  - 一側保持英雄光明面
  - 另一側在眼角、甲紋、髮飾、披風邊或頰側浮出 `祖紋命篆`
  - 背後或肩後半浮出對應靈獸虛影
- 整體氣質要像：
  - 英雄之中藏著古老命格
  - 不是被污染，而是被喚醒

### Icon 關鍵詞

- 光明英雄面
- 古老血脈面
- 非科技
- 非恐怖
- 非重奇幻
- 高級神秘
- 歷史感 + 命格感

### 建議配色

- 主體仍走角色原色
- 血脈面只加一層：
  - 舊金
  - 玉白
  - 墨青
  - 暗赤
  - 月銀

不要整張 icon 都變成邪氣黑紫，否則會跑到別的產品賽道。

## 血脈卡與英靈虎符

這條設定能把《三國傳承》的另一個主賣點一起接進來：`虎符特種軍隊`。

建議世界觀定義如下：

- 名將不是真正消失，而是在壽命盡後留下 `血脈卡`。
- 血脈卡不是普通遺物，而是名將一生意志、命紋與祖靈回聲的凝結。
- 這張卡同時也是一枚 `英靈虎符`，能喚出該名將留下的特種軍隊。
- 戰鬥中插卡觸發時，現身的不是完整活人，而是該武將的 `血脈面 / 英靈面`。

這樣就能把「名將為何不死」與「為何能留下專屬兵種」講通：

- 人會死
- 名將的表象會逝去
- 但其血脈命紋、英靈意志、統軍印記會被封存於血脈卡中
- 因此他留下的不只是故事，還有可被後世喚回的軍勢

## 虎符特種軍隊的美術規則

每支名將專屬軍隊，不能只是換色兵模，而應該承接同一位名將的命紋靈獸與血脈故事。

- 虎符本體要帶有該名將的 `祖紋命篆`
- 特種軍隊裝甲、旗幟、披風、武器飾件，要出現同系 `靈獸符號`
- 軍隊的氣質、編制、視覺輪廓，也要像是由該名將血脈延伸出來的兵種
- 插卡時浮現的英靈面形象，要與角色本尊形成 `表象面 / 血脈面` 對照

### 例子

- 趙雲系特種軍
  - 應帶 `白麟 / 霜槍 / 裂霜命紋`
  - 軍容感覺要像守命、疾進、護主
- 甄姬系特種軍
  - 不一定是前線猛軍，可以是偏儀衛、幻術、靜謀、冷月水紋系兵勢
  - 讓玩家看得出這是她留下的血脈影響，而不是 generic 女角兵
- 孫尚香系特種軍
  - 應帶 `赤翎 / 烈羽 / 王脈火紋`
  - 視覺上應更俐落、迅捷、帶女武將軍勢與公主威儀

## 戰鬥中插卡演出

建議 `血脈卡 / 英靈虎符` 的戰鬥演出節奏固定化：

1. 玩家插入血脈卡
2. 卡面上的 `祖紋命篆` 先亮起
3. 武將的 `命紋靈獸虛影` 先現身
4. 角色的 `血脈面 / 英靈面` 短暫浮出
5. 對應的虎符特種軍隊被召出或獲得血脈加護

這樣玩家會明白：

- 插卡不是單純叫技能
- 而是在喚醒該名將留下的英靈軍勢

## 重複抽卡的血脈分流

這條設定很適合直接變成 duplicate pull 的核心規則，讓玩家抽到同名武將時，不是只覺得重複，而是在做一次血脈選擇。

建議分成兩條主路：

- `復生重練`
  - 玩家選擇讓該名將以新的肉身再度歸來。
  - 原本留下的 `英靈虎符 / 血脈卡` 可以插入新身，形成「帶著前世英靈重練」的玩法。
  - 世界觀上像是讓名將再度以人之表象現世，但骨子裡仍背著自己的血脈面與英靈記憶。
- `血脈灌注`
  - 玩家不讓他重生，而是把這次抽到的同名卡灌注進既有 `英靈虎符 / 血脈卡`。
  - 血脈等級上升、命紋更完整、英靈面更濃、虎符特種軍隊也同步變強。
  - 世界觀上像是讓該名將的一生意志被反覆加厚，從殘響變成更完整的英靈。

這樣一來，抽重複卡就不再只是：

- 升星材料
- 碎片
- 被動重覆值

而是：

- 要讓這位英雄再次活過一世
- 還是讓他的英靈更古老、更強、更接近純血脈的存在

## 復生重練的戲劇性

這條線的魅力在於，它能把名將的「人」與「血脈」分開。

- `人身`
  - 會衰老、會有新的命運、會再被玩家培養。
- `英靈卡`
  - 保留前世軍勢、血脈、命紋、靈獸回聲。

所以玩家不是單純重抽一張趙雲，而是：

- 抽到了一次「讓趙雲再次為人」的機會
- 也可以把舊趙雲的英靈插回新趙雲身上

這會讓整個產品非常有自己的味道。

## 血脈灌注的視覺回饋

如果玩家選擇強化 `英靈虎符 / 血脈卡`，視覺上應該要有明確回饋：

- 卡面上的 `祖紋命篆` 變得更完整
- 靈獸虛影更清晰，從殘影變成半具象
- 角色 `血脈面` 的紋樣覆蓋更成熟
- 虎符特種軍隊的旗幟、甲紋、武器符號變得更鮮明
- 軍隊插卡演出時，英靈面停留時間或層次感更強

也就是說，升級的不是單純數值，而是：

`名將在死亡之後留下的軍魂厚度`

## 產品價值

如果這條線成立，你們的抽卡體驗就會很不一樣：

- 第一次抽到是 `得人`
- 第二次抽到是 `問你要留人，還是養魂`
- 第三次之後則是持續在 `人身養成` 與 `英靈血脈養成` 之間做選擇

這會比一般手游的重複卡體驗更有敘事感，也更符合《三國傳承》的核心主題。

## 產品主賣點收束

如果把這條線立起來，整個產品的辨識會更強：

- 平時看到的是角色的 `表象面`
- 點角色會看到他的 `血脈面`
- 看故事會知道他的 `命紋靈獸`
- 養成時會研究他的 `血脈價值`
- 戰鬥時會用他的 `血脈卡 / 英靈虎符`
- 召出的特種軍隊則是他一生軍勢的殘留與延續

這會讓《三國傳承》從「抽三國名將」變成：

`收集名將、繼承血脈、喚回英靈、延續軍勢`

## 劇本格式

每篇逸聞統一採以下格式：

- 武將定位
- 專屬基因與命紋靈獸
- 情緒基調
- 格數腳本
- UI 文案短版

## 01. 趙雲｜《雪夜白麟》

### 武將定位

- 主題：正氣、克制、守護、天命不折
- 類型：主線男武將、歷史英雄感強

### 專屬基因與命紋靈獸

- 基因名：`霜槍守命`
- 靈獸名：`白麟`
- 視覺語言：雪白短鬃、額生單角、青銀命紋、踏雪不留痕

### 情緒基調

沉靜、堅定、帶宿命感，不賣悲情，賣的是「再冷的夜也會站到最後」。

### 格數腳本

#### 第 1 格

- 畫面：少年趙雲在北地雪夜獨行，風雪將路埋盡，遠處枯樹間浮出一道白影。
- 旁白：`常山的雪，最會試人。`

#### 第 2 格

- 畫面：白麟停在半坡回頭望他，蹄下無痕，額前命紋像冰裂一樣微亮。
- 旁白：`那夜，趙雲第一次見到只會在古老血脈裡現身的白麟。`
- 趙雲台詞：`你是來引路，還是來看我能走多遠？`

#### 第 3 格

- 畫面：兩者並肩穿雪而行，少年趙雲持槍前行，白麟微微低首。
- 旁白：`白麟不應人語，只在強者不退時同行。`

#### 第 4 格

- 畫面：山道盡頭露出天光，白麟的身影化成青銀色命紋，沒入趙雲槍身與腕甲。
- 旁白：`自此之後，他每一次回身護人，霜紋便會沿骨而亮。`

#### 第 5 格

- 畫面：成年趙雲立於戰場前，白麟虛影在身後一瞬浮現。
- 旁白：`世人只知子龍七進七出，卻不知他命裡從來有一頭雪夜不退的白麟。`

### UI 文案短版

`趙雲的血脈命紋為「霜槍守命」。傳說當白麟現身，便代表此人此生註定護人而戰，縱千軍壓境，亦不後退。`

## 02. 甄姬｜《月池螭影》

### 武將定位

- 主題：冷豔、疏離、靜水深流、貴氣與哀感
- 類型：女武將卡池主賣相、宮廷系高級女性角色

### 專屬基因與命紋靈獸

- 基因名：`洛月凝華`
- 靈獸名：`月螭`
- 視覺語言：纖長白螭、淡紫銀光、水紋命輪、近月而生

### 情緒基調

不是可愛互動，而是若即若離、像照鏡一樣的命格回聲。

### 格數腳本

#### 第 1 格

- 畫面：夜深宮池，甄姬獨坐水榭，水面平靜如鏡。
- 旁白：`魏宮深處，最靜的不是夜，而是無人敢驚動的那一池月色。`

#### 第 2 格

- 畫面：水面浮出細長的白色螭影，沿池邊緩慢游行，所過之處泛起淡紫命紋。
- 甄姬台詞：`今夜也是你先來。`
- 旁白：`月螭只會靠近命中帶有洛月之紋的人。`

#### 第 3 格

- 畫面：甄姬伸手，指尖幾乎碰到月螭，卻仍隔著一層薄水。
- 旁白：`它從不真正停在人掌之上。正如她的心，總隔著一層看不見的水。`

#### 第 4 格

- 畫面：水中映出她未來的片段，婚配、宮闈、流言、離散，全都像月影被風吹碎。
- 甄姬台詞：`若命運一定要奪，我至少要讓它先照見我。`

#### 第 5 格

- 畫面：月螭化作一道銀紫命紋，纏上她的腰間玉飾與髮冠。
- 旁白：`人們只看見她的美，卻不知道那份冷，原是月螭留給後代的血脈印記。`

### UI 文案短版

`甄姬的血脈命紋為「洛月凝華」。月螭與她同生於靜水與月影之間，能映照命運，亦讓她的血脈帶著難以靠近的華貴與清冷。`

## 03. 孫尚香｜《赤翎不伏》

### 武將定位

- 主題：英氣、任性、俏皮、帶火性的公主武將
- 類型：吳國卡池女武將、武裝公主路線

### 專屬基因與命紋靈獸

- 基因名：`烈羽王脈`
- 靈獸名：`赤翎`
- 視覺語言：赤金小型靈禽、尾羽帶火紋、速度快、脾氣烈

### 情緒基調

不是哀美，而是帶點挑釁與少年氣的貴族英姿。

### 格數腳本

#### 第 1 格

- 畫面：年少的孫尚香在庭中射靶，最後一箭故意射斷高處鈴索。
- 鈴聲特效：`叮——`
- 旁白：`江東的人都知道，郡主若安靜太久，多半是在想怎麼闖禍。`

#### 第 2 格

- 畫面：一道赤金色小靈禽自廊上翻落，停在箭尾上，歪頭看她。
- 孫尚香台詞：`你也覺得這一箭還不夠漂亮？`
- 旁白：`赤翎只親近骨子裡不服輸的人。`

#### 第 3 格

- 畫面：她追著赤翎跑上長廊，披風飛起，赤翎忽快忽慢地故意逗她。

#### 第 4 格

- 畫面：孫尚香翻身躍上欄杆，終於與赤翎並肩立在高處，晚霞映滿衣甲。
- 孫尚香台詞：`不跟也行，你看著我贏。`
- 旁白：`那一刻，赤翎尾羽忽然盡亮，像是認下了她的名字。`

#### 第 5 格

- 畫面：成年孫尚香披甲立於風中，肩後浮現赤翎靈影，火紋沿胸甲與腰封微亮。
- 旁白：`她的任性從不是撒嬌，而是烈羽王脈天生不肯伏於誰人之下。`

### UI 文案短版

`孫尚香的血脈命紋為「烈羽王脈」。赤翎與她一樣敏捷、傲氣、好勝，當她下定決心時，火紋便會順著鎧甲亮起。`

## 量產建議

- 第一批只挑 `三國名將` 製作，不先鋪滿全部武將。
- 每位名將先做 `1 則主逸聞 + 1 隻命紋靈獸`，驗證玩家記憶點。
- 靈獸採 `family + 專屬變體` 模式，不要每位都從零發明新物種。
- 每篇逸聞都要能反推角色氣質、血脈價值與後代傾向，避免淪為純裝飾小劇場。

## 下一步
