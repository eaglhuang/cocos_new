# Agent1 Generation Brief / UI-2-0039

## 任務

為 TigerTally card art 先做兩條 proof：

1. `A4-tactical-diorama-card` 主語言
2. `A5-hero-panel-card` 對照語言

## A4 主語言要求

- 主題：城樓、要塞、關卡、戰術地形、軍事節點
- 視角：俯視或 3/4 俯視
- 風格：古風戰場卡面，不是寫實插畫海報
- 材質：石牆、木構、地圖紋、暗色 battlefield
- 語意：一眼看出「戰術卡 / 地點卡」
- 禁止：
  - 內嵌文字
  - 內建 rarity badge
  - 過多發光
  - 現代 UI icon 感

## A5 對照語言要求

- 主題：角色半身 + 少量背景
- 只作比較，不作預設方向

## 輸出

- `tiger_tally_card_art_a4_v1_512.png`

- `A4` 是否比 `A5` 更適合 TigerTally 的戰術卡定位
