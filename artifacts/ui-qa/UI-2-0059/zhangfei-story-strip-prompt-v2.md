# 張飛故事帶 Prompt v3

用途：交給 Gemini / Web Gemini 產出 `GeneralDetailBloodlineV3` 底部故事帶用的 `master art 長條圖`。

## 這次要避免的誤解

- 不要做成六張直式分鏡。
- 不要做成六張海報拼接。
- 不要讓每一格都像獨立封面。
- 不要把重要人物頭部放在太高的位置，避免後續裁成橫條時被切掉。

## 正確方向

- 要的是一張「超寬橫向長條圖」。
- 六段故事只是敘事節奏，不是六張獨立卡。
- 畫面用途是遊戲 UI 底部 `story strip`。
- 構圖必須適合後續裁成矮長條。
- 重要主體盡量集中在畫面中下 60% 區域。

## 角色一致性原則

- 主角是張飛。
- 角色辨識重點放在：`紅頭巾 / 蛇矛 / 深色鎧甲 / 壯碩體態 / 蜀軍旗幟`。
- 預設採用弱臉化設計，優先用側臉、背影、半身、遠景、剪影、手勢與武器特寫。
- 避免正臉特寫，降低和專案內正式立繪不一致的風險。

## 可直接貼給 Gemini 的版本

```text
請畫一張「超寬橫向長條圖」，用途是遊戲 UI 底部的 story strip。

非常重要：
不要做成六張直式分鏡。
不要做成六張海報拼接。
不要做成每格都像獨立封面的構圖。
我要的是一張完整的橫向歷史長卷，六段故事只能用同一張橫向畫面自然連續地表現。

正確方向：
- 一張完整橫向長卷
- 左右流動的連續敘事
- 六段之間用地形、旗幟、光影、戰場煙塵、人物動勢自然過渡
- 適合最後裁進很矮的 UI 長條區域

錯誤方向：
- 六張直式卡片
- 六張各自獨立的海報
- 每格都有完整人物全身站姿與大面積天空

主角是張飛，但請採用弱臉化設計，避免正臉特寫。角色辨識重點放在紅頭巾、蛇矛、深色鎧甲、壯碩身形、蜀軍旗幟與戰場氣氛。可以多使用側臉、背影、遠景、半身、剪影、手勢與武器特寫。

六段內容依序為：
1. 燕地少年張飛在鄉里持矛練武，村舍、晨霧、草莽豪氣。
2. 張飛與劉備、關羽相見結義，桃花、旗幟、命運交會。
3. 張飛披甲衝陣，怒喝突擊，亂軍與煙塵形成強烈動勢。
4. 長坂橋前獨據橋頭，震懾追兵，橋面、河流、敵軍與蛇矛形成緊張張力。
5. 夜色中面對發光的命紋碑與祖紋符印，偏背影與靈光場景。
6. 張飛騎馬回望蜀軍，夕陽、軍旗與名將收束感作結。

硬限制：
- 畫布必須是橫向超寬，不可以是直向分格
- 重要人物頭部與武器不要貼近上邊緣
- 重要主體請集中在畫面中下 60% 區域
- 不要文字、不要 UI、不要 logo、不要浮水印
- 要有高品質商業遊戲插畫質感
```

## 尺寸建議

- 優先生成正常比例的大圖，再由 repo 內流程裁成 strip。
- 建議原圖尺寸：`2816 x 1536` 或更高。
- 但構圖必須從一開始就以「橫向長卷可裁切」為前提，而不是六張直式分鏡。

## 補充提醒

- 如果 Gemini 仍然容易誤解，就在開頭再重複一次：
  - `不要六張直式圖`
  - `不要六張海報`
  - `只要一張橫向長卷`
