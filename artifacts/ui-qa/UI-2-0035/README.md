# UI-2-0035 QA Artifact README

這個資料夾保存 `UI-2-0035` 的 icon family 研究產物。

## 內容

- `reference-matrix.md`
  - 至少 20 個 icon 樣本的來源、family、使用條件與判讀理由。
- `icon-family-assignment.md`
  - 把目前已開立的 icon 量產需求單、現行 ui-spec screens/layouts、以及 UI 規格書情境，對應到 F1~F8 family。
- `notes.md`
  - 這一輪研究結論，包含高品質形成規則與後續生成策略。

## 用法

1. 先看 `reference-matrix.md`，理解樣本池與 family 分類。
2. 再看 `icon-family-assignment.md`，確認任務卡與各畫面應該落在哪種 family。
3. 最後看 `notes.md`，快速掌握可落地的 icon 策略。
4. 後續若新增更多 screenshot crop，可繼續沿用同一個資料夾擴充。
