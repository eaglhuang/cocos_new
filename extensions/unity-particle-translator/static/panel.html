🎆 Unity → Cocos 3D 粒子翻譯工具

貼上 Unity .prefab 全文（以文字編輯器開啟）或只貼 ParticleSystem: 區塊。
工具會提取粒子屬性並轉換為 ParticleLayerProfile（我們專案格式）和 ParticleOverride（applyParticleOverride 參數）。

💡 Unity 操作：Project 視窗 → 右鍵 .prefab → Show in Explorer → 記事本開啟 → 全選複製。
或在 Unity Editor Settings → Text Serialization（確認資產格式為 Text）後才能開啟文字。

→ 貼入 BuffParticleProfileConfig.ts 的 variants['MyEffect'] 物件
→ 傳入 applyParticleOverride(ps, override)

※ 若 prefab 含多個 ParticleSystem（子發射器等），目前只轉換第一個 !u!198 區塊。
