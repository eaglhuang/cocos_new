**📜 《三國傳承：全球遠征》完整設計規格書與市場戰略分析 (Comprehensive Design & Market Strategy)**

## **1\. 遊戲核心大綱 (Core Concept)**

* **遊戲定位**：跨世代血統培育與全球征途策略手遊 。

* **目標受眾 (TA)**：35-50 歲資深策略玩家，偏好《三國志》系列的經營成就感與「延時滿足」決策 。

* **視覺風格**：2.5D 冷色調歷史感戰場，融合 AAA 級動態立牌與精緻 UI 。

## **2\. 遊戲主入口：大廳入口規劃 (Main Lobby Hub)**

大廳不再只是靜態選單，而是展現「傳承」與「日常動態」的核心舞台。

* **視覺表現**：背景為玩家當前的主城（如：成都、洛陽），隨現實時間產生**晝夜變化** 。

* **動態看板娘/郎**：顯示玩家當前培育中的「本體名將」。名將會在大廳執行日常動作（如：擦拭武器、讀書、與副將交談），點擊可觸發「人性化動態對話」 。

* **功能分區**：  
  * **點將台（轉蛋）**：抽取名將種子與精銳虎符 。

  * **演武場（名將挑戰賽）**：進入為期兩週的賽季競賽入口 。

  * **祖廟（英靈系統）**：查看歷代名將退役後轉化的「英靈虎符」及其傳承技能 。

  * **政廳（三國日報）**：查閱過去 24 小時的離線事件紀錄，領取「反搶」或「代勞」獎勵 。

  * **遠征（核心戰鬥）**：進入 5x8 棋盤戰場，開啟全球征途 。

## **3\. 全玩法市場優缺點深度分析**

| 核心玩法 | 市場優點 (Strength) | 市場缺點 (Weakness) |
| :---- | :---- | :---- |
|  **賽馬娘式雙軌轉蛋**  | **高營收潛力**：拆分本體與支援卡，增加課金深度；**高耐玩度**：隨機性培育讓每位武將獨一無二。 | **門檻較高**：培育過程耗時長，若挫折感控制不佳，易導致輕度玩家流失。 |
|  **英靈虎符傳承**  | **情感連結**：武將「死亡」不再是損失，而是轉化為永久資產，減輕「角色退環境」的排斥感。 | **數值膨脹壓力**：若傳承技能疊加過多，易導致後期關卡難度難以平衡。 |
|  **5x8 棋盤戰場 (無 DP)**  | **流暢爽快**：取消點數限制改用糧草扣除，戰鬥節奏更緊湊；**低門檻高上限**：AI 委任功能降低操作疲勞。 | **平衡性挑戰**：缺乏 DP 限制可能導致特定強力兵種被過度使用。 |
|  **三國日報動態敘事**  | **極強市場區隔**：目前市面上缺乏真正的「離線社交補償」，能大幅提高玩家黏著度。 | **開發成本**：需大量高品質文本與邏輯拼接，避免事件重複感過高。 |
|  **時間償還商城**  | **友善無課群體**：讓非課金玩家透過「時間」換取價值，創造良好口碑與長線留存。 | **變現節奏變慢**：可能會延後初期營收爆發期。 |

## **4\. 戰略建議：玩法與風格的極致取向**

### **A. 玩法戰略：強化「推理決策」與「社交傳承」**

* **建議**：在培育過程中加入更多「血統選擇」的決策點，例如不同地域（森林、雨天）對虎符成長的影響 ，這能滿足 TA 玩家追求經營成就感的心理。

* **社交區隔**：允許玩家將自己的「英靈虎符」作為「好友支援」借給公會成員 ，這能讓老玩家的成就感透過「提攜後進」得到滿足。

### **B. 美術風格：極致的「陳舊歷史感」與「冷色調」**

* **風格定義**：避開市面上常見的鮮豔二次元色調。採用**青灰色與暗棕色**為基調 ，強調戰場磨損、風化與陳年血跡的質感 。

* **市場區隔點**：  
  1. **動態立牌融合**：小兵與將領不再是漂浮的紙片，而是「踩」在刻印於地表的石磚格線上，並帶有軟陰影 。

  2. **高品質介面**：UI 採用重金屬鑄造與古玉鑲嵌質感 ，讓玩家在操作時感覺像是在擺弄一件精緻的歷史文物，而非點擊塑膠按鈕。
