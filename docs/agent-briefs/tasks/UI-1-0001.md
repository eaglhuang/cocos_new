## 如何驗證
1. 目標路徑下三個 PNG 存在且大小 > 0。
2. 來源與目標每個檔案 SHA256 一致（無損搬遷）。
3. `docs/ui-quality-todo.json` 中 `UI-1-0001` 的 `status` == `done`，`notes` 中有 `<path> | <SHA256> | <size bytes>` 清單。

## Unit Test 建議
**不需要撰寫程式 unit test**（此為檔案搬遷任務，無業務邏輯）。驗收替代方案為：  
PowerShell 腳本計算 SHA256 並比對，輸出貼入 `notes` 欄作為自動化驗收記錄。

## 建議作法
```powershell
# repo 根目錄
$src = "artifacts/ui-layered-frames/family-previews/nav_ink"
$dst = "assets/resources/sprites/ui_families/common/nav_ink"
New-Item -ItemType Directory -Force -Path $dst
foreach ($f in @('btn_primary_normal.png','btn_primary_pressed.png','btn_primary_disabled.png')) {
  Copy-Item (Join-Path $src $f) (Join-Path $dst $f) -Force
  $hash = (Get-FileHash (Join-Path $dst $f) -Algorithm SHA256).Hash
  $size = (Get-Item (Join-Path $dst $f)).Length
  "$dst/$f | $hash | $size bytes"
}
```
