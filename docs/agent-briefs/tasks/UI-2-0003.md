## 開單原因

## 完整描述
1. 在 `assets/resources/ui-spec/ui-design-tokens.json` 新增淺底面板所需 token。
2. 補上 `surfaceParchmentFill`、`textOnParchment`、`dividerOnParchment`、`shadowDefault`。
3. 確保後續 skin 與 runtime 可透過 token 解析，不需重複硬編碼。
