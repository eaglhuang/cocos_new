1. `header` 的玉牌結構已回來，但文字層級與置中感仍不夠穩，還沒到參考圖那種一眼讀得清楚的牌匾感。
2. `crest` 已有中心命紋，但目前仍是 proof bridge；若要進 canonical finish，仍需 final medallion 資產或更完整的 final family。
3. 左側人物舞台已更聚焦，但右側資訊卡整體仍偏平、偏糊，和參考圖的乾淨高級感還有距離。

## 下一步優先序

1. 先收 `header`：優先用 `layout / skin / shell tint` 把右上 header 從「偏亮偏平」收成 jade-parchment 玉牌階層。
2. 再收 `crest`：在 header 階層穩定後，再處理 crest face / ring 的內紋對比與嵌入感，避免同時改兩塊而失焦。
3. 最後收 `portrait frame`：把左側人物框的亮度、通透感與前後景分離往 canonical reference 拉近。
4. 只有當 `header / crest / portrait frame` 三塊都已用 `layout / skin / shell tint` 收過一輪，仍無法進到 canonical finish，才升級為 `.github/skills/dalle3-image-gen/` 補單件資產。
