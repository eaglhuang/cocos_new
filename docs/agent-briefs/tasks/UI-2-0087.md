## 前置包內容（立即可工作，不等 UI-2-0086）

### Batch A2-1：Reference Family Mapping

重新整理參考圖中的 family 對照表，為後續 compare board 建立素材基礎。

**輸出物**：`docs/agent-briefs/tasks/UI-2-0087-family-mapping.md`（或直接補在本卡下方）

候選畫面清單（每個 family 2~3 張代表畫面）：

| Family | 候選畫面 | 說明 |
|--------|---------|------|
| dark-metal | BattleMain、CharacterDetail、TeamFormation | 主戰場大容器 |
| parchment | UnitDetail-tabs、BattleLog、SkillDesc | 文字說明頁 |
| gold-cta | 各畫面的確認按鈕區域 | CTA 按鈕層 |
| destructive | ConfirmDialog、QuitBattle | 紅色警示按鈕 |
| tab | TeamFormation tabs、Shop categories | 頁籤列 |

**評分維度**（核對 FrameRecipe.shadow / .bleed / .frame 是否都清晰可見）：
- frame 邊框是否有層次感
- bleed 溢血是否柔和（無鋸齒）
- shadow 陰影是否存在且自然
- fill 底色是否乾淨（無雜訊干擾文字）

### Batch A2-3：Pilot 驗收樣板前置

**先定 compare board 的評分維度**（不依賴 UI-2-0086 的自動 QA 工具）：

| 維度 | 說明 | 觀察位置 |
|------|------|--------|
| frame 邊框自然度 | 邊框銜接是否流暢，無明顯像素鋸齒 | frame 四邊 |
| bleed 柔和度 | 溢血羽化是否自然融合背景 | frame 外緣 |
| shadow 層存在性 | 是否有明顯陰影分離 container 與背景 | frame 外側 |
| 字色對比 | 文字與背景對比是否達 4.5:1（WCAG AA） | 文字區域 |
| 背景融合 | fill 是否與鄰近 frame 自然融合，無突兀色塊 | 容器內部 |

**Pilot 測試畫面決策**：
- dark-metal: `BattleMain` — 最有代表性的深色金屬主容器
- parchment: `UnitDetail` — 文字密集的說明性容器
- gold-cta: `ShopMain` 買入按鈕行 — 正向 CTA 按鈕密度最高
