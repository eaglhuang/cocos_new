## 完整描述
1. 確認 `UI-1-0002` 已完成（目標路徑存在）。

3. 更新 SpriteFrame 路徑至 `nav_ink` 同結構的 `paper_utility` 路徑：
   - normal：`assets/resources/sprites/ui_families/common/paper_utility/btn_primary_normal`

5. 執行 JSON schema 驗證（`node tools_node/run-tests.js`）。
