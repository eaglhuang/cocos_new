# [UI-3-0100] 虎符卡片 UI 美術規格書 — 基礎兵種首批

## 目標

為戰場 `TigerTallyPanel`（虎符展示面板）建立五大基礎兵種的正式卡片美術資產，確立「虎符卡」的視覺設計語言，為後續全量特種兵卡牌（白馬義從、陷陣營等）提供可複用的製作模板。

---

## 設計規格 (Design Spec)

### 卡片結構層次

```
┌──────────────────────────────┐
│  [品質角標 QualityBanner]        │  ← 右上角三角形色塊（UR=彩虹/SSR=金/SR=銀/R=銅）
│                              │
│     [兵種插畫 TroopArt]          │  ← 主體插圖（武將剪影 or 部隊群像）
│                              │
│     [兵種標誌 EmblemGlyph]       │  ← 中央兵種象徵圖騰（512x512 獨立圖層）
│                              │
│  [虎符篆文 TallyScript]          │  ← 底部古文字裝飾（虎符兩字古篆）
└──────────────────────────────┘
```
