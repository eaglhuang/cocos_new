# P0-1 — 搬遷 `nav_ink` family（Agent2）

短述
- 將 `nav_ink` family 的預覽 PNG 搬遷到 runtime 路徑，並提供每個檔案的 SHA256 與檔案大小以供驗證。

步驟
1. 確認來源資料夾存在：`artifacts/ui-layered-frames/family-previews/nav_ink/`
2. 建立目標資料夾（若不存在）：`assets/resources/sprites/ui_families/common/nav_ink/`
3. 複製以下檔案：`btn_primary_normal.png`、`btn_primary_pressed.png`、`btn_primary_disabled.png`
4. 對每個目標檔計算 SHA256 並記錄 `<path> | <SHA256> | <size bytes>` 列表
5. 在本檔 frontmatter 與 `docs/ui-quality-todo.json` 中把 `status` 更新為 `done`，並在 `notes` 欄填入檔案清單

PowerShell 範例
```
# repo 根目錄
$src = "artifacts/ui-layered-frames/family-previews/nav_ink"
$dst = "assets/resources/sprites/ui_families/common/nav_ink"
New-Item -ItemType Directory -Force -Path $dst
Copy-Item -Path (Join-Path $src 'btn_primary_normal.png'), (Join-Path $src 'btn_primary_pressed.png'), (Join-Path $src 'btn_primary_disabled.png') -Destination $dst -Force
Get-FileHash (Join-Path $dst 'btn_primary_normal.png') -Algorithm SHA256 | Format-Table
```

驗收準則
- 目標檔案存在且大小 > 0
- 來源與目標的 SHA256 一致
- `docs/ui-quality-todo.json` 對應 task `P0-1` 的 `status` == `done`，並包含 `notes` 列表

回報格式（已完成時）
- 每行：`<path>` | `<SHA256>` | `<size bytes>`
- 完成的 task id：`P0-1`
- 簡短 summary（1 段話，3 行）：完成內容、總檔案數、特殊狀況或阻塞
