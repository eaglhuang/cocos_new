## 開單原因

`docs/UI參考圖品質分析.md (doc_ui_0051)` (doc_ui_0051) §2.6 明確記錄兩類 Tab 框體：
1. **水平 Tab**（已由 UI-1-0005 完成）
2. **垂直菱形 Tab**（圖2 裝備介面右側「詳情/戰法/裝備/將魂」選單卡）— 本卡負責

§7.2「需補強」項目中：「Tab 框體家族 — 垂直菱形 Tab **完全缺失**」。  
§2.6 說明：「菱形 Tab：非標準矩形 → **不適用 9-slice**，用獨立 sprite 或 shader 旋轉」

## 完整描述

- **目標路徑**：`assets/resources/sprites/ui_families/common/diamond_tab/`
- **注意**：這是**非 9-slice** 的獨立固定尺寸 sprite，不做拉伸
- `diamond_active.png`（56×56）
  - 菱形多邊形（4 頂點，距邊各留 2px）
  - 金色漸層填充：頂部 `#FFE064` → 底部 `#B48214`
  - 2px 金色邊框（`#C9A028`）+ 1px 內側拋光亮線
- `diamond_inactive.png`（56×56）
  - 同形狀，填充暗墨色 `#2A2520`，alpha=200（約 78%）
  - 1.5px dim 金邊（去飽和 50%）

## 使用場景

| 畫面 | 元素 | 說明 |
|---|---|---|
| 裝備介面（§6.2） | 右側菱形 Tab 列 | active 金色 + inactive 暗墨，包含 icon + 文字 label |

## 注意事項

- **不使用 9-slice**：菱形切角在任何拉伸方向都會變形，必須作為固定尺寸 sprite 使用
- 實際使用時，每個 Tab 按鈕寬高固定，不動態調整尺寸
- icon 內容（文字/圖標）由 Cocos 在此 sprite 上方疊加，本 sprite 只提供背景框

## 如何驗證

1. 兩個 PNG 存在且 `size > 0`
2. 菱形（非矩形）形狀清晰，四角為透明背景
3. diamond_active 底色為金色系，diamond_inactive 底色為深灰系
