把 QA 從「只看單張 PNG」提升為「回到真實畫面條件比對」，避免圖本身看起來精緻，但一放回 UI 就失去層次或辨識度。

## 完成內容

- 建立正式規格：
  - [screen-context-qa-spec.md](C:\Users\User\3KLife\artifacts\ui-qa\UI-2-0051\screen-context-qa-spec.md)

  - `reference board`

## 主要規則

- 所有量產圖都必須先進 `reference board`
- 至少做一組 `v1 / v2 / v3` compare board
- icon 至少回看 `64 / 32`
- portrait 至少回看 crop proof 與 `64 / 32`
- card / container / torso 至少回看 `256 / 128`
- 最後一定要放回對應畫面檢查：
  - HUD
  - panel
  - cell
  - reward screen
