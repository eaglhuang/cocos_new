## 開單原因
general / support / shop / gacha 類畫面包含較多卡片、列表與細節面板。若直接與首波 nav / popup 一起修改，review 範圍會過大，也不利於觀察不同畫面族群是否需要額外命名例外。

## 完整描述
1. 更新下列 skin：`general-detail-default.json`、`general-list-default.json`、`general-portrait-default.json`、`general-quickview-default.json`、`support-card-default.json`、`shop-main-default.json`、`gacha-default.json`。
2. 依 `UI-2-0013` 的命名模板，為主要卡面 / 面板補上 `*.shadow` slot。
3. 若某些 skin 同時有主卡與子卡陰影需求，需先在 notes 記錄命名規則，再落地到 slot。
4. 所有新增 shadow slot 均引用 `sprites/ui_families/common/shadow/shadow_01`，`spriteType=sliced`，`border=[20,20,20,20]`。
5. 完成後更新 manifest 與 umbrella task 的 notes，註明第二波覆蓋清單。
