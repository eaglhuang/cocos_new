# UI-2-0091 資產方向 Pilot 收斂

## 目標

為高頻 UI family 建立可重複使用的正式美術方向，避免每次都從零猜風格。此卡負責定義 family 的視覺語言、禁則、比較基準與驗收方向，供後續 `ArtRecipe`、正式切圖與 runtime 驗證使用。

目前重點 pilot：
1. `dark-metal`
2. `parchment`
3. `gold-cta`
4. `jade-parchment` for `GeneralDetailBloodlineV3`

## 驗收目標

- [ ] 每個 pilot family 都有清楚的方向說明、禁則與 compare 重點
- [ ] family 能對應到 `frame / bleed / shadow / fill / ornament` 的 slot 類型
- [ ] compare board 能明確指出「像參考圖」與「偏離方向」的差異
- [ ] direction 文件可直接支援 `UI-2-0089` 與 `UI-2-0092`

## 2026-04-05 Story Strip Pilot

  - 禁止過度依賴正臉辨識角色

## 2026-04-05 美術總監風格校正

### 正確方向

- `jade-parchment` 必須走扁平化、裝飾化、卷軸化，不可走寫實玉石材質。
- 整體要偏 `水墨渲染風 + 羊皮紙底 + 細金線點綴`。
- 玉件應像嵌飾、符板、卷軸端飾，不應像厚重寶石或立體獎章。
- header、主資訊框、crest、rarity badge 必須屬於同一套 family 語言。

### 明確禁止

- 禁止寫實玉石高光、厚 bevel、強鏡面反射。
- 禁止黑金重金屬、重裝甲、厚 medal 感。
- 禁止像外掛貼片的突兀框體。
- 禁止只為單一尺寸好看，卻破壞 9-slice 延展規則。

### Pilot 驗收補充

- compare board 必須額外標出 `flatness / ink-wash / jade ornament / gold-line restraint` 四個檢查點。
- 若視覺仍偏寫實玉石或厚重金屬，視為 direction 不合格，不得往下送正式切圖。
