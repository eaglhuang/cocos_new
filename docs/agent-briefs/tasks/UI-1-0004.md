## 開單原因
多個 UI 家族的面板效果需要共用的 shadow 陰影紋理與 noise 質感紋理（金屬/宣紙），目前這些共用資產尚未建立，導致各 skin 無法引用，影響視覺質感的一致性。

## 完整描述
- **目標路徑**：
  - `assets/resources/sprites/ui_families/common/shadow/shadow_01.png`（9-slice 可拉伸）
  - `assets/resources/sprites/ui_families/common/noise/metal_noise_256.png`
  - `assets/resources/sprites/ui_families/common/noise/paper_noise_256.png`
- `shadow_01.png`：需設計為 9-slice 可拉伸，用於面板投影效果。
- `metal_noise_256.png`：256x256 金屬質感噪點紋理，用於戰鬥/裝備類按鈕表面。
- `paper_noise_256.png`：256x256 宣紙質感噪點紋理，用於 paper_utility 類按鈕表面。
- 產生後計算 SHA256 與大小，登記至 `notes` 與 manifest。

## 如何驗證
1. 三個目標 PNG 存在且大小 > 0。
2. `shadow_01.png` 可作為 9-slice（長寬 >= 64px，border 為 `[20,20,20,20]`）。
3. manifest `UI-1-0004` 的 `status` == `done`，`notes` 含完整檔案清單。
