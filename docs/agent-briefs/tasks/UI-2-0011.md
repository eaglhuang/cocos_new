## 開單原因
參考圖的材質感不是只靠顏色，而是靠 fill 層上的細噪點。現在 noise 紋理已存在，但 runtime 還沒有疊加入口。

## 完整描述
1. 在 `UIPreviewBuilder` 中加入可選的 noise overlay 子層。
2. 支援 `metal_noise_256` / `paper_noise_256` 兩種紋理來源。
3. opacity 等參數優先由 token 或 skin config 控制。
4. 確保 overlay 不影響文字可讀性與互動。

1. 啟用 noise 的畫面可在預覽中看見細紋理。
2. 未啟用 noise 的畫面不受影響。
3. acceptance 驗證通過。
