## 前置包 Batch A2-2：AI 資產治理盤點（立即可工作）

### 可 AI 生成的資產類型（草案）

> 可 AI 生 = 可作為製作起點，但須人工後處理與 approval 才能入庫。

| 資產類型 | 說明 | 後處理需求 |
|---------|------|----------|
| metal grain / noise 貼圖 | 金屬表面雜訊紋理 | 去邊、調色、確保 tiling 無縫 |
| paper noise 貼圖 | 羊皮紙/木頭紋理 | 去噪、色調統一、確保柔和 |
| metal sheen / 高光反射 | 金屬光澤層 | 調對比、確保不過曝 |
| corner ornament（角落裝飾） | 角落紋樣、印章 | 去白邊、PNG 去背、確認比例 |
| edge glow 羽化 | 框邊向外的霞光 | 調透明度曲線 |

### 禁止直接 AI 生成的資產類型（草案）

> 禁止直接 AI 生 = 結構性精度要求高、或品牌識別不允許 AI 隨意詮釋。

| 資產類型 | 原因 |
|---------|------|
| 九宮格主框體貼圖 | 邊長精度與設計規格強耦合，AI 難以保證 border 尺寸準確 |
| 角色立繪/頭像 | 品牌識別，需由美術定版 |
| 陣營 logo / 徽章 | 品牌識別，不允許 AI 詮釋 |
| 字型相關資產 | 版權問題 |
| button fill 主體色 | 需保持色彩設計一致性，白模先行 |

### ArtRecipe 欄位草案

接在 `MaterialRecipe.aiGenerated = true` 時，`artRecipeRef` 指向的 ArtRecipe id：

```jsonc
// artifacts/ui-source/ai-recipes/{id}.art-recipe.json
{
  "id": "dark-metal-grain-r1",
  "version": 1,
  "outputAssetPath": "textures/materials/metal-grain-dark",
  "tool": "Stable Diffusion XL / Midjourney / ...",
  "prompt": "seamless dark metal texture, no specular, 512x512, game UI asset",
  "negativePrompt": "logo, text, watermark, bright colors",
  "seed": 12345678,
  "postProcess": ["去邊", "調色至 #1a1a2a 色調", "確認 tiling 無縫"],
  "approvalStatus": "pending",  // pending | approved | rejected
  "approvedBy": null,
  "approvedAt": null,
  "basedOn": null,              // 若是從其他 recipe 衍生，填原始 id
  "notes": ""
}
```

### 命名規則（與正式資產對齊）

```
artifacts/ui-source/ai-recipes/{family}-{asset-type}-r{revision}.art-recipe.json
```

範例：
- `dark-metal-grain-r1.art-recipe.json`
- `parchment-noise-r2.art-recipe.json`
- `gold-cta-sheen-r1.art-recipe.json`
