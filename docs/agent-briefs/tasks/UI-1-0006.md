## 開單原因

`docs/UI參考圖品質分析.md (doc_ui_0051)` (doc_ui_0051) §4.2 說明框體「自然融合感」需要三層疊加（frame → bleed → shadow）。目前 shadow 層已有 `shadow_01.png`，但 **bleed 暈邊層** 完全缺失，導致框體邊緣留下「貼紙感」（硬切邊）。

§9.6 規範明確要求：`*.bleed` slot 必須宣告（可設透明但 slot 不可省略）。本卡為補全 bleed 素材的前置步驟。

## 完整描述

- **目標路徑**：`assets/resources/sprites/ui_families/common/bleed/`
- `bleed_dark.png`（128×128）
  - 適用於：深色金屬框體（dark-metal family）
  - 中央 80×80 區域透明（框體本體可見）
  - 外圈：多層暖銅色柔光暈（`#C9A028` 系）從外到內 alpha 45→90→60→35→18
  - 視覺效果模擬 Gaussian blur 外發光
  - 9-slice border `[24, 24, 24, 24]`
- `bleed_parchment.png`（128×128）
  - 適用於：羊皮紙框體（parchment family）
  - 中央同樣透明，外圈暖棕色柔光暈（`#5A3A1A` 系）alpha 35→70→50→28→14
  - 9-slice border `[24, 24, 24, 24]`

## 使用方式

在 skin slot 中，bleed sprite 疊放於 frame 正下方（比 frame 大 8~16px），比 shadow 還高一層：

```
z-order（由下往上）：
  shadow  ← shadow_01.png（最底）
  bleed   ← bleed_dark.png / bleed_parchment.png（次底）
  frame   ← 框體主 sprite（最頂）
```
