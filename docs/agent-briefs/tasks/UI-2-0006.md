## 開單原因
`nav_ink`、`paper_utility`、`warning` 三組共享按鈕素材已搬入 runtime，但目前缺少一張專屬任務卡來驗證所有引用點是否都遵守 `[20,20,20,20]` 的九宮格規範。

## 完整描述
1. 搜尋所有引用 shared button family 的 skin slot。
2. 檢查 `border` 是否統一為 `[20, 20, 20, 20]`。
3. 對照目前按鈕高度，確認 `20 + 20 < height` 的拉伸條件成立。
4. 若專案尚無對應腳本，新增或擴充驗證腳本到 acceptance 流程。
5. 將驗證結果寫回 manifest 與 `notes`。
