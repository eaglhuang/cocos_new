## 背景與動機

§9.6「文件 §2~§6 的可執行化對照」列出 5 條可自動驗證的品質規則：

| 規則 | 驗證條件 |
|---|---|
| §2.1 角區 20~24px | dark-metal 框體 border ≥ [20,20,20,20] |
| §3.3 金色邊框永遠存在 | 所有框體 sprite 包含 ≥ 1px 金色邊界線（目視規則，腳本做路徑存在確認） |
| §4.2 bleed 層必須存在 | skin slots 中 `*.bleed` 不可省略 |
| §4.3 主面板不透明度 85~95% | skin fill slot 的 color alpha 在 `DD`~`F0` (`#221~240`) 範圍 |
| §5.4 深底文字必有描邊 | 深底上的 label-style `outlineWidth ≥ 1` |
| §5.2 淺底文字不加描邊 | 淺底上的 label-style `outlineWidth = 0` |
