## 開單原因

`docs/UI參考圖品質分析.md (doc_ui_0051)` (doc_ui_0051) §2.1 定義「深色金屬框（Dark Metal Frame）」為**所有資訊面板的主框體家族**——武將詳情右側數值面板、裝備詳情卡、技能欄位均屬此家族。但 `assets/resources/sprites/ui_families/common/` 下完全沒有 `dark_metal/` 路徑，導致：

- §7.3「框體分類完整性」評分 3/10，此家族是最大的分類缺口
- §9.2 的按鈕家族 ↔ 框體家族交叉映射無法成立（`nav.ink` / `equipment.primary` 的面板環境缺失）

## 完整描述

- **目標路徑**：`assets/resources/sprites/ui_families/common/dark_metal/`
- `frame.png`（80×80）
  - 填充：深褐 `#1E1A12`，alpha=238（93%），含頂部至底部極微暖褐漸層
  - 邊框：2.5px 金色漸層（`#C9A028` 主調），+ 內側 1px 高光亮線
  - 圓角：R=8px（偏方正、鋼硬感，符合 §2.1 規範）
  - 四角金屬光點：2px 亮金圓點，強調金屬鉚釘感
  - 9-slice border：`[20, 20, 20, 20]`
- `fill.png`（64×64）
  - 純深色填充層，可無縫拉伸；`#1C1810`，alpha=238，+微暖褐垂直漸層
- `bg.png`（64×64）
  - 底層背景，alpha=220，中心微亮 PathGradient（增加面板中心深度感）

## 如何驗證

1. 三個 PNG 存在且 `size > 0`
2. `frame.png` = 80×80，視覺上為深褐底+金色圓角邊框+四角亮點
3. `fill.png` + `bg.png` = 64×64，可作為 9-slice 中間拉伸區
4. 9-slice border `[20,20,20,20]`：角區含金屬光點，拉伸後中段為純深色
