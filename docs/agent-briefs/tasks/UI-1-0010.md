## 背景與動機

§2.7 item-cell 框體家族規格指出 item cell 需有：
- **左上角稀有度標籤 badge**：小型 tag sprite（N/R/SR/SSR/UR/LR） 
- **右下角數量 badge**：數字疊加框（暗色半透明底）
- **Lv badge**（可選）：等級指示標

§6.3 戰令商城的 RANK 圓形 badge（深色底 + 粗金環）。

§6.4 信件系統的**紅色通知 badge**（99+）。

這些 badge sprite 是 item-cell 家族的「標注層」，目前完全缺失，導致 §7.3「物品格子標準化」2/10。
