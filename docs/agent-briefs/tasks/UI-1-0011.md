## 背景與動機

§2.4 Gold CTA Frame 描述的是**最高互動層級**的框體家族（升級/試玩角色/購買進階戰令按鈕）。  
目前 `commerce/` 和 `equipment/` 目錄只有**按鈕 sprite**；沒有 `gold_cta/` 的獨立**面板框體**（frame/fill/bg）。

§9.2 指出 `commerce.primary` 按鈕住在 Gold CTA 框體內：框體本身就是按鈕的「場域」（如按鈕周圍的粗金邊容器面板）。若要把大型 CTA 卡、推廣橫幅、戰令升級區域做成獨立面板而非只有按鈕，需要有 frame/fill/bg 三層。

**§2.4 規格摘要**：
- 邊框：3~4px 粗金色漸層（`#C9A028` → `#FFE088` → `#C9A028`）
- 填充：深褐 `#2A2018` 或深黑 `#1A1510`
- 圓角：6~10px
- 外 glow：`#FFE08840`，blur 4px（用 bleed 層模擬）
- 角區保留：24~32px（邊框粗）
