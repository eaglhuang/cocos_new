# [UI-2-0049] 建立 family 批次生產模板

## 目標

把量產流程從「單張補圖」改成「同功能 family kit 一次做完」，確保風格、材質、狀態與縮圖表現保持一致。

## 完成內容

- 建立正式規格：
  - [family-batch-production-spec.md](C:\Users\User\3KLife\artifacts\ui-qa\UI-2-0049\family-batch-production-spec.md)
- 針對下列類型定義 family kit：
  - icon
  - portrait
  - card
  - container
  - torso
- 建立命名模板與 QA 要求：
  - `carrier`
  - `glyph`
  - `badge`
  - `size_set`
  - `reference board`
  - `compare board`

## 主要規則

- 先做完整 family kit，再從 kit 長出單張圖
- icon 類至少一起做 `normal / selected / pressed / disabled`
- portrait 類至少一起定義 `crop.rule / frame.rule / badge.safe-zone`
- card / container / torso 類必須先定義 `silhouette.rule / material.rule / value.signal`
- 所有 family kit 都要附 `32px` 或等比例縮圖 QA
