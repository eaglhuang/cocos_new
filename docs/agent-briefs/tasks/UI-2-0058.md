# [UI-2-0058] 收斂 GeneralDetailBloodlineV3 第一批 Slot 契約

讓 `GeneralDetailBloodlineV3` 的第一批視覺骨架具備可重用的正式 slot family，避免 `ui-spec`、runtime 與後續美術 proof 各自長出不同版本。

1. `portrait` 區有正式 carrier / image / 未持有提示 slot。

5. 正式規格與交叉索引已同步記錄 `UI-2-0058 -> UI-2-0059 -> UI-2-0061` 的收斂順序。
