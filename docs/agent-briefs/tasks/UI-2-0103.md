# UI-2-0103 Data Showcase Freeze

## 目標

把 `assets/resources/ui-spec/content/general-detail-overview-states-v1.json` 從「smoke-only samples（全部 legendary）」升級為「正式展示樣本 package」，讓後續 Agent / QA 可用真實多樣化資料驗收整套 UI。

## 現況與缺口

| 項目 | 現況 | 目標 |
|------|------|------|
| 已有樣本 | 9 個（cao-cao, liu-bei, sun-quan, zhuge-liang, zhou-yu, guan-yu, zhang-fei, zhao-yun, lu-bu, sima-yi） | 保留，補充展示樣本 |
| rarityTier 覆蓋 | 僅 `legendary` | 補 `common`, `rare`, `epic`；確認 `mythic` 是否需要 |
| crestState 覆蓋 | `rumored` ✓、`revealed` ✓、`awakened` ✓（lu-bu） | 維持三態覆蓋 |
| 陣營覆蓋 | wei / shu / wu ✓ | 補充非三國主陣營或女性角色 |

## Schema 注意事項

- `rarityTier` 的 enum 目前為 `["common", "rare", "epic", "legendary"]`
- `UI-2-0097` 已處理 mythic badge，若需 `mythic` tier 展示樣本，**必須先更新 schema 的 enum 欄位**
- 在決定是否加入 mythic 前，以現有 4 tier 補齊為優先

## 展示樣本設計建議

### showcase-huang-zhong（rare tier — 老將 / 非典型）
- `rarityTier: "rare"`，陣營：shu，定位：先鋒
- 覆蓋：較低稀有度的 UI 顯示、老將形象的 coreStats 分布
- `crestState: "rumored"`

### showcase-diao-chan（epic tier — 女性 / 社交型）
- `rarityTier: "epic"`，陣營：neutral，定位：策略
- 覆蓋：epic tier accent 色、女性角色的 portraitResource 命名（`diao_chan_portrait`）
- 政治/魅力型 stats（武低魅高）
- `crestState: "rumored"`

### showcase-gongsun-zan（common tier — 基礎邊境武將）
- `rarityTier: "common"`，陣營：neutral，定位：先鋒
- 覆蓋：common tier 最低視覺權重、無特殊 awakening 進度
- `awakeningProgress: 0.1`，展示幾乎空的進度條
- `crestState: "rumored"`

## 必填欄位清單

所有新增樣本必須覆蓋 `general-detail-overview-content.schema.json` 的所有 28 個 required 欄位。
欄位清單：headerTitle, headerName, headerMeta, rarityLabel, rarityTier, portraitModeHint, coreStatsTitle, coreStatsValue, roleTitle, roleValue, traitTitle, traitValue, bloodlineTitle, bloodlineName, awakeningLabel, awakeningProgress, personalityLabel, personalityValue, bloodlineCardTitle, bloodlineBody, crestTitle, crestHint, crestState, crestGlyphPrimary, crestGlyphSecondary, crestFaceResource, storyCells（6 cells）, portraitResource

**禁止**：留 "???" 或空字串在展示樣本中。所有欄位需有語意上合理的內容。

## storyCells 節奏規範

六格故事帶每格應使用 4–6 個漢字的短語，節奏整齊、可閱讀。格序：
1. `origin` — 出身環境
2. `faction` — 入伍 / 歸屬
3. `role` — 戰場定位行為
4. `awakening` — 覺醒轉折
5. `bloodline` — 血脈命紋意象
6. `future` — 結局 / 遺留懸念

## 交付目標

1. `assets/resources/ui-spec/content/general-detail-overview-states-v1.json` 版本升至 v6，新增 3 個正式展示樣本（`showcase-huang-zhong`, `showcase-diao-chan`, `showcase-gongsun-zan`）
2. 三個新樣本分別覆蓋 `rare`, `epic`, `common` tier
3. 若需加入 `mythic` 樣本，須同步更新 schema enum（需另行決策，不強制本卡完成）
