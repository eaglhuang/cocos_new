# [UI-2-0032] 修正 BattleScene Icon 品質差異（v2）

## 目標

根據 `UI-2-0028` 的第一張候選稿與 `docs/UI品質參考圖/` 的比對結果，做出更貼近正式美術語言的 BattleScene icon v2。

## 修正重點

- 降低過亮、過飽和的藍紅配色
- 強化金屬做舊、內陰影、噪點與邊緣磨損
- 調整 carrier 語言，避免過度乾淨的對稱圓章感
- 加粗主 glyph，確保 32x32 仍能辨識
- 產出 128 / 64 / 32 三種尺寸預覽

## 驗證

1. v2 與參考圖對照時，不再出現明顯的「過亮、過乾淨、過現代」觀感。
2. 32x32 尺寸下主 glyph 仍清楚可辨。
3. 人工 review 可接受作為 BattleScene icon family 的第一版風格母版。
