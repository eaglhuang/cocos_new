# [UI-2-0050] 建立狀態組與尺寸組標準

## 目標

統一各類量產素材應同時輸出的狀態與尺寸，避免每張卡再各自決定要不要做 `pressed`、要不要出 `32px`。

## 完成內容

- 建立正式規格：
  - [state-size-standard.md](C:\Users\User\3KLife\artifacts\ui-qa\UI-2-0050\state-size-standard.md)
- 定義主要類型：
  - icon
  - portrait
  - card / container / torso

## 主要規則

- icon 預設 `state_set`
  - `normal`
  - `selected`
  - `pressed`
  - `disabled`
- icon 預設 `size_set`
  - `128`
  - `64`
  - `32`
  - `24` 僅 micro badge 類
- portrait 預設至少出：
  - `proof-a`
  - `proof-b`
  - `64`
  - `32`
- card / container / torso 預設至少出：
  - `512`
  - `256`
  - `128`
- 所有生圖 brief 之後必須顯式填 `state_set` 與 `size_set`
