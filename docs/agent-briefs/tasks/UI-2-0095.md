# UI-2-0095 GeneralDetail Crest Formal Production

## 目標

生產 `GeneralDetailOverview` 專用的正式 crest medallion 家族，取代目前暫時升格的候選版。

## Art Direction

- 必須符合 `Warm Parchment Base + Jade Header Chrome + Controlled Dark Contrast + Readable Atlas Hierarchy`
- `jade` 是 accent，不是厚重深綠主底
- 不可使用參考圖截圖、proof bridge、source flat sample 當正式件
- `face` 要像卡面內建命紋，不像貼圖貼片
- `ring` 要提供結構秩序，不可搶過 face

## 生產範圍

1. `crest_face_formal`
2. `crest_ring_formal`
3. 若需要，補 `crest_socket_formal`

## 交付幾何規格

- `crest_face_formal` 與 `crest_ring_formal` 都是**固定尺寸單件 sprite**，不是 9-slice，也不是一整張素材板的一部分。
- `crest_face_formal` 必須只包含 medallion 內部圖面，不可把外圈 ring 一起畫死在同一張裡。
- `crest_ring_formal` 必須是獨立外圈，中心留空，方便 runtime 疊在 face 外層。
- 若需要 `crest_socket_formal`，也必須獨立成單張，不可與 face/ring 烤在一起。
- 每個交付檔都必須是一張資產對一個檔名，禁止輸出 `2 in 1`、成對拼版、整頁候選板。

## AI 生成禁則

- 禁止讓 AI 一次輸出 `face + ring + glow` 的整套合成圖，再靠裁切硬拆。
- 禁止輸出帶白底展示框、投影底板、說明字、樣張排版。
- 禁止用「完整徽章示意圖」冒充可接線資產；AI prompt 必須明講 `single isolated asset`。
- 若 AI 輸出的是展示稿而不是可接線單件，視為無效交付。

## 推薦 Prompt 模板

### `crest_face_formal`

```text
2D Game UI Asset. Single isolated crest medallion face only. Chinese fantasy. Warm parchment base with pale jade dragon emblem carved inside a circular seal. Flat premium graphic design, elegant and breathable, controlled dark contrast. NO outer ring, NO socket, NO glow plate, NO frame, NO background card, NO mockup, NO text, NO letters, NO words. Transparent or clean isolated background. Deliver exactly one asset.
```

### `crest_ring_formal`

```text
2D Game UI Asset. Single isolated ceremonial outer ring for a crest medallion only. Chinese fantasy. Antique gold ring with small jade inlays and fine cloud/geometric engravings. Center empty and clean so another face asset can be layered inside. Flat premium graphic design. NO inner emblem, NO face art, NO background plate, NO mockup, NO text, NO letters, NO words. Deliver exactly one asset.
```

### `crest_socket_formal`（若需要）

```text
2D Game UI Asset. Single isolated crest socket plate only. Chinese fantasy. Pale jade-parchment support plate for a medallion, subtle and low-contrast, designed to sit behind a crest face and ring. Flat premium UI graphic. NO emblem, NO outer ring, NO combined full crest, NO text, NO letters, NO words. Deliver exactly one asset.
```

## Reject If

- 看起來像深綠 relic / 黑金裝備徽章
- 有 AI 燒字、logo、殘字
- 套進 runtime 後右下角又退回白盒或浮貼感
- 明顯比 `r24-card-family-rebalance` 更重、更髒、更不透氣
- `face` 與 `ring` 沒有拆件，導致 runtime 無法分層疊合
