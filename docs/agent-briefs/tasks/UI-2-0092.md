## 目前退件原因

- runtime 仍混用 `proof` 與舊 `v3_parts` 路徑，尚未完成 final family 切換。
- `jade panel` 尚未完整，仍存在缺檔或不一致的 panel 組件。
- 視覺上仍偏寫實玉石與硬切片感，未達到美術總監要求的扁平 `jade-parchment` 方向。
- 目前交付結果看起來不像完整的 9-slice panel kit，而比較像將大圖裁片後直接掛上。

## 重做禁則

### 必須做到

- `jade-parchment` 要偏扁平、裝飾化、水墨渲染感，不可偏寫實玉石。
- `header / info panel / crest / badge` 要屬於同一個 family 語言。
- 正式 panel 必須為 9-slice 使用而設計，不可只在單一尺寸成立。
- `crest medallion` 要偏符紋、印章、紋章感，不可偏厚重金屬獎章。
- `rarity badge` 要融入同一套 jade/parchment family，不可像外掛商業貼紙。

### 明確禁止

- 禁止寫實玉石高光、厚 bevel、強鏡面反射。
- 禁止黑金重金屬、重裝甲、厚 medal 感。
- 禁止以 proof 路徑充當 final 成果。
- 禁止只做單張好看的大圖，再交給 runtime 硬撐尺寸。
