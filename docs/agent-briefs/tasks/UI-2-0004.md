## 開單原因
淺底面板若沿用深色 UI 的白字加描邊，會破壞參考圖中的宣紙閱讀感，因此需要獨立 typography preset。

## 完整描述
1. 在 token 中定義 `bodyOnParchment` / `labelOnParchment`。
2. 讓 `UISkinResolver` 支援 `label-style.style` preset 解析。
3. 預設淺底字取消描邊，維持深褐色字色。
