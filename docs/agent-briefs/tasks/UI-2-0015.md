## 完整描述
1. 更新下列 skin：`battle-hud-default.json`、`battle-log-default.json`、`action-command-default.json`、`deploy-panel-default.json`、`unit-info-panel-default.json`、`tiger-tally-default.json`、`style-check-default.json`。
2. 依前兩波共識，為主要容器、卡面或浮層補上 `*.shadow` slot。
3. 若戰場 HUD / system 面板出現不同尺寸需求，需在 notes 標記例外與原因。
4. 所有新增 shadow slot 均引用 `sprites/ui_families/common/shadow/shadow_01`，`spriteType=sliced`，`border=[20,20,20,20]`。
5. 完成後回寫 `UI-2-0007`，將 umbrella task 轉為可關閉狀態。
