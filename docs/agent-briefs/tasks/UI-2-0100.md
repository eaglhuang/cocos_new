# [UI-2-0100] TigerTally 兵種卡圖資產 Suite

## 目標

為 `TigerTallyPanel` 建立每個兵種的卡面主圖資產，補齊卡片內容層的識別度，與兵種符號 suite 分工。

## 範圍

- `cavalry / infantry / shield / archer` 的卡面主圖或剪影
- TigerTally 卡片內容區主圖路徑與 runtime 接線
- 卡圖裁切、透明背景、對位與 readability QA

## 不包含

- badge underlay / troop-type glyph
- ATK / HP / Cost 小 icon
- 卡框、卡底、cost badge、rarity border

## 風格要求

- 圖像屬於內容層，不得把框體層烘進資產
- 要服從 battle scene 的深墨戰場語言，輪廓在暗底上仍要清楚
- 可帶水墨、剪影、軍種輪廓語言，但不可變成寫實插畫卡牌風

## 驗收

- TigerTally 四張卡的兵種主圖不再依賴 placeholder/fallback
- 圖像與 badge、rarity、cost 元件互不打架
- runtime capture 可清楚區分四種兵種
- 資產透明背景正確，仍能讓框體層與內容層分離
