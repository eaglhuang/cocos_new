## Phase G 第一版欄位草案（proof-contract-v1）

### 檔案位置

```
assets/resources/ui-spec/proof/
  proof-contract.schema.json     ← JSON Schema
  screens/{screenId}.proof.json  ← 每張畫面獨立 proof contract
```

### proof-contract-v1 JSON 結構

```jsonc
{
  "$schema": "../proof-contract.schema.json",
  "version": "1.0",
  "screenId": "ShopMain",
  "proofSource": "figma://file/XXX?node=YYY",  // 或 "ref://docs/UI品質參考圖/shop-main.png"
  "capturedAt": "2026-04-05",
  "capturedBy": "Agent2",
  "confidence": 0.85,                         // 0~1，Agent2 自評與參考圖的對齊度

  // 視覺區域劃分（Zone = 畫面的語意區塊）
  "visualZones": [
    {
      "id": "header",
      "label": "頂部標題列",
      "family": "dark-metal",           // 對應 FrameFamily
      "frameRecipeRef": "dark-metal-v1", // 對應 FrameRecipe.id
      "bounds": { "x": 0, "y": 0, "w": 1920, "h": 120 }  // 設計稿座標
    },
    {
      "id": "item-grid",
      "label": "商品格陣列",
      "family": "item-cell",
      "frameRecipeRef": "item-cell-v1",
      "bounds": { "x": 0, "y": 120, "w": 1920, "h": 800 }
    }
  ],

  // 組件意圖（Component Intent = 每個節點的語意宣告）
  "componentIntents": [
    {
      "zone": "header",
      "nodeHint": "panel",         // 對應 UINodeType
      "skinSlot": "header.bg",
      "textKey": "shop.title",
      "notes": "深色金屬框＋金色字體"
    },
    {
      "zone": "item-grid",
      "nodeHint": "scroll-list",
      "skinSlot": "item-cell.frame",
      "bind": "shop.items[]"
    }
  ],

  // 排版比例系統
  "spacingRecipe": {
    "containerPadding": 24,
    "itemSpacing": 16,
    "sectionGap": 32,
    "unitBasis": 8             // baseline grid（8pt / 4pt）
  },

  // 內容插槽宣告（與 content contract 對齊）
  "contentSlots": [
    { "id": "title",       "type": "label",  "placeholder": "商城" },
    { "id": "item-list",   "type": "list",   "minCount": 1, "maxCount": 20 },
    { "id": "cta-button",  "type": "button", "family": "gold-cta" }
  ]
}
```

### 欄位邊界原則

| 欄位 | 負責人 | 備註 |
|------|--------|------|
| `visualZones` | Agent2 負責填，Agent1 負責讀 | Agent2 從截圖/Figma 分析並填入 |
| `frameRecipeRef` | Agent2 依 family 分析填入 | 必須對應 FrameRecipe.id |
| `componentIntents` | Agent2 草稿，Agent1 scaffolder 讀取生成 layout | |
| `spacingRecipe` | Agent2 從設計稿量取 | 對齊 ui-design-tokens.spacing |
| `contentSlots` | Agent2 草稿，需與 ContentContractRef.requiredFields 對齊 | |

### Scaffolder 讀取流程（UI-2-0081 接點）

```
proof.json 存在
  ↓
scaffolder 讀 visualZones → 預設 family
scaffolder 讀 componentIntents → 預填 layout node types
scaffolder 讀 contentSlots → 預填 ContentContractRef.requiredFields
scaffolder 讀 frameRecipeRef → 預填 screen.recipeRef.frameRecipeId
  ↓
輸出 layout.json + screen.json 草稿（不含 skin 細節，仍需人工確認）
```
