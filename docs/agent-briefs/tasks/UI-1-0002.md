## 開單原因
`duel-challenge` 畫面的拒絕/工具型按鈕應使用 `paper_utility` family（淺宣紙 / 中性底），但 runtime 路徑尚未建立，skin JSON 無法引用。

- **來源**：`artifacts/ui-layered-frames/family-previews/paper_utility/`
- **目標**：`assets/resources/sprites/ui_families/common/paper_utility/`
- **搬遷檔案**：`btn_primary_normal.png`、`btn_primary_pressed.png`、`btn_primary_disabled.png`

## 如何驗證
1. 目標路徑下三個 PNG 存在且大小 > 0。
2. 來源與目標每個檔案 SHA256 一致。
3. `docs/ui-quality-todo.json` 中 `UI-1-0002` 的 `status` == `done`，`notes` 含檔案清單。

## Unit Test 建議
**不需要**（搬遷任務以 SHA256 比對作為驗收，無業務邏輯）。

## 建議作法
```powershell
$src = "artifacts/ui-layered-frames/family-previews/paper_utility"
$dst = "assets/resources/sprites/ui_families/common/paper_utility"
New-Item -ItemType Directory -Force -Path $dst
foreach ($f in @('btn_primary_normal.png','btn_primary_pressed.png','btn_primary_disabled.png')) {
  Copy-Item (Join-Path $src $f) (Join-Path $dst $f) -Force
  $hash = (Get-FileHash (Join-Path $dst $f) -Algorithm SHA256).Hash
  $size = (Get-Item (Join-Path $dst $f)).Length
  "$dst/$f | $hash | $size bytes"
}
```
