## 開單原因

品質分析文件中，「圓形 icon 按鈕」在多個畫面出現且缺乏統一素材：

- §6.1（武將詳情）：「生平」「獎勵」按鈕 = 圓形，`#2A2520CC` + 金色環 2px
- §6.2（裝備介面）：「重鑄」「打磨」icon 按鈕 = 圓形金環
- §6.4（信件系統）：底部動作列的「翻譯/收藏/刪除/跳轉」= 小金屬按鈕

這些圓形按鈕是跨畫面的通用 UI 元素，需要統一 sprite 家族而非各畫面各自定義。

## 完整描述

- **目標路徑**：`assets/resources/sprites/ui_families/common/circle_icon/`
- `circle_normal.png`（80×80）
  - 圓形填充：`#2A2520`，alpha=204（80%）
  - 金色環：2.5px，`#C9A028` 系
  - 內側亮線：1px 細亮線增加拋光感
  - 外圈微暗暈：增加「嵌入感」厚度
- `circle_pressed.png`（80×80）
  - 填充稍亮 `#3A3428`，alpha=220
  - 金色環加厚至 3px，亮度 +10%
  - 外圈金光暈（低 alpha），按壓發光效果
  - 頂部 140° 高光弧（拋光反射感）
- `circle_disabled.png`（80×80）
  - 填充灰化 `#2D2D2A`，alpha=130（50%）
  - 暗灰環（金色去飽和）
