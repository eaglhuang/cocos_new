- 供 UI 線稿圖對照 node / slot / rail / badge 命名

- 供後續 fragment 化拆分與 prefab-like module 對齊

## 與現況產能的相容性

### 高可行

- 靜態人物圖 slot
- 右資訊卡模組
- 故事 strip 固定 cell
- 命鏡頁雙人物 + 中央門 + CTA

### 中度成本

- crest family 一致性
- 命鏡中央圖像資產 family
- 英靈面專屬性

### 暫不建議重押

- 大量動態粒子
- 長時間故事帶漂移
- 高複雜度多層透明演出
