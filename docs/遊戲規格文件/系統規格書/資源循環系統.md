## A. 系統描述

精力（Vitality）是武將最核心的消耗資源，統一管理戰鬥、繁殖、培育三大場景的行動力。男女精力消耗不同（男-50/女-100），女性受孕後進入待產凍結。精力每季自然回復，但回復速度受紅色因子、年齡、負傷影響。補元散為加速回復的核心課金道具。

---

## B. 系統目的

1. **統一行動力**：一個資源管三個場景→精打細算
2. **性別差異**：男女消耗不同→配種策略的深度
3. **課金管道**：補元散是最直接的課金出口
4. **節奏控制**：精力限制防止玩家跑太快

|---|---|

## F. 公式相關

### 精力消耗
| 行動 | 男性 | 女性 |
|---|---|---|
| 戰鬥（每戰） | -30 | -30 |
| 結緣 | -50 | -100（+待產凍結） |
| 培育（入學） | -20 | -20 |

### 精力回復
```
Vitality_Recovery_Per_Season = Base(20) + Red_Gene_Bonus + Age_Modifier
Red_Gene_Bonus = Red_Gene_Level × 5
Age_Modifier: 青壯年 ×1.0, 老年 ×0.6
```

### 精力上限
```
Vitality_Max = 100 + Red_Gene_Level × 20
// 近親劣化: Vitality_Max × 0.7
```

### 補元散效果
```
Vitality += 50  // 基礎版
Vitality += 100 // 高級版
// 不超過 Vitality_Max
```

## I. Data Schema 需求

### 本機端 (Client)
```json
{
  "General_Vitality": {
    "UID": "G001",
    "Vitality_Current": 65,
    "Vitality_Max": 140,
    "Recovery_Per_Season": 30,
    "Is_Pregnant": false,
    "Pregnancy_Remaining": 0,
    "Red_Gene_Level": 3
  }
}
```

### Server 端
| 欄位 | 型別 | 說明 |
|---|---|---|
| General_UID | VARCHAR(16) FK | 武將 |
| Vitality_Current | INT | 當前精力 |
| Vitality_Max | INT | 精力上限 |
| Is_Pregnant | BOOLEAN | 是否待產 |
| Pregnancy_End_Season | INT | 待產結束季度 |
