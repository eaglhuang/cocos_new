# UI Proof資料夾模板補遺 2026-04-04

## 正式規則

後續所有新的 AI UI proof，統一使用：

- `docs/UI品質參考圖/UI_PROOF_TEMPLATE/`

作為模板來源。

## 固定流程

1. `brief`
2. `proof`
3. `select`
4. `wireframe`
5. `slotmap`
6. `codegen`

## 原則

1. proof 圖不能直接當實作規格。
2. 必須先做線稿與 slot-map。
3. `layout / skin / screen` JSON 必須回掛到同一批 proof。
4. 後續所有 AI 產碼與 prefab scaffold，優先吃這套資料夾結構。
