# 教官系統（支援卡）規格書

> **來源文件**：養成系統與教官來源定義.md (doc_spec_0151)、傭兵職能分流與傳承機制.md (doc_spec_0134)、鏡像時空育才系統設計.md (doc_spec_0157)、子嗣系統規格書：三國傳承.md、三國傳承：賽馬娘養成融合提案.md、戰棋遊戲的武將傳承與繁衍.md (doc_spec_0126)

---

## A. 系統描述

教官（支援卡）是培育模式的核心驅動角色。類似《賽馬娘》的支援卡，教官提供 36 回合培育中的屬性加成、戰法秘笈啟發與 TP 點數。教官**不參與血統因子遺傳**，僅作為「教學事件」與「TP 資源」的提供者。

> **Unity 對照**：教官卡類似一張 ScriptableObject 資料卡片，綁定在培育 Session 上，提供屬性加成與事件觸發的 Modifier。

---

## B. 系統目的

1. **培育深度**：提供 36 回合中的隨機事件樂趣（教官驚嘆號觸發秘笈）
2. **卡池營收**：支援卡（教官）池的核心抽取標的
3. **無課回流**：退役武將轉教官，讓無課玩家也有頂級教官資源
4. **職能分離**：教官不等於親生父母，遺傳歸血統、教學歸教官
5. **賽馬娘式體驗在地化**：保留支援卡帶來的培育戲劇性，但不把血統、英靈與虎符重新混入抽卡或突破線

## C. 商業套路

| 商業點 | 說明 |
|---|---|
| 支援卡（教官）池 | SSR 教官卡抽取，提供獨有結緣事件劇情 |
| 突破 (Limit Break) | 重複卡片合成提升星等，解鎖更高靈感戰法事件與加成效果 |
| 求籤令 | 消耗道具指定特定名士深度占卜 |
| 入學令 | 門檻道具控制培育啟動頻率：無課每日 1~2 次，課長可購買加速 |
| **戰法事件星等突破** | 满突破（5 星）教官才能觸發全部戰法靈感事件，強化突破動機 |

## F. 公式相關

### F-1. 支援卡 UI 結構（對照賽馬娘）

> **賽馬娘對照**：支援卡詳情頁分四頁籤（支援效果 / 持有技能 / 培育事件 / 小插曲），本項對應三國名將教官卡的版本。

| 頁籤 | 對應內容 |
|---|---|
| **支援效果** | 屬性加成清單（可能鎖定到特定星等）、友情加成、訓練機率 |
| **培育事件** | 本卡可觸發的全部戰法靈感事件（未突破者顯示鎖頭） |
| **小插曲** | SSR/SR 指定教官雙方對話劇情 |

### F-2. TP 計算
- **基礎 TP**：每回合訓練基礎 TP = 10
- **教官加成**：TP_per_round = 10 × (1 + Support_TP_Bonus)
- **因子折扣**：若子嗣繼承對應因子，學習該戰法 TP 消耗 ×0.5

### F-3. 靈感事件觸發機率

```
P(靈感) = Base_Rate × (1 + Friendship_Bonus) × (1 + 子嗣_LUK / 200)
```
- 36 回合內特定靈感未觸發 → 該戰法不出現在結業點選清單

### F-4. 卡片星等突破系統

> 重複獲得同名教官卡 = 突破一次（最高 5 星 = 累計5 次）。戰法靈感事件與部分加成效果根據星等逐步解鎖。

| 星等 | 友情加成 | 訓練效果 | 已解鎖戰法靈感事件 | 其他加成 |
|---|---|---|---|---|
| ★0（無突破） | ×1.0 | ×1.0 | 事件組 1（基礎戰法） | 屬性加成基礎值 |
| ★1 | ×1.1 | ×1.05 | 事件組 1–2 | 初期情誼量 +5 |
| ★2 | ×1.2 | ×1.1 | 事件組 1–3 | 訓練機率 +3% |
| ★3 | ×1.3 | ×1.15 | 事件組 1–4 | 屬性加成 ×1.1 |
| ★4 | ×1.4 | ×1.2 | 事件組 1–5 | TP 加成 +10% |
| ★5（全5次） | ×1.5 | ×1.3 | 全部事件組（含頂級 SSR 戰法） | 固有加成最高級 |

**卡片頁面結構說明（UI）**：
- 各頁籤的效果加成對應星等顯示鎖頭圖示，突破後自動解鎖
- 支援效果頁：顯示所有效果和對應出現需求的星等
- 培育事件頁：顯示本卡全部戰法靈感事件，未突破者顯示鎖頭

### F-7. 名將傳道機制（Legendary_Mentoring）

> **來源**：遊戲機制推演與玩家歷程.md (doc_spec_0064)

正式共識修正如下：

1. 名將**退役**時只進入教官 / 傳承路徑，不直接生成英靈卡。
2. 名將**死亡**後才會留下英靈卡；英靈卡的主系統歸 [英靈世家系統.md](C:\Users\User\3KLife\docs\系統規格書\英靈世家系統.md (doc_spec_0022)) (doc_spec_0022) 管理。
3. 名將若生前已先退役，最終死亡時仍需補發英靈卡與虎符卡；退役身分不會覆蓋死亡結算。
4. 本系統只負責定義：當玩家在培育流程中啟用英靈卡傳道權限時，如何把效果掛接到培育事件與專屬奧義授與流程。

### F-8. 訓練格爆發機率系統（Training Slot Burst）

> **設計參考**：《賽馬娘》支援卡「亮彩（彩色）」機制——支援卡出現在自己擅長訓練科目時，訓練效果大幅提升。
> **Unity 對照**：類似 Animator Override Controller，依條件切換到「增強狀態」材質與動畫。

#### 爆發規則

- 每回合訓練時，系統依照教官的 `Slot_Burst_Rate` 決定此回合是否「站在擅長格」
- **站在擅長格（爆發）**：訓練點數×1.5、友情訓練觸發率+10%、教官頭像亮彩特效 ← **亮彩效果**
- **不在擅長格（普通）**：基礎點數 ×1.0，正常觸發率

#### 爆發機率範圍

| 稀有度 | 爆發率範圍 | 說明 |
|---|---|---|
| R | 10%~25% | 低可靠度，隨機性高 |
| SR | 25%~50% | 中等穩定 |
| SSR（基礎 ★0） | 40%~65% | 強化穩定 |
| SSR（滿突破 ★5） | 55%~80% | 滿突破核心回報之一 |

#### 爆發效果對照

| 狀態 | 訓練點數 | 靈感觸發率 | 友情觸發率 |
|---|---|---|---|
| 普通出現（非擅長格） | ×1.0 | 基礎 | 基礎 |
| 爆發出現（擅長格亮彩） | ×1.5 | 基礎 +5% | 基礎 +10% |

> **實作備注**：`Slot_Burst_Rate` 儲存在支援卡 Schema，每回合在伺服器端骰一次。爆發結果回傳客戶端後，在培育 UI 的教官頭像上播放「亮彩閃光」粒子效果。

```
// 名將傳道觸發條件：
// 1) 名將已死亡並留下英靈卡
// 2) 玩家指定目標子嗣（需在培育狀態中）
// 3) 消耗「傳道令」×1（英靈卡傳道道具）

Teaching_Result = {
    Skill_Granted: General.Exclusive_Skill,  // 名將的專屬奧義
    Guarantee: true,                          // 100% 傳授，無失敗機率
    Cooldown: "one_time_per_general"          // 每位名將一生只能傳道 1 次
}
```

**設計意圖**：讓高價值名將在死亡後留下的英靈卡不只是收藏品，而是可對培育產生一次高價值「師徒傳承」的情感羈絆事件；同時維持退役教官與死亡英靈卡的職能分離。

### F-9. 教官、保種與英靈傳道的正式邊界

1. 支援卡池販售的是**培育深度**，不是血統種子；玩家若要擴充可繁衍家系，仍需回到名將本體與結緣系統處理。
2. 退役中的活體名將，若仍保留 `General` 身分，可先完成保種型結緣；一旦卡片化成 `Support_Card`，便永久轉入教學線，不再參與親傳。
3. 教官的 `Breakthrough / Slot_Burst / Friendship_Bonus` 只影響 36 回合培育與戰法事件，不得回寫血統因子、祖先矩陣或受孕率。
4. `Legendary_Mentoring` 是死亡後英靈卡的一次性傳道入口，不併入支援卡池保底、突破與溢出轉化規則。
5. 戰場俘虜、傭兵結義或退役轉化帶來的新教官，只代表新的教學資源來源，不代表新增可繞過家族系統的血統捷徑。

## J. 名詞定義

| 名詞 | 代碼/Key | 定義 |
|---|---|---|
| 教官 | Mentor / Support | 培育模式中提供屬性加成與戰法傳授的支援卡 |
| 支援卡 | Support_Card | 教官的卡片化形式 |
| 教學流派 | Teaching_Style | 如醫療、槍法、計略、魅力、火攻 |

## M. 溢出轉化系統

> **設計理念**：升星機制回歸純粹——只能靠重複抽到同名支援卡來突破，無碎片庫。滿星（★5）溢出時自動轉化為資源，避免重複卡「白送」的挫敗感，同時提供額外回報誘因（類似《FGO》的龍鱗/聖晶石回饋）。

### M-1. 突破規則

| 操作 | 觸發條件 | 效果 |
|---|---|---|
| 自動突破 | 獲得重複支援卡，Current_Star < 5 | Current_Star +1，播放突破動畫 |
| 溢出轉化 | 獲得重複支援卡，Current_Star == 5（已滿） | 轉化為溢出獎勵（Overflow_Reward） |

> **突破動畫（B5 確認）**：抽到重複卡時直接播放演出動畫（無需玩家選擇），突破完成後顯示新星等與新解鎖效果。

### M-2. 溢出獎勵（Overflow_Reward）

> 滿 ★5 後再抽到同名支援卡，無法繼續突破，改為轉化分配以下獎勵：

| 稀有度 | 溢出獎勵 | 說明 |
|---|---|---|
| SSR | 積分 ×500（點數商店） | 可在點數商店兌換道具/材料 |
| SR | 積分 ×100 | |
| R | 積分 ×20 | |

> 積分商店後續可視收益調整獎勵內容（道具、培育加速令、入學令等）。

### M-3. 溢出觸發流程

```
獲得重複支援卡
  → Current_Star < 5 → 自動突破：Star +1 → 播放突破動畫 → 顯示新解鎖效果
  → Current_Star == 5 → 溢出動畫 → Overflow_Reward 入帳 → Toast：「{CardName} 已滿突破，轉化為積分×{N}」
```
