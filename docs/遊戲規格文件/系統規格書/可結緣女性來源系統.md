# 可結緣女性來源系統規格書

> **來源文件**：傭兵系統規格書：人才經營進化.md、養成系統與教官來源定義.md (doc_spec_0151)、子嗣系統規格書：三國傳承.md、轉蛋與戰場武將處置系統.md (doc_spec_0155)

---

## A. 系統描述

可結緣女性來源系統定義玩家獲取女性武將的所有管道與機率。女性武將在本遊戲中扮演「母體」的角色，是血統傳承一代的關鍵。不同來源的女性具備不同的品質上限、因子特性與取得成本，構成從免費到付費的完整梯度。

> **Unity 對照**：類似 Unity 的 Item Acquisition Pipeline（物品獲取管線），不同管道（商城/掉落/任務/製作）輸出相同物件但屬性分佈不同。此處輸出的「物件」是武將實體。

---

## B. 系統目的

1. **母體供給**：確保各付費層級的玩家都有穩定的女性武將來源，維持配種循環
2. **品質梯度**：課長可透過轉蛋獲得高品質 SSR 女將；無課靠農系統積累
3. **多元體驗**：五大管道各有不同的遊玩體驗（抽卡刺激/戰場征服/養成期待/探索冒險/血脈培育）
4. **經濟引擎**：女性武將是配種系統的入口，間接驅動精力消耗→補元散銷售

## C. 商業套路

| 商業點 | 說明 |
|---|---|
| SSR 女將稀有度 | 貂蟬、孫尚香等頂級女將僅在轉蛋池出現，UP 機率低 |
| 安胎藥/送子符 | 低品質母體受孕率低，付費道具可強行突破 |
| 傭兵美酒 | 結義高品質女傭兵需消耗美酒（$0.99/瓶） |
| 入學令 | 女兒培育需入學令，課長可大量購買加速 |
| 刷新券 | 傭兵池/地下城手動刷新消耗道具 |

## F. 公式相關

### 各管道女性品質分佈

| 管道 | 品質範圍 | 六色因子上限 | 取得成本 |
|---|---|---|---|
| 轉蛋池 | SSR~R | SSR: 1500+（5轉） | 抽卡貨幣 |
| 傭兵結義 | SR~N | SR: 1000 | 美酒 + 黃金（6日結義） |
| 戰場俘虜 | SR~N | 隨武將原始能力 | 免費（戰場勝利） |
| 地下城救美 | R~N | 民女 300、千金 500、貴族 700 | 免費（8h 刷新任務） |
| 子嗣（女兒） | 依父母 | 由遺傳決定（無上限） | 配種＋培育精力 |

### 受孕率修正

```
base_rate = 0.6  // 基礎受孕率 60%
if (mother.source == "GACHA_SSR"): rate_mod = 1.2
if (mother.source == "DUNGEON_COMMONER"): rate_mod = 0.8
if (母齡 > 40): rate_mod *= 0.7
if (使用安胎藥): rate_mod *= 1.5
final_rate = base_rate × rate_mod
```

### 地下城救美品質機率

| 結果 | 機率 | 品質 |
|---|---|---|
| 民女 | 60% | R 級，基礎因子 |
| 富家千金 | 30% | R+ 級，額外政治/魅力因子 |
| 落難貴族 | 10% | SR- 級，可能攜帶名門白色因子 |
