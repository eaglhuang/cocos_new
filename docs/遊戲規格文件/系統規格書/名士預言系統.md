# 名士預言系統規格書

> **來源文件**：名士預言：血統模擬器新玩法.md、逐步開鎖系統的設計.md (doc_spec_0097)

---

## A. 系統描述

名士預言系統是結緣前的「付費占卜」機制。玩家消耗求籤令指定一位名士（諸葛亮、華佗、司馬徽等），名士會根據父母双方因子給出帶有該名仕個人偏見的預言提示。不同名士有不同偏好面向（武力/智力/壽命），且預言有刻意設計的「不完全準確」，形成先知陷阱（越信越花錢，但方向大致正確）。

---

## B. 系統目的

1. **付費精準預覽**：讓玩家有「花錢看運勢」的體驗
2. **不確定性保留**：名士偏見＋準確率上限讓結果仍有期待
3. **情感代入**：三國名士作為 NPC，形成社交話題
4. **消耗設計**：每次占卜消耗求籤令（稀缺資源）

---

## C. 商業套路

| 商業點 | 說明 |
|---|---|
| 求籤令 | 每次占卜消耗 1 張，免費玩家日產 1 張，課金無上限 |
| 名士收集 | 解鎖新名士需滿足條件（如占卜 50 次解鎖龐統） |
| 先知陷阱 | 名士預言 70~85% 準確→玩家會「想再看一個名士」以交叉驗證 |
| VIP 名士 | 限定活動名士（如左慈）加成效果更獨特 |

## F. 公式相關

### 名士偏好表
| 名士 | 偏好面向 | 準確率 | 偏見方向 |
|---|---|---|---|
| 諸葛亮 | 智力/政治 | 85% | 高估智力型子嗣，低估武力型 |
| 華佗 | 壽命/體魄 | 80% | 偏重紅色因子預測 |
| 司馬徽 | 全面/中庸 | 75% | 客觀但不精準 |
| 許劭 | 性格/天賦 | 80% | 白色因子偏好 |
| 龐統 | 軍事/戰法 | 85% | 綠色因子偏好 |
| 左慈 | 運氣/天命 | 70% | 紫色因子偏好，預言最模糊 |

### 預言生成邏輯（修正與嚴謹化）
```
// accuracy_rate 為 0.0~1.0 的機率值，表示名士給出「非扭曲」預言的機率
if (Random() <= accuracy_rate) {
  Prediction = WeightedSample(child_actual_genes, sage_bias_weight)  // 合理預言
  Is_Distorted = false
} else {
  Prediction = Distort(WeightedSample(child_actual_genes, sage_bias_weight))  // 故意偏離
  Is_Distorted = true
}
Comment = TemplateFormat(sage_style, Prediction)
```

### 驗證示例（用於單元測試）
// 假設：accuracy_rate = 0.85
// 範例 1: Random() = 0.10 → 0.10 <= 0.85 → 預言為正確（Is_Distorted = false）
// 範例 2: Random() = 0.90 → 0.90 > 0.85 → 預言被扭曲（Is_Distorted = true）
// 範例 3: Random() = 0.50 → 0.50 <= 0.85 → 預言為正確（Is_Distorted = false）

// 備註：以上示例應以 unit tests 驗證隨機函數的行為，可用固定種子覆寫 Random() 回傳以達成 deterministic 測試。

## K. 血統模擬器嵌入與祖輩顯靈規格

> **整併來源**：名士預言：血統模擬器新玩法.md

### 結緣介面嵌入方式

- 名士預言必須能嵌入 `結緣系統（配種）` 的主介面，而不是切去另一個完全獨立頁面。
- 標準版配置：中央為血統模擬器、左右為名士批語區、底部為求籤令與更換名士入口。
- 預測結果採「批語 + 色彩傾向 + 關鍵詞」表達，不直接顯示精準五維數字。

### 祖輩顯靈與認同感設計

- 若父母血統矩陣命中特定祖先標籤，可觸發祖輩顯靈式的補充台詞。
- 顯靈台詞的作用是強化玩家對血統來源的認同感，例如提示某個兵種適性、人格傾向或戰法種子來源。
- 顯靈台詞不應繞過名士偏見與準確率設計；其本質是敘事包裝，不是額外一次完全準確的占卜。

### 矛盾批語的玩家解讀

- 系統應鼓勵玩家把不同名士的矛盾意見當作「職能分流建議」：適合培養成武將、軍師、教官或晚成型角色。
- UI 上可顯示「觀點相左，需待 5 歲開獎驗證」之類的提示，避免玩家誤以為系統資料錯亂。
