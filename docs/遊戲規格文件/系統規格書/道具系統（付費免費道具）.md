# 道具系統（付費免費道具）規格書

> **來源文件**：傭兵系統規格書：人才經營進化.md、因子血統論系統規格書.md (doc_spec_0073)、子嗣系統規格書：三國傳承.md、養成系統與教官來源定義.md (doc_spec_0151)、武將裝備道具系統.md (doc_spec_0174)
> **資料層對齊**：本檔是 `Inventory Service` 下的「一般道具」子域規格；持有、堆疊、過期與商店來源的共用真相以 `武將裝備道具系統.md` 為準。

---

## A. 系統描述

道具系統管理遊戲內所有可消耗、可持有的物品，分為「付費道具」與「免費道具」兩大類。付費道具透過商城直接購買或間接付費管道取得；免費道具由任務獎勵、戰場掉落、掛機產出等途徑獲取。道具是連結各子系統（結緣、傭兵、培育、戰場）的消耗媒介。

> **Unity 對照**：相當於 ItemDatabase（ScriptableObject 資產庫）+ InventoryManager（持有/消耗/增刪邏輯）。每個 ItemSO 定義品質、效果與消耗條件，InventoryManager 管理 runtime 增刪與 UI 綁定。

---

## B. 系統目的

1. **經濟媒介**：將玩家的時間/金錢投入轉化為遊戲內具體物品
2. **系統連結**：各子系統的關鍵操作（結緣/培育/刷新/加速）都需要消耗特定道具
3. **付費分層**：課長靠買道具加速；無課靠農產出慢慢累積
4. **營運彈性**：透過活動/禮包調節道具投放量，控制遊戲經濟節奏

## I. Data Schema 需求

### 本機端 (Client)

```json
{
  "ItemDef": {
    "Item_ID": "string (道具 ID)",
    "Name": "string",
    "Description": "string",
    "Category": "CONSUMABLE | MATERIAL | SPECIAL | CURRENCY | TREASURE",
    "Quality": "SSR | SR | R | N",
    "Max_Stack": "number (堆疊上限)",
    "Is_Paid": "boolean (是否付費道具)",
    "Effect_Type": "RESTORE_VITALITY | BOOST_CONCEPTION | UNLOCK_TRAINING | REFRESH | ORACLE | EQUIP",
    "Effect_Value": "number",
    "Price_Gem": "number (玉璧價格，0=免費)",
    "Expire_Duration": "number (有效期秒數，0=永久)"
  }
}
```

```json
{
  "PlayerInventory": {
    "Player_ID": "string",
    "Items": [
      {
        "Item_ID": "string",
        "Count": "number",
        "Acquired_At": "string (時間戳)",
        "Expire_At": "string (過期時間，null=永久)"
      }
    ]
  }
}
```

### Server 端

| 欄位 | 型別 | 說明 |
|---|---|---|
| inventory_id | VARCHAR(32) PK | 持有記錄 ID |
| player_id | VARCHAR(32) FK | 玩家 |
| item_id | VARCHAR(32) FK | 道具 ID |
| count | INT | 數量 |
| acquired_at | DATETIME | 取得時間 |
| expire_at | DATETIME NULL | 過期時間（NULL=永久） |
| source | VARCHAR(32) | 來源管道 (SHOP/TASK/DROP/EVENT) |

## J. 名詞定義

| 名詞 | 定義 |
|---|---|
| 付費道具 | 需透過商城購買（直接或間接使用現金/付費貨幣）才能取得的道具 |
| 免費道具 | 可透過遊玩（任務/掉落/掛機/推薦）取得的道具，無需直接付費 |
| 消耗品 | 使用後數量減少、產生即時效果的道具（如補元散、美酒） |
| 材料 | 用於合成或升級的中間物品（如戰法筆記） |
| 傳家寶物 | 俘虜在無期徒刑中交出的稀有物品，可裝備或販賣 |
| 玉璧 | 遊戲付費貨幣，透過商城儲值取得 |
| 掛機產出 | 離線或掛機狀態下自動產生的資源與道具 |
| 掉落表 | 定義各戰場/任務中物品掉落機率的配置資料 |
| 禮包 | 將多種道具組合為單一商品的特殊銷售形式 |
