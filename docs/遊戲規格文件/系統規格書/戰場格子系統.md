# 戰場格子系統

## A. 系統描述
透過「Tile-State Machine (格子狀態機)」將戰場邏輯與視覺分離，支援一套底層機制多種玩法表現。

## B. 核心格子狀態
1. **Normal (基礎)**：普通行軍與占領。
2. **Hazard (傷害)**：onEnterDamage(ratio)。如：火燒（延燒）、落石（固定高傷）。
3. **Force-Move (位移)**：onEnterPush(direction)。如：水淹將敵軍向後推，若受阻則轉為傷害。
4. **Stealth (掩體)**：onEnterHide(boolean)。如：森林埋伏，進入後隱蔽防遠程。

## C. 美術視覺與特效
格子系統作為底層容器，上層可掛載各種 VFX 與 Decal，使同套傷害邏輯可表現為毒沼或火海。
