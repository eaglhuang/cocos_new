## A. 系統描述

英靈世家系統負責處理名將死亡後的兩條正式傳承產物，以及這些產物與家族積分、姓氏歸屬、供養與血脈延續的關係。

正式共識如下：

1. 名將在**死亡結算**時，同步留下 `英靈卡` 與 `虎符卡`。
2. 名將在**生前退役**時，只進入教官 / 傳承路徑，不直接生成英靈卡與虎符卡。
3. `英靈卡` 用於世家供養、祖靈加成、傳道與血脈陳列。
4. `虎符卡` 用於兵種軍勢化身、特殊兵種調度與戰場配置。
5. 轉蛋系統不再產出英靈卡或虎符卡；轉蛋只保留 `三國名將池（含女性）` 與 `支援卡（教官）池`。

> **Unity 對照**：英靈卡接近長線帳號級的 `Legacy Relic / Account Relic`，負責被動加成與家族展示；虎符卡接近可裝備的 `Equipment + Unit Unlock Token`，負責解鎖與部署特殊軍勢。兩者都來自角色生命週期結算，而不是一般抽卡掉落。

---

## B. 系統目的

1. **死亡不等於消失**：讓名將死亡後轉化為可持續經營的世家資產。
2. **傳承拆層**：把祖靈供養與戰場兵符拆成兩種不同用途的卡，避免單一系統過載。
3. **姓氏零和**：家族歸屬、絕學傳承與功勳結算統一走單一姓氏歸屬，不允許雙邊重複受益。
4. **長線世家經營**：透過人頭分、功勳分與名聲值，建立可持續但不失控的英靈成長曲線。
5. **回收付費價值**：高成本培養的名將死亡後，仍可透過英靈卡與虎符卡延續戰略價值與展示價值。

## C. 核心邊界

### C-1. 名將死亡產物

| 產物 | 用途 | 主責規格 |
|---|---|---|
| 英靈卡 | 世家供養、祖靈加成、傳道、陳列室 | 本檔 |
| 虎符卡 | 特殊兵種解鎖、軍勢化身、戰場裝備 | 兵種（虎符）系統.md (doc_spec_0012) |

### C-2. 與其他系統的邊界

| 系統 | 邊界 |
|---|---|
| 轉蛋系統 | 只抽三國名將（含女性）與支援卡（教官），不產英靈卡 / 虎符卡 |
| 教官系統（支援卡） | 名將生前退役可轉教官；死亡後留下的英靈卡不是一般支援卡突破材料 |
| 武將壽命系統 | 死亡事件觸發英靈卡與虎符卡結算；退役只切換存活中的職能 |
| 血統理論系統 | 姓氏歸屬、近親衰減、命紋與祖紋語言由血統系統提供上游定義 |
| 兵種（虎符）系統 | 虎符卡的戰場效果與兵種數值由虎符系統主責，本檔只定義來源與歸屬 |

## D. 英靈卡生命週期

### D-1. 誕生條件

1. 史實名將死亡時必定生成 1 張對應英靈卡。
2. 女性名將若具正式名將模板，與史實名將同樣適用必出規則。
3. 非名將角色一律不生成英靈卡；若未來要開放例外，必須另立正式規格。
4. 名將若生前已進入退役 / 教官路徑，死亡時仍需依正式規則補發英靈卡與虎符卡。

### D-2. 唯一性原則

1. 同一存檔世界中，同一名將模板只維持 1 張正式英靈卡主體。
2. 同名模板再次死亡時，不新增第二張平行主卡，而是進行舊卡覆蓋與成長結算。

### D-3. 屬性覆蓋規則

```text
SpiritCard.FinalStat = max(ExistingSpiritCard.Stat, DeathSnapshot.Stat)
SpiritCard.LineageScore = ExistingLineageScore + CurrentDeathContribution
```

1. 舊卡與新死亡快照以「高者覆蓋」處理核心數值。
2. 家族貢獻與供養等級採累計制，不因新死亡結算而清空。
3. 已解鎖的祖靈階位不倒退，只允許維持或升階。

### D-4. 傳道權限

1. 英靈卡可以授與特定子嗣或血脈分支祖靈加護。
2. 傳道屬於英靈卡路徑，不屬於一般教官卡的抽卡突破路徑。
3. 英靈卡若要直接傳授專屬奧義，需消耗專屬資源與次數，不等於一般培育教官事件。

## E. 虎符卡來源與歸屬

### E-1. 虎符卡正式來源

1. 虎符卡只來自名將死亡結算。
2. 虎符卡與英靈卡在同一次死亡結算中同步產生，但用途不同。
3. 虎符卡不屬於轉蛋道具，也不是一般商店常駐掉落物。

### E-2. 虎符卡與英靈卡的差異

| 比較項 | 英靈卡 | 虎符卡 |
|---|---|---|
| 主用途 | 世家供養、祖靈加成 | 特殊兵種、軍勢化身 |
| 裝備位置 | 世家 / 血脈 / 陳列室 / 傳道介面 | 武將虎符槽 |
| 成長來源 | 家族積分、供養、名聲值 | 虎符精華、戰績、適性與突破 |
| 視覺語言 | 祖紋命篆、命紋靈獸、宗族匾額 | 兵種旗幟、甲紋、軍勢圖騰 |

### E-3. 雙卡同裝正式規則

1. 同一名將死亡結算生成的英靈卡與虎符卡，允許由同一名角色同時裝備。
2. 英靈卡占用祖靈 / 世家層插槽，虎符卡占用戰場虎符槽，兩者不互斥。
3. 若同時滿足同姓或血脈共鳴條件，可分別吃到各自系統的加成；不因雙卡同裝而打折。

## F. 家族積分與世家等級

### F-1. 積分構成

家族積分由 `人頭分` 與 `功勳分` 組成。

正式範圍採 **完全個人存檔制**：只統計玩家自己存檔中的家族線，不採同伺服器共享，也不混入他人同姓積分。

```text
ClanScore = HeadScore + MeritScore
```

#### 人頭分

| 類型 | 權重 |
|---|---|
| 本家後代 | 100% |
| 俘虜轉化後納入本家者 | 80% |
| 外嫁且最終選擇隨母姓的後代 | 120% |

#### 功勳分

功勳分由下列行為合成：

1. 戰場總傷害
2. 總擊殺數
3. 攻城與據點貢獻
4. 特定名將挑戰 / 劇本節點達成

### F-2. 世家階位

| 階位 | 說明 |
|---|---|
| 傳家之寶 | 解鎖英靈卡基礎加成與陳列 |
| 名門望族 | 解鎖宗族加成與次級祖靈效果 |
| 萬世宗親 | 解鎖跨代供養強化與視覺升格 |
| 武聖神祇 | 解鎖高階祖靈顯化與頂階傳道權限 |

### F-3. 名聲值與供養消耗

1. 英靈卡的高階加成不能無成本常駐，必須消耗 `家族名聲值`。
2. 家族成員若出現叛逃、作惡、低評價結局，會削弱家族名聲值與供養倍率。
3. 名聲值不足時，英靈卡仍存在，但高階世家效果會降階或失效。

## G. 姓氏歸屬與絕學限制

### G-1. 姓氏零和原則

1. 子嗣必須在父姓與母姓中擇一歸屬。
2. 一旦選定姓氏，該子嗣的家族積分、世家供養、宗族關係與絕學權限皆綁定該姓氏。
3. 禁止同一子嗣同時為父母兩家累積正式世家積分。

### G-2. 家傳絕學限制

1. 家傳絕學需符合該姓氏的宗族權限才能發揮完整威力。
2. 非本姓但裝備同名英靈卡或虎符卡時，可獲得弱化版效果，不得享有完整宗族特權。
3. 同姓或具正式血脈共鳴者，裝備對應虎符卡時可觸發額外共鳴效果。

---

## H. 防火牆與反膨脹規則

### H-1. 近親衰減

1. 近親聯姻不視為高效率刷分手段。
2. 近親後代容易出現隱藏屬性劣化、負向因子增加與家族積分效率下降。

### H-2. 同宗不相殘

1. 同宗姓氏間互戰時，功勳積分應打折計算。
2. 該規則用於阻止玩家透過內戰刷英靈卡成長。

### H-3. 質重於量

1. 高品質後代應提供較高回向收益。
2. 低品質後代不應因數量優勢就把高品質世家培養線完全稀釋。

## J. 正式共識摘要

1. 轉蛋只保留 `三國名將池（含女性）` 與 `支援卡（教官）池`。
2. 名將死亡時，同步產出 `英靈卡` 與 `虎符卡`。
3. 退役只進教官 / 傳承路徑，不直接產卡。
4. 英靈卡主責世家與祖靈系統，虎符卡主責軍勢與兵種系統。
5. 姓氏歸屬採零和制，家族積分與絕學權限不得雙邊重複計算。
6. 英靈卡門檻鎖定為「史實名將 + 女性名將（具正式名將模板）必出，其餘不出」。
7. 世家積分採個人存檔制，且英靈卡與虎符卡允許同角色同時裝備。
