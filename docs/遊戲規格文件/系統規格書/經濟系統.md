# 經濟系統規格書

> **來源文件**：多份來源文件中的貨幣與資源設計、SLG 遊戲養成與商業化討論.md (doc_spec_0129)

---

## A. 系統描述

遊戲經濟分為基礎資源（糧草/黃金/兵力/名聲）、稀缺貨幣（精元/結義點數）與道具（補元散/求籤令/入學令/轉職令/延壽丹）。基礎資源從治理產出，稀缺貨幣從轉蛋/活動取得，道具為直接課金或任務獎勵。經濟設計核心：免費玩家靠時間換資源，課金玩家靠金錢換效率。

---

## B. 系統目的

1. **資源壓力**：有限資源創造取捨決策
2. **課金出口**：每個系統都有對應的課金道具
3. **循環閉環**：治理→資源→戰場→俘虜→更多資源
4. **公平設計**：付費買的是效率不是數值天花板

## C. 商業套路

| 商業點 | 說明 |
|---|---|
| 補元散 | 精力回復=最高頻消耗品 |
| 求籤令 | 名士占卜=中頻消耗 |
| 入學令 | 培育名額=中頻消耗 |
| 月卡 | 每日精力+結義點數加成 |
| 通行證 | 季度活動獎勵追蹤 |
| 許願池免費轉化 | 免費玩家透過「許願」換取商品，但需等待 8~24h 時間償還 |
| 加速卡轉化 | 免費許願過程可用「加速卡」縮短等待，將免費贈品轉為付費消耗 |
| 特訓加速包 | 以 `兵法書 / 古籍` 為包裝的等待加速道具，縮短特訓或勸降流程 |
| 軍備維護包 | 以 `高級糧草包 / 軍械修復盒` 為包裝的高頻戰場消耗品，支援長線遠征與高壓推圖 |
| 功能位擴充 | 特訓槽、教官位等生產位擴充，賣的是同時處理效率而非戰力上限 |

### 地窖保護與每日補貼

```typescript
const CELLAR_PROTECTION = {
  food_floor: 1500,
  gold_floor: 600,
  trigger_after_failed_defense: true,
};

const DAILY_SUBSIDY = {
  food_if_below: 1200,
  food_grant: 400,
  gold_if_below: 500,
  gold_grant: 150,
  troops_if_below: 600,
  troops_grant: 200,
};
```

- 地窖保護的目的是防止玩家被連續戰敗或高頻部署直接打到零資源，導致玩法停擺。
- 每日補貼只補到「可重新開局」的最低操作水位，不補到舒服線，避免養成無成本掛機。
- 被保護的資源屬於**保底層**，不可被掠奪、不可再拿去當作高額商業折現。

### 許願池商城（時間償還系統）

> **設計理念**：免費玩家用「時間」換商品，課金玩家用「金錢」換效率。兩條路都能拿到商品，但課金更快。

```typescript
// === 許願池機制 ===
// 每日開放 3 巺F（早/中/晚），每巺可選 3 項商品許願
const WISH_POOL_CONFIG = {
    rounds_per_day: 3,          // 早(06:00)/中(12:00)/晚(18:00)
    wishes_per_round: 3,        // 每巺可許願 3 項
    max_concurrent_wishes: 5,   // 同時最多 5 個許願在進行中
};

// === 雙軌取得方式 ===
// 課金要徑：精元立即支付 → 100% 產量立即入帳
// 免費要徑：選擇商品「許願」 → 進入 8~24h 償還期 → 獲得下修產量

// === 許願償還期與產量 ===
type WishItem = {
    item_id: string;
    full_amount: number;           // 課金即買數量
    wish_amount: number;           // 許願獲得數量（下修）
    wish_yield_rate: number;       // 產量比率（0.05 ~ 0.30）
    redemption_hours: number;      // 償還時間（8 ~ 24h）
    jing_yuan_price: number;       // 精元訂價（課金即買）
};

// 典型商品許願產量比率
const WISH_YIELD_TABLE: Record<string, { rate: number; hours: number }> = {
    Vitality_Pill:     { rate: 0.20, hours: 8 },   // 補元散：課金 5 個→許願得 1 個，8h
    Oracle_Token:      { rate: 0.15, hours: 12 },  // 求籤令
    Admission_Token:   { rate: 0.10, hours: 16 },  // 入學令
    Tally_Essence:     { rate: 0.10, hours: 24 },  // 虎符精華
    Job_Change_Token:  { rate: 0.05, hours: 24 },  // 轉職令
    Longevity_Pill:    { rate: 0.05, hours: 24 },  // 延壽丹
};

// === 加速卡機制 ===
// 可用「加速卡」縮短許願等待時間，將免費贈品轉化為付費道具消耗
// 1 張加速卡 = 減少 2h 償還時間（不可低於 0）
// 加速卡來源：月卡每日遞送 1 張 / 精元 5 = 1 張
const SPEED_CARD_HOURS = 2;        // 每張縮短 2h
const SPEED_CARD_PRICE = 5;        // 精元 5 購買 1 張
```

### 戰場 / 挑戰賽 / 商業化資源回圈

1. 治理產出 `Food / Gold / Troops`，支撐戰場部署與挑戰賽報名。
2. 戰場部署消耗糧草，回收戰績、虎符養成材料與俘虜收益。
3. 名將挑戰賽消耗報名次數與培育資源，回收段位獎勵、限定稱號與賽季虎符動機。
4. 商業化只加速「補元散 / 入學令 / 加速卡 / 額外報名」，不直接出售勝負結果。
5. 地窖保護與每日補貼維持最低遊玩回圈，避免敗局把玩家踢出經濟系統。
6. 全球遠征屬於後段高壓資源井，主要回收稀有戰利品、異域素材與長線展示型成就，不應回寫成前期必修經濟壓力。
