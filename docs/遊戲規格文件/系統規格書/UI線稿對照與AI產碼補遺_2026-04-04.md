## 正式原則

1. proof 圖不直接當成實作規格，必須先回壓成 wireframe。
2. wireframe 必須標出 root / module / slot / label / badge / story cell。
3. `layout / skin / screen` JSON 是 codegen-ready 主依據。
4. AI 產碼時，不得自行變更故事格數與大區塊 hierarchy。
