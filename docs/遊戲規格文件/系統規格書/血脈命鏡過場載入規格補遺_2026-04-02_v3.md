# 血脈命鏡過場載入規格補遺 2026-04-02 v3

> 對應 proof：`突破頁 v3 / 血脈命鏡 v3`
> 任務來源：`UI-2-0054`

## 定位

`血脈命鏡` 正式定位為以下三類畫面的共用高階展示版型：

1. `Loading Screen`
2. `英靈升星 / 二轉突破 / 覺醒頁`
3. `未持有名將展示頁`

這不是日常資訊頁，而是偏世界觀、收藏感、命格演出的高光頁。

## v3 正式母型

`突破頁 v3` 可作為目前正式母型，核心結構固定為：

1. 左側：`表象面 / 現世名將`
2. 中央：`命鏡裂隙 / 古老之門`
3. 右側：`血脈面 / 英靈面`
4. 下中：`定心者 / 靈魂載體`
5. 下方：`5 格命運故事帶`

## 視覺規則

### 左側表象面

- 角色為實體、明亮、可辨識的名將本尊。
- 背景偏現世、戰場、軍營、塵土、旗幟等人間空氣。
- 不承載大量資訊，主要負責角色魅力與現世身份。

### 右側血脈面

- 必須明顯仍是同一名將，不可變成 generic 靈魂武將。
- 使用半透明、命紋、淡光、幽夜色調表現英靈感。
- 禁止恐怖裂臉、怪物化、重科幻化。
- 命紋、鎧甲細紋、胸口印記、武器光痕可承載血脈語言。

### 中央命鏡裂隙

- 中央不是 generic 魔法 portal，而是 `命格之門 / 命鏡裂隙 / 古老靈門`。
- 形狀宜偏鏡門、古框、裂隙、薄霧，不宜做成高彩度特效球。
- 左右世界需往中央互相滲透，不可像硬切比較表。

### 定心者

- 小型盤坐人影作為靈魂載體象徵。
- 不可大於左右主角。
- 可用 aura 顏色表達稀有度：
  - `金色`：傳說
  - `紫色`：史詩
  - `藍色`：稀有
  - `銀白`：低階或未完整覺醒

### 5 格命運故事帶

- 由左往右讀成一條完整故事，不做左右切開。
- 建議節奏固定為：
  1. `Mortal Beginning`
  2. `Heroic Rise`
  3. `Crisis or Sacrifice`
  4. `Bloodline Awakening`
  5. `Spirit Continuation`

## 狀態變體

同一版型至少應支援以下 state：

1. `loading-default`
2. `awakening-confirm`
3. `ascension-success`
4. `unowned-preview`
5. `owned-legendary`

差異主要落在：

- CTA 是否顯示
- tips 是否顯示
- 右側英靈亮度
- 定心者 aura 色
- 未持有 badge / 鎖定標記

## 可行性評估

### 高可行

- 左右雙人物大圖
- 中央命鏡裂隙
- 下方 5 格故事帶
- 頂部名稱列
- 未持有 badge / CTA / tips slot

以上都可視為靜態圖層 + 少量 UI 節點，適合現階段流程。

### 中可行

- 中央門霧化流動
- 定心者 aura 呼吸光
- 右側英靈邊緣淡閃

這些可先做低頻、低成本的 Cocos tween / sprite opacity / additive glow，不需追求重特效。

### 低優先，不建議首波硬做

- 長駐複雜粒子門
- 大量骨架動畫
- 故事帶自動長距離滑動
- 角色與英靈同步動態互視

## 與目前產能的適配

以目前團隊條件判斷，`血脈命鏡 v3` 屬於 `中高價值、可落地`：

1. 主體依然是靜態角色圖與固定版型。
2. 命鏡、英靈、定心者都可先用靜態圖層與簡單光效完成。
3. 最難的不是前端，而是：
   - 右側英靈角色圖 production
   - 5 格故事圖 production
   - 命紋 / 徽記 / 稀有度 aura 一致性
