## A. 系統描述

因子是《三國傳承》的遺傳基因單位，分為六色（藍/粉/綠/紫/白/紅）。因子在子嗣 5 歲開獎時觸發繼承，來源僅限父母與祖輩（教官不參與）。因子初始為隱藏狀態（灰色鎖頭），需透過培育窺探或戰場覺醒才能解鎖。因子與戰法系統聯動：綠色因子提供 TP 折扣，粉色因子影響戰場地形效果。

---

## B. 系統目的

1. **核心遺傳單位**：連結血統系統與培育系統的橋樑
2. **開盲盒樂趣**：初始隱藏→逐步覺醒的驚喜感
3. **策略深度**：六色各有功能，不同配種策略產生不同因子組合
4. **商業驅動**：「洗因子」是長線消耗（精力→補元散）
5. **差異化**：比《賽馬娘》多紫色（運氣）與紅色（體魄），策略更深

## I. Data Schema 需求

### 因子字典 (Tag Pool / 共用)
```json
{
  "Gene_Dictionary": [
    {"Gene_ID": "BLUE_STR_05", "Type": "Blue", "Attr": "STR", "Max_Level": 5},
    {"Gene_ID": "PINK_CAVALRY_03", "Type": "Pink", "Subtype": "兵種", "Name": "騎兵適性", "Max_Level": 5},
    {"Gene_ID": "GREEN_DRAGON_04", "Type": "Green", "Skill": "龍膽", "Max_Level": 5},
    {"Gene_ID": "PURPLE_LUK_03", "Type": "Purple", "Name": "天命之力", "Max_Level": 5},
    {"Gene_ID": "WHITE_CALM_02", "Type": "White", "Trait": "冷靜", "Max_Level": 3},
    {"Gene_ID": "RED_VIT_04", "Type": "Red", "Attr": "Lifespan", "Max_Level": 5}
  ]
}
```

### GeneObject（武將身上）
```json
{"Type": "Green", "ID": "GREEN_DRAGON_04", "Level": 4, "Is_Locked": true, "Discovery_Level": 20}
```

// 欄位定義說明：
// - Discovery_Level: 表示此因子在 UI 上對玩家可見的最小門檻（非真實解鎖）。例如 Discovery_Level=20 表示當玩家的培育等級或觀察能力 >=20 時，UI 會顯示因子摘要（但因子仍可能處於鎖定狀態）。
// - Is_Locked: 真正的運行時鎖定狀態。若 Is_Locked==true，該因子尚未被覺醒（不能提供效果），需透過戰場動作、培育窺探或名士指令進行解鎖。
// 兩者關係：Discovery_Level 決定玩家是否能在 UI 上「看到」該因子的存在或提示；Is_Locked 決定該因子是否實際生效。此設計讓視覺提示與真實可用性分離，方便設計驚喜提示與漸進式解鎖。

### Server 端 — 因子字典表
| 欄位 | 型別 | 說明 |
|---|---|---|
| Gene_ID | VARCHAR(24) PK | 因子代碼 |
| Type | ENUM('Blue','Pink','Green','Purple','White','Red') | 色系 |
| Name | VARCHAR(32) | 顯示名稱 |
| Max_Level | INT | 最高星等 |

### 爆發力加分因子標準化補充

下列項目屬於爆發力估算時可額外考慮的補正來源，與 `血統理論系統.md` (doc_spec_0011) 的 EP 公式共同構成完整評分依據：

| 加分項目 | 觸發條件 | 建議補正 |
