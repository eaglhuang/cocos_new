**💰 二、 傭兵 UI 與薪資懲罰機制：【人情壓力】**

為了增加留存，我們不能讓武將直接消失，要給玩家「救回」的機會：

1. **看板娘化**：傭兵武將會出現在主城 UI 或傭兵介面。  
   * **提醒**：「主公，營中糧草（黃金）快沒了，將士們心不穩啊...」  
   * **狀態按鈕**：巨大的 **\[贈送美酒 $0.99\]** 按鈕，旁邊顯示 **\[結義進度：4/6\]**。  
2. **【狀態凍結】(欠薪處理)**：  
   * 如果當天沒付黃金，該武將進入「罷工狀態」。  
   * **懲罰**：無法出征、無法當教官、結義進度停止。  
   * **24 小時倒數**：UI 出現紅色倒數。倒數結束前「補齊黃金 \+ 補一瓶酒」即可恢復正常。  
   * **保險機制**：傭兵武將在戰場上「不會被俘虜」（他會自動撤退回營），確保玩家的 $0.99 投資不會因為一場意外戰敗而打水漂。

**📣 三、 無廣告行銷手段：【人脈裂變計畫】**

既然沒有廣告預算，我們就利用「人脈」來換取「酒精」：

1. **「五虎將助陣」分享禮**：  
   * 推薦一位好友下載並進入遊戲（綁定推薦碼），**立刻送 5 瓶美酒**（價值約 $5 美金）。  
   * **誘因**：這 5 瓶酒剛好可以讓玩家「白嫖」一個傭兵名將到結義前夕。這對無課玩家是巨大的誘惑。  
2. **「家族血統」曬圖**：  
   * 在社群分享你子嗣的 \[關係描述\] 截圖（例如：導師是趙雲）。  
   * **獎勵**：每週首分享送 1 瓶美酒。這能讓《三國傳承》的血統特色在 Facebook/Line 群組瘋傳。  
3. **「酒鬼任務」**：  
   * 雖然不賣廣告，但我們可以設「每日極限任務」（例如：連勝 3 場 5x8 戰役）。  
   * **獎勵**：極低機率掉落「劣質高粱」（效果等於半瓶酒），讓純無課玩家靠「肝」也能留住心儀武將。

* **數值防線**：
