這是一個非常有野心的設計！將《Winning Post》（賽馬大亨）的**血統配種系統**與《三國志》的**武將養成**結合，能創造出極深的回合制策略與長期經營樂趣。

根據您提供的「戰鬥文件」 與「養成玩法」，我為您規劃了這套\*\*「五圍血統因子」\*\*系統。這套系統不僅決定戰場表現，更決定了家族繁衍的潛力。

## ---

**1\. 武將五圍：戰場效能與遺傳價值**

在您的遊戲中，五圍不再只是冷冰冰的數字，它們是會隨著代際傳遞的「基因片段」。

| 屬性 | 戰場 / 內政作用 (實質效果) | 遺傳因子作用 (Winning Post 邏輯) |
| :---- | :---- | :---- |
| **武力** | 決定**單挑**時的擲骰加成與傷害基礎。影響小兵衝锋時的戰力加倍幅度。 | **「鬥魂」因子**：高武力父母容易產出「早熟型」小孩，戰鬥起點高。 |
| **智力** | 影響**計謀能量 (SP)** 的獲取速度。決定防禦單挑挑戰時的「拒絕判定」精準度。 | **「智脈」因子**：高智力能抵銷基因突變中的負面數值，穩定後代的平均五圍。 |
| **魅力** | 影響**俘虜投降**的倒數計時（魅力越高，敵將越快投降）。影響后宮受孕率。 | **「名門」因子**：提高小孩出生時獲得「稀有傳家寶」或特殊技能的機率。 |
| **健康** | 決定**壽命**上限。高健康武將在 80 歲轉任教官後，提供的經驗加成更高。 | **「長青」因子**：極重要！決定後代的「成長型」（晚成/常盛），並降低夭折率。 |
| **幸運** | 影響單挑獲勝後「奪取裝備/坐騎」的機率。影響戰場隨機地板 Buff 的好壞。 | **「奇跡」因子**：決定**基因突變**的方向。幸運高者，突變容易往「全能力 \+10」發展。 |

**2\. 血統配種系統：仿《Winning Post 9》設計**

為了達成您提到的「培養出愈來愈強的小武將」，我們引入以下機制：

### **A. 因子活性化 (Factor Activation)**

* 如果父母雙方都有高\*\*「武力」\*\*，小孩會觸發「爆發力」補償，武力上限有機率突破父母的最高值。  
* **近親迴避**：如果讓小孩與旗下有血緣關係的武將結婚，雖然能力傳承穩定，但「健康」與「壽命」會大幅下降。

### **B. 傳家寶與坐騎的「繼承」**

* 當武將壽命結束時留下的**傳家寶**，如果裝備在後代身上，會產生「血緣共鳴」，額外增加 10% 的全屬性。這鼓勵玩家經營特定的「名門望族」。

### **C. 成長曲線 (早熟 vs 晚成)**

* **早熟**：24 小時（24 歲）即可達到巔峰值的 90%，但壽命較短，過了 60 歲數值就開始下滑。  
* **晚成**：24 小時（24 歲）僅有巔峰值的 40%，但 48 歲時會爆發出超越父母的極限數值，且健康度極高。

**3\. 戰場上的「血緣激勵」**

這部分結合您的單挑與衝鋒規則：

* **父子/夫妻同場**：當武將衝出去激勵士氣時，若場上有其子女或配偶，戰力加倍的回合數從 2 回合延長至 **4 回合**。  
* **復仇機制**：若父親在單挑中被俘虜或陣亡，其子女在場上會立即獲得滿額 **100 SP** 並觸發狂暴狀態。
