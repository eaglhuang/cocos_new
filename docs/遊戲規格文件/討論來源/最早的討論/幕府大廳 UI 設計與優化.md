### **5\. 核心主玩法：幕府議事廳 (The Shogunate War Room)**

#### **A. 視覺佈局與牆面 UI 設計 (Wall UI Architecture)**

大廳採左右對稱的宏大佈局，強調「列席感」與「功能區隔」：

1. **左側大牆：【全國軍事/外交牆】**  
   * **視覺特徵**：深紅色背景，鑲嵌金色邊框，氣氛嚴肅、肅殺。  
   * **下拉選單機制**：頂部為大分類標籤（如：曹魏動向、荊州戰線、外交互動）。  
     * 點擊下拉後，以列表形式呈現「夜襲、追擊、圍堵」等軍事任務。  
     * **任務標籤**：明確標註難度（一般至地獄）與預計獲得的「兵力/虎符」獎勵。  
   * **置頂功能**：已接取的任務會自動移至列表最上方並高亮顯示，方便玩家隨時點擊進入戰場。  
2. **右側大牆：【領地內政/人脈牆】**  
   * **視覺特徵**：暖木色或淡黃色背景，氛圍較為安定、和諧。  
   * **下拉選單機制**：頂部為類別標籤（如：治安維護、民生建設、人才尋訪）。  
     * 點擊下拉後，呈現「賑災、剿匪、宴請浪人」等內政任務。  
     * **任務標籤**：標註獎勵類型（金錢、糧草、武將親密度、官職經驗）。  
3. **中央區域：【世界沙盤 & 武將列席】**  
   * **世界沙盤**：一開始僅佔據中央一小塊區域，作為背景氛圍裝飾（飄出迷霧 Toast）。隨官職提升，可點擊擴展為全屏的「全球遠征視窗」。  
   * **武將群像**：大廳地面站立著玩家目前的主力名將。當左側牆面出現高難度任務時，對應的武將會做出「拔劍」或「思考」的動作，增加參與感。

2. **置頂邏輯**：

### **📈 總結**

這樣的入口設計讓玩家每天登入時，會有種\*\*「我是主公，眾將等我決斷」\*\*的強烈責任感。軍事與內政的雙牆區隔，讓不同類型的玩家（愛打戰的或愛經營的）都能迅速找到自己的目標。

**您覺得這兩面牆是否應該設計一個「一鍵掃蕩（委任）」的按鈕？** 針對那些「簡易」或「一般」難度的領地任務，讓武將自動處理，節省玩家重複操作的時間。
