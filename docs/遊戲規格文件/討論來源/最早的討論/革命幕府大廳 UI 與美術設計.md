**📜 《三國傳承：全球遠征》規格書：幕府大廳篇**

## **1\. 空間佈局與畫面感 (Environment & Visuals)**

### **A. 美術風格 (Art Direction)**

* **色調**：採暖色調的燭光與古樸木質感，營造一種深夜會議、眾志成城的「革命小圈子」氛圍，而非金碧輝煌的冰冷皇宮。  
* **構圖**：採用 **2.5D 透視視角**。玩家視角固定在殿堂入口處，左右兩側為巨大的任務牆，正中央延伸至深處為沙盤與武將。

### **B. 視覺層級 (Visual Layers)**

1. **背景層**：深夜的幕府建築內部，可以看到微微搖曳的燭光、屏風、以及遠處的盔甲架。  
2. **互動點位層**：大廳中央與牆邊分布著 6-8 個預設點位，武將隨機刷新於此。  
3. **UI 覆蓋層**：左右兩面巨大的牆面 UI 採半透明質感，與場景融為一體。

## ---

**2\. 核心 UI 佈局 (UI Interface Architecture)**

### **A. 左右雙牆系統 (The Dual-Wall System)**

| 區塊 | 名稱 | UI 表現形式 | 任務性質 |
| :---- | :---- | :---- | :---- |
| **左側大牆** | **全國戰報牆** | **紅色基調**，大型下拉式選單。 | **軍事外交**：攻城、夜襲、間諜圍堵、敵國軍閥動向。 |
| **右側大牆** | **領地事務牆** | **土黃色基調**，大型下拉式選單。 | **內政人脈**：賑災、治安剿匪、尋訪浪人、名將敘舊。 |

* **交互設計**：點擊牆面標題展開選單，已接取任務自動置頂。任務條上標註「難度」與「主責建議（如：建議武力型）」。

### **B. 中央世界沙盤 (Central World Sandtable)**

* **初始狀態**：擺放在大廳中央的木桌上，迷霧籠罩。顯示 Toast：*"世界情報不明，待查探。"*  
* **進階狀態**：隨官職解鎖「派遣任務」，可點擊放大為全螢幕地圖，查看羅馬、蒙古等海外列強動向。

## ---

**3\. 武將動態刷新系統 (Hero Dynamic Presence)**

### **A. 定點刷新機制 (Static Node Refresh)**

* **邏輯**：系統預設 N 個交互坐標。每次進入大廳或定時（如 60s）觸發 Refresh\_Hall()。  
* **表現**：隨機抽取 3-5 位已擁有的武將，閃現（Fade-in）至點位。頭頂懸浮醒目的 **\[官職\] 姓名**。

### **B. 原地互動動作 (Fake Idle Interaction)**

不使用路徑尋路，透過循環動畫營造「腦力激盪」感：

* **【思考者】**：站在軍事牆前，手托下巴，循環播放「點頭」與「嘆氣」動畫。  
* **【對談者】**：兩位武將刷新在沙盤兩側，循環播放「比劃手勢」與「對著地圖指點」。  
* **【文官】**：在內政牆前拿著奏摺，循環播放「翻閱」動作。

## ---

**4\. 系統摘要與技術重點 (Technical Summary)**

* **技術實現**：使用 **Prefab Pool** 管理武將模型，減少重複實例化的開銷。  
* **任務生成**：由 MissionEngine 根據「官職」與「軍師智力」動態填充雙牆選單內容。  
* **情緒反饋**：當玩家從「戰報牆」接下地獄任務，場內武將集體切換至 Salute（抱拳）動作 2 秒，隨後恢復循環動畫。
