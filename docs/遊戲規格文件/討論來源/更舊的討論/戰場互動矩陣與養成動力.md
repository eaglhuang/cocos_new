要讓《賽馬娘》式的養成有意義，5x8 的棋盤必須像是一盤\*\*「戰術殘局」**，玩家在養成階段選擇的屬性、計策、奧義，就是破解這些殘局的**鑰匙\*\*。

以下是針對您要求的「武將屬性 \+ 奧義 \+ 策略」在 5x8 戰場上的功能定義與互動矩陣：

## ---

**⚔️ 《三國傳承》5x8 戰術互動矩陣 (Tactical Interaction Matrix)**

### **1\. 武將五維屬性：決定戰場的「操作權限」**

在 5x8 棋盤中，屬性不只是加傷害，而是改變遊戲規則：

| 屬性項目 | 養成對標 | 戰場實際功能 (5x8 Grid) |
| :---- | :---- | :---- |
| **機動力 (速)** | 速度訓練 | **移動與優先權**：決定每回合可移動格數（3-5 格）。高速度可實現「繞後」，直接攻擊敵軍後排弓兵或軍師。 |
| **破陣力 (力)** | 力量訓練 | **位移與粉碎**：攻擊時有機率「擊退」敵人。若敵後方有「落石」或「火海」，可將敵軍撞入陷阱造成雙重傷害。 |
| **計謀力 (智)** | 智力訓練 | **範圍與命中**：決定「場外軍事任務」的覆蓋範圍（如火燒範圍從 $2\\times2$ 變 $3\\times3$）與計策發動的成功率。 |
| **韌性 (意志)** | 意志訓練 | **反制抵抗**：降低被敵方計策干擾的機率（如水淹不退後、火燒不扣生命上限）。 |
| **運勢 (運)** | 休息/事件 | **資源掠奪率**：擊破敵軍或落石時，獲得高階「教官」或大量「糧草」的機率。 |

### ---

**2\. 場外策略 (Strategies)：對應「敵人兵種與棋盤地形」**

策略是由高智力武將在場外發動的「降維打擊」，專門用來處理**大面積敵軍兵團**：

* **🔥 火燒連環 (針對：密集盾兵方陣)**  
  * **棋盤玩法**：若敵方多個單位相鄰，火焰會隨回合\*\*「傳導」\*\*。  
  * **戰術價值**：逼迫敵方散開陣型，瓦解其防禦加成。  
* **🌊 水淹圍堵 (針對：高機動騎兵衝鋒)**  
  * **棋盤玩法**：大水由我方推向敵方。敵軍遇水**強制後退**並進入「泥濘」狀態（移動力 \-2）。  
  * **戰術價值**：防止敵方騎兵迅速切入我方後排。  
* **🌲 森林埋伏 (針對：敵方遠程弓兵)**  
  * **棋盤玩法**：在特定格位設置「陷阱」。敵軍進入後強制停止移動並被\*\*「鎖定」\*\*，無法發動遠程攻擊。  
  * **戰術價值**：廢掉敵方的遠程火力輸出。  
* **🪨 峽谷落石 (針對：敵方固定箭塔/營帳)**  
  * **棋盤玩法**：指定格子出現大落石。敵方若不離開該格，每回合受 50% HP 傷害。  
  * **戰術價值**：拆除敵方防禦工事，我軍擊破後還能回收「石材」強化本體。

### ---

**3\. 武將奧義 (Ultimates)：對應「敵人主將與精英」**

奧義是培育中學到的「殺手鐧」，專門用來處理**敵方主將 (Boss)** 的特殊機制：

* **🛡️ 破軍咆哮 (張飛類奧義)**  
  * **對抗機制**：對付擁有「護盾」或「格擋」的敵將。  
  * **效果**：震懾周圍 $3\\times3$ 敵軍，強制破除所有防禦狀態，並讓敵將進入「恐懼」狀態，傷害大幅降低。  
* **⚡ 萬劍歸宗 (刺客型奧義)**  
  * **對抗機制**：對付躲在小兵後方的敵方軍師主將。  
  * **效果**：無視中間所有棋子，直接瞬移至敵將身後發起致命一擊。  
* **🤝 英靈附體 (傳承型奧義)**  
  * **對抗機制**：當我方兵力陷於劣勢時。  
  * **效果**：召喚培育過程中所繼承的「祖先」虛影，同步施放祖先的奧義，實現一回合兩次大招逆轉。

### ---

**🏆 為什麼這樣才有動力培育？ (The Challenge)**

如果沒有這套機制，玩家只需要養「高傷害」就好。但有了這套系統，關卡設計可以變成：

**【挑戰關卡：定軍山】**

* **戰況**：敵將夏侯淵在高地，後路被巨石封死，前排是無敵盾兵，且每 2 回合發動一次全圖箭雨。  
* **玩家的解法（動力來源）**：  
  1. 必須培育一個**高智力**子嗣，帶領「火燒計策」去燒掉盾兵。  
  2. 必須培育一個**高機動**子嗣，使用「破魔奧義」去沉默夏侯淵的箭雨。  
  3. 必須培育一個**高力量**子嗣，去撞開後路巨石，讓夏侯淵失去防禦加成。

**這就是決策的趣味：玩家會為了贏得這場「定軍山大賽」，去針對性地尋找適合的「支援卡/女官」，並反覆培育出最完美的「子嗣實例」。**
