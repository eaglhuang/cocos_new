## **第一部分：登入大廳 — 「幕府議事廳」**

大廳不採用冰冷的選單，而是一個動態、有生命的「創業團隊會議室」。玩家視角固定在殿堂入口處，採用 **2.5D 透視視角**，營造出「眾將等我決斷」的豪氣。

### **1\. 空間佈局與美術氛圍 (Environment & Art)**

* **視覺風格**：採暖色調的燭光與古樸木質感，營造「深夜會議、革命小圈子」的沉浸感，而非金碧輝煌的皇宮。  
* **動態環境（天時）**：系統根據玩家當地時間自動切換 **白天/夜晚** 氛圍。白晝清亮 actively；夜晚則轉為暖橘色燭光，並加入微弱蟲鳴。  
* **節慶變裝（商業化）**：大廳背景、裝飾層與前景粒子（如浮塵、螢火蟲、春節紅屑）均為可替換組件。節慶時可一鍵替換背景大圖並啟用特定裝飾 Prefab（如紅燈籠），以最低成本營造節日氛圍。

### **2\. 核心 UI 組件 (Core UI Components)**

大廳採左右對稱的宏大佈局，強調「列席感」與「功能區隔」：

| 區塊 | 名稱 | UI 表現形式 | 任務性質與獎勵 |
| :---- | :---- | :---- | :---- |
| **左側大牆** | **【全國軍事/外交牆】** | **深紅色基調**，斜切透視大牆。採大型下拉式選單（Prefab 動態生成）。 | **軍事外交**：攻城、夜襲、間諜圍堵。獎勵核心為：兵力、虎符碎片、稀有武將材料。已接取任務自動置頂。 |
| **右側大牆** | **【領地內政/人脈牆】** | **暖木/土黃色基調**。同樣採大型下拉式選單。 | **內政人脈**：賑災、治安剿匪、尋訪浪人、敘舊。獎勵核心為：金錢、糧草、加速卡、武將親密度/經驗、官職經驗。 |
| **中央區域** | **【世界沙盤 & 武將列席】** | 一開始僅佔中央一小塊區域，作為背景氛圍裝飾（飄出迷霧 Toast）。**隨官職提升解鎖**，可點擊擴展為全屏的「全球遠征視窗」。 | **Endgame/終極目標**：統一中原後，解鎖地獄/神話難度的遠征任務。課長可派遣主力部隊與 5 星英靈虎符硬推；無課則需極致利用兵種互剋。此區有機率抓取全服最強課長的 JSON 快照作為「鏡像 AI」敵軍。 |

### **3\. 人性化互動系統 (Humanized Interaction)**

* **武將動態參與**：大廳採\*\*「定點刷新、原地互動」\*\*機制（MVP 邏輯）。  
  * **流動感**：系統預設 N 個熱點坐標，每次進入大廳或定時（如 60s）觸發 refreshHall()，隨機抽選 3-5 位已擁有武將「閃現」至點位，頭頂常駐顯示醒目的 \*\* 姓名\*\*。  
  * **腦力激盪動作**：不需移動，透過循環動畫營造忙碌感。例如站在軍事牆前「抬頭看牆」→「點頭」→「低頭思考」；兩武將在沙盤兩側「比劃手勢」→「指點地圖」。  
* **自告奮勇**：點擊牆上任務時，旁邊契合武將有機率直接快步走上前發言（如張飛剿甘寧：*"此等小賊，何須主公親出，末將願往！"*）。答應他，該任務轉為派遣任務，且**成功率直接上升 50%**。  
* **任務卷軸 (商城連動)**：商城販售「神秘任務卷軸」（只知難度），玩家可刷新出高級/菁英以上任務，解決卡關焦慮。

**第二部分：許願商城 — 「誠心祭壇」**

摒棄現代感強烈的選單，將商城包裝成一個具備儀式感的\*\*「家族祭壇」**或**「書房奏摺」\*\*。

### **1\. 視覺風格 (Visual Vibe)**

整體介面採古樸木質色調，背景為隱約可見的家祠祭壇或是一張展開的長幅絹布。

### **2\. 佈局說明 (Layout)**

* **頂部狀態欄 (Header)**：左側顯示「糧草」、「金錢」；右側顯示「鑽石」與「通用加速卡數量」。中央顯示玩家的家族堂號（例如：穎川荀氏）。  
* **中央核心區：【三巡祭壇】**。分為三個「奏摺」插槽（Slot 1-3），對應早、中、晚三個時段的高 CP 值許願商品：  
  * **空位狀態**：顯示虛擬的「祭拜」手勢，文字提示：*「請主公誠心許願」*。  
  * **進行狀態**：顯示捲起的奏摺，上方有沙漏倒數圖標。⏳  
  * **完成狀態**：奏摺散發淡淡金光，提示：*「顯靈！點擊領取」*。領取時伴隨名將半身像的賀語：*「主公至誠感天，此物已為您備好。」*。  
* **底部切換區 (Tabs)**：採古樸木質選單，包含三個頁籤：  
  1. **【珍寶閣】**：即時課金區（現貨、禮包）。  
  2. **【誠心願】**：本系統核心（時間償還區）。  
  3. **【陳列室】**：已獲得英靈虎符與傳家寶展示。

### **3\. 人性化事件與技術實作 (人性化事件 & 技術)**

* **人性化事件**：許願過程中可能隨機觸發武將性格事件，如張飛突現「神速」踢碎沙漏（時間減半）；小喬在旁「多得」放花（領取數變 2 張）。  
* **Vibe Coding 數據驅動**：使用簡單陣列管理插槽狀態（id, targetItem, startTime, duration, isFinished）。在 update() 或 onEnable() 時，讀取本地存儲進行離線進度校驗。

**第三部分：核心戰鬥棋盤 — 「プレミアム立牌」 (Premium Card-Stand Style)**

為了發揮 **AI 繪圖** 優勢並規避 3D 動畫技術坑，我們採用務實且視覺精緻的 **「3D 棋盤 \+ 2.5D 精美立牌」** 風格。

### **1\. 核心棋盤與立牌系統 (Core Battlefield & Stand System)**

* **5x8 棋盤**：保留 5x8 的棋盤格與俯視視角。棋盤地板改為更真實的風化石磚與木紋。  
* **精美立牌 (Premium Card-Stand System)**：所有單位均為 **去背將領立繪**（Premium go-by illustration），張貼在透明材質的板子上，並立於一個石質/金屬底座上（簡單的 cc.Cylinder 或 cc.Box）。  
  * **兩邊主將**：  
    * **我方將領** (張飛)\] 立牌位於棋盤左下角。底座為較大的金色石質圓柱，並有特殊的金色發光特效。  
    * **敵方將領** (曹操)\] 立牌位於棋盤右上角。底座為紅色圓柱。  
    * 兩位主將的立牌尺寸明顯大於小兵，頭頂常駐顯示醒目的 \*\* 張飛\*\* 與 \*\* 曹操\*\* 名稱標籤。  
  * **兵種立牌**：取消全身 3D 模型站位。替換成較小的 2.5D 紙片人立牌（sprite），頭頂常駐顯示醒目的 \*\* 兵種名稱\*\*。底座顏色區分敵我（我方藍、敵方紅）。頭頂懸浮血條、兵種 Icon 符號（如步兵劍符號）與數值 UI。

所有 UI 按鈕都設計成扁平化、正面對鏡頭、手機好按的大小與位置：

| 區塊 | 名稱 | UI 表現形式與交互 | 開發重點 |
| :---- | :---- | :---- | :---- |
| **左上角** | **Player Profile** | 顯示客製化的書寫體 Icon (傳)，旁邊清晰標註了玩家姓名「 張飛」與官職。 | **官職驅動**：官職決定大廳生成的任務難度上限（簡易至地獄）, Lv99\]。 |
| **最左邊** | **兵種按鈕** | 垂直排列的矩形 按鈕組。包含步兵、槍兵、盾兵、弓兵、騎兵。每個按鈕都有清晰的 2D Icon 與文字。 | **一鍵委任 AI**：按鈕上應有明顯標註，針對簡易/一般任務可讓 AI 自動處理，節省玩家重複操作。 |

### **3\. 技術實作重點 (Technical Focus)**

* **Vibe Coding Tween 動畫**：利用 cc.tween 實作立牌的「呼吸縮放（Q彈）」、跳躍移動、受擊劇烈抖動、以及放大遮螢幕配合全螢幕 2D 特效的「必殺技」動作。  
* **Prefab 重複利用**：小人偶們圍繞中原地圖討論（營造腦力激盪氛圍），武官（張飛）、軍師（諸葛亮）、女官（黃月英）都有各自多套服裝設計，場內隨機刷新不同服裝。方便區分不同武將同時滿足輕量化與多樣性。
