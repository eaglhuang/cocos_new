**📑 《三國傳承》英靈供養與家族血脈系統：正式規格書**

## **1\. 英靈卡：世界唯一性與覆蓋機制 (Heroic Uniqueness)**

* **唯一性**：每個 SSR 武將在當前存檔世界中僅擁有一張英靈卡。  
* **英靈覺醒**：當武將（如：關羽）首次離世，生成基礎英靈卡。  
* **屬性覆蓋 (Best-of-Both)**：若玩家在後續流程（如轉世或二週目）中再次使用關羽並離世，系統將進行「數值比對」。  
  * **邏輯**：取 \[舊卡基礎值\] 與 \[新世終點值\] 之**最高者**進行覆蓋。  
  * **深度意義**：這會激勵玩家在某一世中「逆天改命」，例如刻意培養關羽的智力或魅力，將其英靈卡從單純的「武神」洗鍊成「文武全才」。

## **2\. 姓氏防線：零和繼承規則 (The Surname Wall)**

* **抉擇代價**：子嗣出生後，姓氏二選一。  
* **貢獻歸屬**：  
  * **絕對化**：隨父姓則父家獲益，隨母姓則母家獲益。**嚴禁雙邊受益。**  
* **絕學斷絕 (Exclusive Arts)**：  
  * **家傳武學限制**：每個顯赫家族皆有專屬絕學（如：關家 \[青龍刀意\]、祝融系 \[烈火飛刀\]）。  
  * **門檻**：子嗣必須**冠其姓氏**方可習得或發揮該絕學的 100% 威力。若關家女兒外嫁且小孩隨趙姓，該小孩將徹底失去習得「關家絕學」的資格。

## **3\. 本尊與後代：供養分工邏輯 (Ancestry Synergy)**

* **後代職責 (The Battery)**：只有冠名的後代子嗣在戰場上立功、在主城內政，其產出的「功勳/人頭積分」才會注入英靈卡。  
* **本尊職責 (The Executioner)**：  
  * **不計功勳**：重生版（轉世）的關羽本尊在戰場上的行為**不產生任何積分**。  
  * **究極加成**：本尊存在的唯一目的，是配戴英靈卡獲得 **20%** 的全屬性加成，成為戰場上的終極兵器。  
  * **意義**：這創造了「老祖宗帶隊，子孫拼命刷分供養」的有趣場景。

### **A. 英靈卡更新邏輯 (TypeScript)**

TypeScript

function updateHeroicSpirit(existingSpirit, newlyDeceasedHero) {  
    // 遍歷五維屬性，取最高值覆蓋  
    for (let attr in existingSpirit.baseStats) {  
        existingSpirit.baseStats\[attr\] \= Math.max(  
            existingSpirit.baseStats\[attr\],   
            newlyDeceasedHero.finalStats\[attr\]  
        );  
    }  
    // 更新後發送「祖靈進化」視覺通知  
}
