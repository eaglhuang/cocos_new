## **0\. 核心設計哲學 (Core Philosophy)**

* **定錨點**：出生時，子嗣已根據父母血統（image\_12.png 的 14 人家族樹）生成了底層的 **「起始點 (Base)」** 與 **「潛力上限 (Max Potential)」**，但在 UI 上完全隱藏（顯示為 ? 或 未覺醒）。  
* **樂趣源泉**：**「開榜」** 的悸動。玩家必須透過讓子嗣參與活動，才能逐步「試出」其才華，並決定其未來的培養方向（例如：發現其 INT 上限極高，則轉向文官培養）。

**1\. 屬性生成與隱藏邏輯 (Generation & Hiding Logic)**

### **1.1 底層屬性 (The 5 Main Attributes)**

對標經典三國遊戲，設定五大基礎屬性：

* **武力 (STR)**：決定戰鬥傷害、單挑能力。  
* **智力 (INT)**：決定謀略成功率、內政效率。  
* **統帥 (LEA)**：決定部隊防禦力、徵兵效率。  
* **政治 (POL)**：決定外交、人才搜尋、市場收益。  
* **魅力 (CHA)**：決定登用人才、民忠回覆、生小孩（配種）因子繼承率。

**2\. 終身成長三階段 (Three Phases of Life)**

### **2.1 第一階段：襁褓與啟蒙 (0-3 歲) — 【天命初現】**

* **內容**：子嗣僅能進行簡單的互動（如：抓周、起名）。  
* **隱藏 UI 表現**：所有屬性皆為 ?。  
* **唯一「開榜」機會**：**抓周事件**。  
  * **操作**：玩家選擇抓取物品（刀、書、印、金）。  
  * **效果**：**隨機隱性增加** 某一屬性的 \[潛力上限\]。例如：抓刀，底層數據 STR Max \+2，UI 不變。  
* **商業化定錨**：抓周時提供 **$0.99 絕世抓周道具包**（可同時抓兩樣物品，加倍潛力提升）。

### **2.2 第二階段：培訓與遊歷 (3-16 歲) — 【才華試煉】**

這是屬性「開榜」的核心階段。子嗣不直接上戰場，而是進入不同的「培訓所」。

* **2.2.1 內政培訓所（內政任務）**  
  * **操作**：指派子嗣參與市場管理（POL/INT）、農田徵收（LEA/STR）。  
  * **開榜邏輯**：任務成功時，**底層數據（Current）增加**。當 Current 達到某個閾值，或觸發特定成功率時，**該屬性的「潛力上限」在 UI 上「覺醒」顯示**。例如：多次市場管理後，UI 顯示：政治潛力：\[? \-\> 80 (覺醒)\]。  
* **2.2.2 演武場/講武堂（培訓）**  
  * **操作**：修習武藝（STR/LEA）、研讀兵法（INT/POL）。  
  * **開榜邏輯**：類似內政。修習武藝可能覺醒 STR 起始點；研讀兵法可能覺醒 INT 潛力上限。  
  * **特殊事件：名師授業**。若有名師（如：諸葛亮、周瑜）在場，觸發 INT/POL 潛力上限提升事件，並大幅增加「開榜」機率。  
* **2.2.3 商業化定錨**：  
  * **$0.99 加速培訓券**：立即完成 10 次培訓，並大幅增加「起始點/潛力上限」開榜機率。  
  * **$0.99 名師課堂**：隨機一位 SSR 文官/武將親自授業 7 天，保證覺醒至少一個隱藏潛力。

### **2.3 第三階段：成年與戰場 (16歲+) — 【傳承覺醒】**

此時子嗣正式成為「武將/文官」，所有已覺醒的屬性皆可自由培養與升級。

* **2.3.1 戰場試煉**：  
  * **開榜邏輯**：上戰場戰鬥、觸發「雲龍斬」等綠色因子技能、或參與外交事務。  
  * **效果**：戰鬥勝利或事件成功，有極高機率**覺醒未顯現屬性的起始點與潛力上限**。例如：打了兩場仗，UI 顯示：統帥：\[? \-\> 98 (全覺醒)\]。  
* **2.3.2 因子傳承（生小孩）**：  
  * **開榜邏輯**：當 CHA（魅力）屬性的「潛力上限」被覺醒並培養至 Lv5 時，生出來的下一代將獲得更強大的 **🟣 紫色（運氣）因子** 與 **遺傳 bonus**（繼承父母最強屬性的機率）。  
* **2.3.3 最終定錨：血統繼承儀式 ($29.99)**：  
  * **操作**：當子嗣成年，所有屬性潛力皆已覺醒，可付費與其父母（例如：關羽+祝融）進行「血統儀式」。  
  * **效果**：將關烈原本隱藏但極高的屬性（例如：武力潜能 105）直接注入關羽的身體中，關羽覺醒出更強大的 **SP 關羽（萬人敵・火神之魂）**。這是大課長的終極圓夢點：**親手打造一個比原作更強的宿命名將！**
