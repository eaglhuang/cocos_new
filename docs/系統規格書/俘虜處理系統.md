---

## A. 系統描述

戰場中擊敗敵方武將後，被俘武將進入「俘虜」狀態。玩家有四種處置選項：❶ 勸降（成功率由魅力/運氣決定）、❷ 招募（消耗資源直接加入）、❸ 無期徒刑（關押取得情報/資源）、❹ 放走（獲得名聲+未來可能回報）。每種選擇有不同的資源回報與長線效果。

// 補充：傭兵（來源於傭兵系統／Template.Flag = MERC_TRIAL）為試用角色，為保護付費/交易行為，**傭兵在戰場上不可被俘虜或納為 POW 選項**；系統在產生 POW 選項時，將過濾傭兵來源（即：若 Target.IsMercenary == true，則不出現勸降/招募選項，僅允許放走或移交）。

---

## B. 系統目的

1. **戰後決策層**：戰鬥不只是打贏，處置俘虜是第二層策略
2. **資源多元化**：不同處置=不同類型資源回報
3. **道德選擇感**：放走 vs 關押的人性抉擇
4. **社交話題**：「我放走了呂布」vs「我把呂布關了一輩子」

## F. 公式相關

### 勸降成功率
```
Persuade_Rate = Base(20%) + CHR_bonus + LUK_bonus - Target_Loyalty
CHR_bonus = (Persuader.CHR - Target.CHR) × 0.5
LUK_bonus = Persuader.LUK × 0.2%
Target_Loyalty = Target.Loyalty × 0.3%
```

### 招募消耗
| 目標稀有度 | 黃金 | 兵力 |
|---|---|---|
| R | 500 | 200 |
| SR | 2000 | 500 |
| SSR | 8000 | 1500 |

### 放走回報
```
Fame_Gain = Target.Rarity × 100
Future_Return_Chance = 5%  // 未來某季該武將主動來投
```

### 監獄每季產出
```
Intel_Per_Season = Target.INT × 0.1
Resource_Per_Season = 50 + Target.POL × 0.5
```

## K. 戰後處置延伸分支補充

> **整併來源**：轉蛋與戰場武將處置系統.md
> **衝突處理**：戰後主流程仍維持正式版的四選一介面；下列內容屬於四選一之後的上下文分支，不另開第二套主選單。

### 四選一之後的延伸結果

- **勸降**：可額外套用 `結義金蘭卡`、名聲或忠誠度修正，作為高階招降強化手段。
- **招募**：招入後可在治理模式透過送禮、和親或嫁女兒等方式鞏固關係，必要時轉入家族繁衍鏈。
- **無期徒刑**：除情報 / 資源外，可追加「傳家寶物」掉落池，強化長期關押的差異價值。
- **放走**：除名聲外，允許觸發「傳道 / 頓悟」結果，讓我方隨機武將學會或升級對方代表戰法。

### 年齡與身分類型處理

- 若俘虜或城破後擄獲的目標年齡已高（例如 60 歲以上），勸降 / 招募成功後可優先轉為教官卡或支援角色。
- 若目標屬於女性武將、教官或特殊治理人才，仍使用同一套四選一主流程，不另外裂出專用處置介面。
- 傭兵來源角色依本檔前文規則維持過濾，不因早期討論而恢復為可完整俘虜對象。
