- 肩甲扣件 / 披肩扣件

## 落地原則

- 男武將優先強化胸甲、腰封、肩甲扣件與武器掛飾。
- 女武將優先強化胸甲、腰封、髮飾、玉佩與肩扣，但不能只做成珠寶裝飾。
- 血脈面 / 英靈面可在胸口、腰封、額心、武器脈紋上增加命紋浮現，但不能變成恐怖裂痕。
- 同一角色的 slot 語言需回流到 UI crest、badge、虎符兵牌、特種軍隊識別。
