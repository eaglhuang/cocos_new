# 武將壽命系統規格書

> **來源文件**：遊戲時間流逝與壽命機制.md、名將時空保值制：玩家導向設計.md、遊戲機制：精氣、大器、回流.md

---

## A. 系統描述

每位武將有壽命上限（Max_Lifespan），受紅色因子、精力狀態、戰傷等影響。武將一生分為多個階段：幼兒(0~5)→少年(5~15)→青年(15~40)→壯年(40~55)→老年(55~65)→退役候選(65+)。壽命到期後武將可退役轉教官或死亡。名將通過「演武聖境」等機制保值，避免課金名將過早廢棄。

---

## B. 系統目的

1. **世代壓力**：壽命有限→玩家必須持續培養下一代
2. **退役回流**：老將轉教官→延續價值→課金保值
3. **紅色因子價值**：壽命=紅因子的核心功能
4. **名將保值**：花大錢抽到的名將不會因年齡直接報廢

## F. 公式相關

### 壽命上限
```
Max_Lifespan = Base_Lifespan(65) + Red_Gene_Bonus + Random_Variance(-5~+10)
Red_Gene_Bonus = Red_Gene_Level × 3  // 紅色因子每星+3歲
```

### 死亡率（超過壽命時）
```
Over_Age = Current_Age - Max_Lifespan
if (Over_Age > 0):
    Death_Chance_Per_Season = min(0.8, Over_Age × 0.1)
```

### 年齡階段能力修正
| 階段 | 年齡 | 能力修正 |
|---|---|---|
| 幼兒 | 0~5 | 不可用 |
| 少年 | 5~15 | ×0.6 |
| 青年 | 15~40 | ×1.0 |
| 壯年 | 40~55 | ×1.1（巔峰） |
| 老年 | 55~65 | ×0.8 |
| 超齡 | 65+ | ×0.5 + 退役候選 |

### 名將保值（演武聖境）
```
// 160 季壽命計算（40 年 × 4 季）
// 前 20 歲培育+成長，20~55 歲巔峰戰力，55~65 歲衰退
// 演武聖境：死後精華快照=TP×0.5 轉為可繼承資源
```

## K. 壽命消耗設計（雙軌計時制補充）

> **來源**：遊戲時間與壽命機制改良.md、養成循環與營收節奏探討.md

### 不老哲學

武將只有在**執行主動指令**時才消耗壽命。純粹的掛機/放置狀態下，武將壽命**凍結不流逝**。這讓玩家能理性控制每位武將的壽命消耗速率，同時為付費道具（補元散、入學令）提供策略性消耗動機。

### 壽命消耗表

| 行動類型 | 壽命消耗 | 說明 |
|---|---|---|
| 放置掛機 | 0 月 | 無主動指令→壽命凍結 |
| 培育指令（每次） | 1 月 | 每次執行培育回合 |
| 攻城行動（每次） | 3 月 | 發動攻城戰役 |
| 結緣行動（每次） | 6 月 | 執行配種結緣 |

### 精力值（Vigor）系統

武將體力狀態，獨立於壽命之外，影響結緣冷卻與戰鬥出擊效率。

```
Vigor_Recovery_Rate = 1 unit / governance_month  // 每治理月自然回復 1 精力
Post_Breeding_Cooldown = 3 governance_months      // 結緣後女方精力凍結恢復 3 個治理月
Lifetime_Breeding_Cap = 3~5 次                    // 每位武將一生最多親傳子嗣次數
Vigor_Low_Penalty: Vigor < 30% → 受孕率 ×0.1     // 精力不足時大幅降低受孕率
```

> **與壽命的關係**：精力影響「能否行動」，壽命影響「行動後的代價」。兩者共同構成武將生命力的雙軌管理機制。

## L. 精力共用制與戰場軟懲罰補充

> **整併來源**：精力系統優化建議.md、遊戲機制：精氣、大器、回流.md
> **衝突處理**：本章為現行壽命 / 精力規格的補強，不另立第二套疲勞值系統。

### 單一精力資源原則

- 專案不再拆分疲勞值與精氣值，統一使用 **精力（Vigor / Vitality）** 作為戰鬥與傳承共享資源。
- 戰場主動戰法、強行軍、結緣等高價值行為都消耗同一條精力，讓「打仗」與「傳承」形成同一套節奏博弈。
- 普攻與一般移動屬於低負擔行為，不應因精力不足被整體封鎖，避免無課玩家直接失去可玩性。

### 男女武將的精力職能差異

| 類型 | 設計傾向 | 正式規則含義 |
|---|---|---|
| 男性武將 | 戰場持久型 | 精力上限可較高，較適合長期出征與多次戰法循環 |
| 女性武將 | 傳承爆發型 | 懷孕 / 待產鎖定成本更高，但在精力滿載時更容易產出高品質子嗣 |

- 上表是數值設計方向，不代表強制的性別數值模板；實際數值仍由角色資料、紅色因子與事件決定。
- 若女性武將進入待產期，精力回復凍結與暫離戰場的規則仍以 `結緣系統（配種）.md` 為準。

### 戰場中的精力軟懲罰

| 精力區間 | 戰場效果 |
|---|---|
| >50% | 可正常發動主動戰法與高耗能行為 |
| 10%~50% | 禁止高耗能終極戰法，但仍可使用基本技能 / 普攻 / 移動 |
| <10% | 仍可移動與普攻，但不得再發動任何主動戰法，且撤退風險提高 |

> 原則是「效率下降」而不是「禁止遊玩」。精力不足會讓武將變平庸，但不應讓玩家連基本操作都失效。

### UI 可視化要求

- 武將頭像旁應預留精力狀態的常駐視覺提示，例如火苗、氣焰或能量環。
- 精力充足時用飽和冷色 / 紫光表示「可作高價值決策」；精力接近枯竭時改為灰暗或弱火提示。
- 這種提示屬於資訊視覺化，不應取代數值條本身；玩家仍需能讀到明確百分比或區段狀態。
