## A. 系統描述

運氣（LUK）是第六屬性，對應紫色因子（🟣天命）。運氣不直接影響戰鬥數值，而是透過「天命波動」公式修正爆發力（EP），並影響事件觸發率、覺醒速度、暴擊概率等。紫色因子在 Lv46 以上才可見（Phase 3 解鎖），形成後期天花板關鍵。

---

## B. 系統目的

1. **隱藏天花板**：前中期看不到 LUK，後期才發現「這才是最重要的」
2. **EP 公式隨機性**：天命波動讓相同血統配種出的子嗣有差異
3. **後期驅動**：紫色因子成為 PvP 終局的勝負手
4. **差異化**：比《賽馬娘》多一層運氣維度

## F. 公式相關

### 天命波動（EP 修正項）
$$天命波動 = LUK \times Random(-0.5, +1.5)$$

### EP 完整公式
$$EP = 50 + [共鳴] + [異脈] + [契合] + [天命波動(LUK)]$$

### LUK 對事件觸發
```
Event_Rate = Base_Rate × (1 + LUK / 200)
// LUK=100 → 事件率 +50%
```

### LUK 對覺醒速度
```
Awakening_Speed_Modifier = 1 + (LUK / 300)
// LUK=90 → 覺醒速度 ×1.3
```

### 紫色因子對 LUK
```
LUK_Max = 50 + Purple_Gene_Level × 10
// ★5 紫因子 → LUK 上限 100
```

### 繼承成功率修正（配種補充）

> **來源**：遊戲系統設計：運氣、刷寶、轉蛋優化.md

LUK 正向影響子嗣繼承父母優質因子的成功率，是配種策略的隱性天花板。

```
Inheritance_Rate = Base_Rate + (ParentAvgLUK × 0.01)
// 父母平均 LUK=60 → 繼承成功率 +60%（相對基礎）
ParentAvgLUK = (Father.LUK + Mother.LUK) / 2
```

### LUK 特殊門檻效果

| LUK 門檻 | 效果 |
|---|---|
| LUK ≥ 80 | 觸發「戰法強制複製」：父親最強戰法 100% 繼承 |
| LUK ≥ 95 | 觸發「血統變異」：隨機一個因子突破 ★ 等（上限 ★4） |
| LUK = MAX(100) | 可解鎖「天命傳承」：額外繼承一個曾祖代因子 |
