# 未持有武將標記規則補遺 2026-04-04

## 套用範圍

- 日常人物頁 `GeneralDetailBloodlineV3`
- `BloodlineMirrorLoading`
- `BloodlineMirrorAwakening`

## 正式規則

1. 未持有武將仍顯示完整立繪，不使用黑影剪影。
2. 狀態識別以 `badge / ribbon / hint / 輕 veil` 為主。
3. Loading 與命鏡頁可用較高濃度遮罩；日常人物頁維持低濃度。
4. 用詞以 `尚未結識 / 待招募 / 命鏡現影 / 英靈未納` 為主。
5. 禁用 `未開放 / 敬請期待 / ???` 等偏功能鎖定語彙。

## Skeleton 對應

### General Detail

- `gdv3.unowned.ribbon`
- `gdv3.label.unowned`
- `gdv3.unowned.badge`

### Bloodline Mirror

- `mirror.unowned.veil`
- `mirror.unowned.badge`
- `mirror.label.unowned`

## 製作建議

- 優先先做一套中性 badge family。
- 後續若需要，再延伸 faction / rarity 變體。
- 不新增高成本動態 unlock 特效，先用靜態版型成立。
