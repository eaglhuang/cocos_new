# 血脈命鏡過場載入規格書

> **文件目的**：定義《三國傳承》專屬的高世界觀過場載入畫面，用於 Loading、英靈升星、二轉突破、血脈覺醒等高光場景。
> **適用範圍**：場景切換 Loading、角色覺醒頁、升星頁、限定活動角色展示頁、未持有武將預覽頁。
> **定位**：這不是日常人物詳情頁，而是 `品牌級世界觀展示畫面`。

---

## 1. 核心定位

血脈命鏡過場圖的任務不是提高資訊效率，而是把《三國傳承》的核心主張一次講清楚：

- 名將不是只有歷史表象
- 每位名將都有內在血脈面
- 角色的命運、勇猛、魅力與執念，與古老血脈密切相關
- 玩家未持有該名將時，也能透過過場畫面產生收藏慾與抽卡動機

一句話定位：

> `平常看人，覺醒看魂；過場看命。`

---

## 2. 使用場景

### 2.1 正式適用場景

1. 場景切換 Loading 畫面
2. 名將升星成功後的過場畫面
3. 名將二轉 / 覺醒成功畫面
4. 血脈卡 / 英靈虎符啟動前的演出載入畫面
5. 限定活動頁的角色世界觀展示圖

### 2.2 不適用場景

1. 日常人物詳情頁
2. 大廳武將列表卡
3. 一般 tooltip 或碎片說明視窗
4. 需要高密度數值閱讀的常駐資訊頁

---

## 3. 畫面骨架

### 3.1 版型結構

畫面採 `上重下輕` 的三段式結構：

1. **上方主舞台**
   - 左側：名將表象面
   - 右側：名將血脈面 / 英靈面
   - 中央：命鏡裂隙 / 空間門 / 命格之門
2. **中央儀式節點**
   - 一名小型 `定心者 / 承載者`
   - 象徵一具肉身承載兩面靈魂
3. **下方命運故事帶**
   - 5 格或 6 格連續故事圖
   - 由左往右讀成一整段生命與血脈故事

### 3.1.1 v2 視覺母型

依據目前 proof，推薦 v2 母型如下：

- 左側：現世戰場 / 人世場景中的名將表象面
- 右側：月夜、祖靈遺跡、魂霧場景中的血脈面 / 英靈面
- 中央：古老 `命鏡裂隙`
- 中央下方：定心者
- 底部：5 格命運故事帶

這個骨架已能同時滿足：

- 覺醒頁
- 升星頁
- Loading 輪播畫面
- 未持有武將命格展示

### 3.2 主視覺比例

- 左右雙人物加總為畫面主要視覺重量的 `70%~75%`
- 中央命鏡裂隙為主舞台焦點，不可小到失去存在感
- 中央定心者應小於主角色，定位為 `儀式核心`，不是第三主角
- 下方故事帶高度約佔全畫面 `18%~24%`

## 4. 視覺語言

### 4.1 雙面命格

- 左側為 `表象面`
  - 歷史名將、現世自我、公開人格
- 右側為 `血脈面`
  - 祖靈映像、命格之影、內在血脈人格

兩側不是硬切比較表，而是：

- 彼此互相滲透
- 中央以光霧、裂隙、紋樣、氣流、墨痕融合
- 左邊有一絲血脈面痕跡
- 右邊仍保有同一人輪廓與神情
- 左側可以保有較真實的戰場旗幟、塵土、兵勢
- 右側可以保有月光、遺跡、魂霧、祖紋殘頁
- 但兩邊應在中線附近自然漸層，不可像比較表硬切 50/50

### 4.2 命鏡裂隙

中央不做科幻傳送門，而是偏：

- 古老命格之門
- 靈魂鏡像裂口
- 祖靈共鳴裂隙
- 玉鏡般的命運之門

避免：

- 科技 UI 圓環
- 賽博光圈
- generic fantasy portal
- 純發光特效大過內容本體

### 4.3 定心者

中央下方的定心者是關鍵辨識點。

其意義：

- 一具肉身承載兩面命格
- 需透過定心 / 修練 / 覺悟，才得以融合靈魂

視覺原則：

- 可為男性或女性通用人形
- 盡量象徵化，不做具體角色肖像
- 以光暈顏色表現 rarity / spirit rank

### 4.4 國際化原則

這套畫面未來要能轉成國際版本，因此：

- 不可過度依賴重書法、篆刻、古董邊框
- 中國元素應作為可替換 `motif layer`，不是核心骨架
- 核心骨架應是國際也看得懂的：
  - self vs spirit self
  - mirror world
  - life story strip
  - awakening gate

---

## 5. 文案與資訊層

### 5.1 Loading 版

Loading 版不應放過多數值，建議只放：

1. 角色名稱
2. 稀有度 / 陣營 / 標記
3. 一句世界觀副標
4. 1~2 行 tips

### 5.2 覺醒 / 升星版

可增加：

1. 本次升星 / 二轉階段
2. 血脈面名稱或覺醒稱號
3. 解鎖能力關鍵詞
4. 命紋靈獸或祖紋命篆簡述

---

## 6. 下方命運故事帶

### 6.1 格數

- 建議固定為 `5 格` 或 `6 格`
- 由左往右閱讀
- 必須能讀成一條完整故事
- 目前 v2 母型優先採 `5 格`

### 6.2 敘事節奏

建議結構：

1. 左 2 格：歷史人生 / 公開人格
2. 中 1 格：命運轉折 / 血脈召喚 / 內心崩裂
3. 右 2~3 格：血脈真相 / 祖靈共鳴 / 英靈延續

目前 v2 proof 建議優先節奏：

1. 少年 / 初志
2. 歷史功業 / 關鍵戰役
3. 崩裂、重傷、命運轉折
4. 定心 / 覺悟 / 命紋亮起
5. 英靈延續 / 血脈面現身

### 6.3 動態建議

- 預設靜態可讀
- 可選擇做 `極慢速橫向漂移`
- 但不可犧牲閱讀性
- 載入時間較短時，優先保留靜態，不強推動畫

---

## 7. Tips 文案池

Loading 過場下方 tips 建議分成三池輪播：

1. `玩法提示`
   - 例如：重複抽到同名武將時，可選擇復生重練或血脈灌注
2. `歷史趣聞`
   - 例如：趙雲一生以忠勇著稱，長坂之戰奠定其傳奇形象
3. `血脈傳聞`
   - 例如：相傳趙雲體內沉眠龍膽之脈，越在絕境中越能顯現真正命格

---

## 8. 未持有角色展示原則

這套過場非常適合展示尚未取得的武將。

原則：

1. 允許 loading 輪播尚未持有角色
2. 未持有角色應額外標示：
   - `未結識`
   - `命格未開`
   - `尚未得其英靈`
3. 不直接劇透全部數值，但可保留世界觀誘因與角色魅力
4. 目的在於提升：
   - 收藏慾
   - 抽卡動機
   - 對角色故事的好奇心

---

## 9. 視覺禁區

### 9.1 不要做成

1. 普通角色資訊頁加 ghost overlay
2. 科技基因實驗室
3. 恐怖裂臉污染
4. 西方奇幻召喚獸頁
5. 單純海報圖，沒有 UI 邏輯

### 9.2 必須避免

1. 左右切太硬，像比較表
2. 右側血脈面只是 generic 藍色靈魂武將
3. 中央門太發光 fantasy，缺乏古老感
4. 下方故事格數太少，無法形成完整敘事
5. 定心者過大，搶走雙主角重量

---

## 10. 與其他 UI 的分工

### 10.1 與日常人物頁的分工

- 日常人物頁：
  - 人物屬性與角色本體為主
  - 血脈元素為輔
- 血脈命鏡過場：
  - 靈魂、命格、血脈與世界觀為主
  - 資訊效率退後，情緒與品牌辨識前置

### 10.2 與故事抽屜的分工

- 日常頁的故事抽屜偏 `可閱讀補充`
- 血脈命鏡的故事帶偏 `情緒與命運縮影`

## 12. 驗收標準

### 12.1 世界觀辨識

- 拿掉 Logo 後，觀者仍能說出：
  - 這是同一個角色的雙面命格
  - 這個遊戲和血脈、祖靈、傳承有關

### 12.2 商品吸引力

- 未持有角色放上去時，能有效提升：
  - 想抽
  - 想了解故事
  - 想收集角色

### 12.3 國際化彈性

- 更換配色、符號 layer、人物畫風後，骨架依然成立
- 不可建立在過度本土、只能中文世界理解的圖像隱喻上
