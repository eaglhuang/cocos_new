# 正式版劇本文案

## 1. 文件目的

本文件收錄新手開場的正式上線文案，並保留原始構想中具備趣味性與記憶點的台詞作為備選庫，方便後續選擇或過濾。

正式版文案的目標是：

- 語氣一致
- 可直接上線
- 保留情懷
- 保留角色辨識度
- 避免過度口語化或過度戲謔

### 文風原則

- 必須保留文化與現代、古代之間的反差感
- 台詞可以帶玩笑感，但不能失去角色氣質
- 不能寫成純說明書或純抒情文，避免讓玩家看了打瞌睡
- 每段對白都應該至少保留一個記憶點或畫面感
- 若某句話沒有節奏、沒有畫面、沒有情緒，就不應該放進正式版
