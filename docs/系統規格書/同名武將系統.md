# 同名武將系統規格書

> **來源文件**：賽馬娘模式：平行實體優缺點.md、趙雲分身與神聖趙雲包裝.md

---

## A. 系統描述

同一位名將可有多個版本（如「常山趙子龍」「白馬趙雲」「神聖趙雲」），每個版本是獨立 Template_ID，但共用 Bloodline_ID。版本差異來自不同的初始因子、戰法、虛擬祖先配置。「名號限定制」確保同一隊伍中不可放入同 Bloodline_ID 的武將，避免 P2W 疊加。

同名卡不再只是傳統碎片來源，而是正式分流成兩條用途：

- `復生重練`：讓同名名將再以人身歸來，並可插回既有英靈血脈
- `血脈灌注`：直接灌入既有血脈卡 / 英靈虎符，提升血脈濃度與對應軍勢強度

// 補充說明：你可以持有多個版本（例如「電趙雲」與「火趙雲」），但編隊階段系統會以 `Bloodline_ID` 作為不可同隊的判定依據，因此這兩個版本不能同時上戰場（符合「同名不可同隊」的硬規則）。

---

## B. 系統目的

1. **轉蛋多版本**：同名將不同版本驅動持續抽取
2. **收藏驅動**：收集同一武將的所有版本
3. **血統多樣性**：不同版本=不同虛擬祖先=不同子嗣方向
4. **公平限制**：名號限定制防止 P2W 疊加
5. **重複卡戲劇性**：玩家抽到同名卡時，得到的是「復生」還是「養魂」的抉擇，而不只是碎片

| 商業點 | 說明 |

## F. 公式相關

### 名號限定制
```
if (Party.any(g => g.Bloodline_ID == newGeneral.Bloodline_ID)):
    reject("同名武將已在隊伍中！")
```

### 戰魂寄宿
```
Dead_General → Soul_Fragment
Soul_Fragment.TP = Dead_General.Total_TP × 0.2
// 可注入同 Bloodline_ID 的其他版本
Target_General.TP += Soul_Fragment.TP
```

### 血脈歸宗
```
// 消耗「歸宗令」
Merge(V1_Gene, V2_Gene) → V1 獲得 V2 的 1 個特殊因子
// V2 不消失，但該因子標記為「已歸宗」
```

### 復生重練與血脈灌注
```
if (Duplicate_Draw):
    Choice = REVIVE_RETRAIN | BLOODLINE_INFUSION

if (Choice == REVIVE_RETRAIN):
    NewBody = CreateSameHeroTemplate(Bloodline_ID)
    NewBody.Embedded_Spirit_Card = ExistingSpiritCard
    NewBody.Rebirth_Count += 1

if (Choice == BLOODLINE_INFUSION):
    ExistingSpiritCard.Bloodline_Level += 1
    ExistingSpiritCard.Spirit_Power += DuplicateCardValue
    ExistingSpiritCard.Linked_Tally_Level += 1
```

## I. Data Schema 需求

### 本機端 (Client)
```json
{
  "General_Version": {
    "Template_ID": "Zhao_Yun_V3_Holy",
    "Bloodline_ID": "BL_ZhaoYun",
    "Display_Name": "神聖趙雲",
    "Version_Title": "聖光降臨",
    "Unique_Genes": [
      {"Type":"White","ID":"WHITE_HOLY_01","Level":5}
    ],
    "Soul_Fragments": [
      {"Source_UID":"G001","TP_Bonus":120}
    ],
    "Bloodline_Level": 3,
    "Spirit_Tally_Level": 2,
    "Rebirth_Count": 1
  }
}
```

### Server 端 — 版本表
| 欄位 | 型別 | 說明 |
|---|---|---|
| Template_ID | VARCHAR(32) PK | 版本唯一碼 |
| Bloodline_ID | VARCHAR(16) FK | 血源 ID（共用） |
| Version_Title | VARCHAR(32) | 版本稱號 |
| Initial_Genes_JSON | TEXT | 該版本初始因子 |
| Release_Date | DATE | 上架日 |
| Is_Limited | BOOLEAN | 是否限定 |

## J. 名詞定義

| 名詞 | 代碼/Key | 定義 |
|---|---|---|
| Template_ID | Template_ID | 武將版本唯一碼 |
| Bloodline_ID | Bloodline_ID | 同名武將共用的血源識別碼 |
| 名號限定制 | Name_Restriction | 同隊不可放同 Bloodline_ID |
| 戰魂寄宿 | Soul_Transfer | 死後 TP 轉移到同名其他版本 |
| 血脈歸宗 | Bloodline_Merge | 合併兩版本特殊因子 |
| 歸宗令 | Merge_Token | 執行血脈歸宗的道具 |
| 版本稱號 | Version_Title | 同名武將不同版本的標題 |
| 復生重練 | Revive_Retrain | 讓同名英雄再度以人身歸來，並承接既有英靈卡 |
| 血脈灌注 | Bloodline_Infusion | 把重複卡直接灌入既有血脈卡 / 英靈虎符，提升血脈濃度 |
| 血脈卡 | Spirit_Card | 名將死後或被提煉後留下的血脈 / 英靈載體 |
