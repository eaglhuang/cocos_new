# 戰場適性系統規格書

> **來源文件**：五色因子與爆發力公式設計.md、因子血統論系統規格書.md、賽馬娘式因子與戰法系統.md

---

## A. 系統描述

戰場適性系統管理三大維度的適性值：**兵種適性**（騎/步/弓/機械）、**地形適性**（平原/山地/水域/林地/沙漠）、**天氣適性**（晴天/雨天/大霧/大風/夜戰/雷暴）。適性由粉色因子（💗）與白色因子（⚪）決定，影響武將在不同戰場環境下的攻防、移動力、命中率等數值。

> **Unity 對照**：類似 Terrain Modifier System + Unit Affinity Matrix。粉色因子驅動的適性矩陣類似 Navigation Mesh 的 Area Cost 概念——不同適性等級影響不同地形上的移動/戰鬥效率。

---

## B. 系統目的

1. **策略多樣性**：不同戰場環境需要不同適性的武將，避免「一隊通關」
2. **因子價值化**：粉色因子的具體價值在戰場適性中體現，驅動配種策略
3. **地圖特色**：每張戰場地圖的地形與天氣組合創造獨特挑戰
4. **長線養成**：培養不同適性的武將組合需要多代血統累積
5. **差異化體驗**：同一武將在不同地形/天氣下有截然不同的表現

## F. 公式相關

### 兵種適性

| 兵種 | 代碼 | 強勢地形 | 弱勢地形 |
|---|---|---|---|
| 騎兵 | CAVALRY | 平原（移動 +2） | 山地（移動 -2）、林地（移動 -1） |
| 步兵 | INFANTRY | 山地（防禦 +30%） | 平原（易被騎兵衝鋒） |
| 弓兵 | ARCHER | 林地（隱蔽 +20%） | 水域（射程 -1） |
| 機械 | SIEGE | 平原（射程最大化） | 全地形移動力最低 |

### 適性等級定義

| 等級 | 代碼 | 數值修正 | 粉色因子需求 |
|---|---|---|---|
| S | 最強 | 攻防 +20%、移動 +1、命中 +15% | 因子 ★3 |
| A | 擅長 | 攻防 +10%、命中 +10% | 因子 ★2 |
| B | 普通 | 無修正 | 因子 ★1 |
| C | 不擅長 | 攻防 -10%、移動 -1 | 無因子 |
| D | 極差 | 攻防 -25%、移動 -2、命中 -15% | 排斥因子 |

### 地形適性效果

| 地形 | 說明 | S 級效果 | D 級效果 |
|---|---|---|---|
| 平原 | 開闊地帶 | 騎兵極速、全軍加速 | 無掩蔽易受傷 |
| 山地 | 崎嶇地形 | 防禦堡壘、弓兵高處優勢 | 移動困難、騎兵受限 |
| 水域 | 河流湖泊 | 可執行合攻、水戰專精 | 移動力歸 1、攻擊減半 |
| 林地 | 森林樹叢 | 隱身伏擊、迴避 +30% | 視野受限、被伏擊機率高 |
| 沙漠 | 荒漠地帶 | 耐久戰優勢、精力消耗減半 | 每回合扣精力、移動減速 |

### 天氣適性效果

| 天氣 | 代碼 | 正面效果（高適性） | 負面效果（低適性） |
|---|---|---|---|
| 晴天 | SUNNY | 基礎狀態，無特殊 | 無 |
| 雨天 | RAINY | 火攻無效、水域擴大 | 弓兵命中 -20%、機械故障率 +10% |
| 大霧 | FOG | 免疫=100% 命中 | 視野 -3 格、命中 -30% |
| 大風 | WINDY | 火攻傷害 ×2 | 弓兵射程 -2、遠程命中 -15% |
| 夜戰 | NIGHT | 夜戰加成=暴擊 +20% | 全軍命中 -25%、士氣 -10 |
| 雷暴 | THUNDER | 雷擊免疫 + 士氣不降 | 機械 30% 直接損毀、全軍士氣 -20 |

### 適性計算公式

```
// 最終戰鬥修正
terrain_mod = TERRAIN_TABLE[general.terrain_aptitude][current_terrain]
weather_mod = WEATHER_TABLE[general.weather_aptitude][current_weather]
troop_mod = TROOP_TABLE[general.troop_aptitude][assigned_troop_type]

final_attack = base_attack × (1 + terrain_mod.atk) × (1 + weather_mod.atk) × (1 + troop_mod.atk)
final_defense = base_defense × (1 + terrain_mod.def) × (1 + weather_mod.def) × (1 + troop_mod.def)
final_move = base_move + terrain_mod.move + troop_mod.move
final_accuracy = base_accuracy + terrain_mod.acc + weather_mod.acc
```

### 粉色因子→適性轉換

```
aptitude_level = APTITUDE_MAP[pink_gene.level]
// ★0 → D, ★1 → C, ★2 → B, ★3 → A
// 特殊：14人血統中 ≥5 人攜帶同一適性因子 → 升一級（最高 S）
```

### 競技場積分公式（名將挑戰賽用）

> **來源**：遊戲機制優化與策略建議.md、賽馬娘機制轉化三國戰場.md

在名將挑戰賽等 PvP 競技場景中，武將的參考戰力分數使用以下複合公式計算：

```
Score = Σ[(屬性總和 × 地形適性倍率) + (戰法點數 × 剋制係數)] × 存活係數

// 地形適性倍率（對應等級）
TERRAIN_MULTIPLIER = { S: 1.5, A: 1.2, B: 1.0, C: 0.85, D: 0.70 }

// 存活係數 = 當前HP / MaxHP（0.0~1.0）
SURVIVE_FACTOR = current_hp / max_hp

// 說明：
// - 屬性總和 = STR + INT + LEA + LUK（不含 HP/DEF 等防守屬性，側重輸出面）
// - 戰法點數 = 已配備的戰法等級總和
// - 剋制係數 = 戰法對當前地形/天氣的克制倍率（1.0~1.5）
```

**設計意圖**：此積分反映武將在「特定戰場環境」下的綜合貢獻，而非純粹強弱排名，促使玩家根據賽季場地特性針對性培育。

### 五維屬性-戰場維度對應（賽馬娘對照）

> **來源**：賽馬娘機制轉化三國戰場.md

| 賽馬娘屬性 | 三國對應屬性 | 戰場功能 |
|---|---|---|
| 速度（Speed） | 機動力（行動力 AP） | 決定每回合行動次數與先手優勢 |
| 耐力（Stamina） | 糧草續航 | 連戰能力，長途征戰的持久性 |
| 力量（Power） | 破陣威壓（STR） | 對陣型/地形的穿透與突破力 |
| 毅力（Guts） | 軍心堅毅 | 抗控制與意志力（避免潰退） |
| 智力（Wits） | 戰術謀略（INT） | 戰法技能的觸發率與覆蓋廣度 |

## J. 名詞定義

| 名詞 | 定義 |
|---|---|
| 兵種適性 | 武將指揮騎/步/弓/機械部隊的效率等級（S~D） |
| 地形適性 | 武將在不同地形環境（平原/山地/水域/林地/沙漠）下的戰鬥效率等級 |
| 天氣適性 | 武將在不同天氣條件（晴/雨/霧/風/夜/雷）下的戰鬥效率等級 |
| 粉色因子（💗） | 決定兵種與地形適性的遺傳因子 |
| 白色因子（⚪） | 提供天氣抗性與特殊天賦的遺傳因子 |
| 適性等級 | S（極強）到 D（極差）的五級評定 |
| 環境偵察 | 出戰前查看戰場地圖地形分佈與天氣預報的功能 |
| 天氣變化 | 戰鬥進行中天氣隨回合自動切換的機制 |
| 適性修正 | 根據適性等級對攻防/移動/命中等數值的加減乘除調整 |
