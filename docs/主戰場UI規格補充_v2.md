## Zone 1：TopBar（改造）

### 佈局參考：圖三風格

```
┌─────────────────────────────────────────────────────────────────────┐
│ [趙雲肖像]  趙雲        第 5 回合：我方行動          曹操  [曹操肖像] │
│  蜀國上將   ■■■■■■■■░░ FORTRESS HP ░░■■■■■■■■   魏國主將    │
│                  DP: 1500                  [SP ▓▓▓░ 60%]    │
└─────────────────────────────────────────────────────────────────────┘
```

### 精確規格

| 屬性 | 值 |
|---|---|
| **Widget** | `top: 0, left: 0, right: 0` |
| **高度** | 88px（含 ProgressBars 合併） |
| **背景** | 半透明深色面板 `rgba(8, 8, 12, 0.92)` + 底部金色分隔線 1px `#D4AF37` |

### 子節點結構

#### 左側：我方主將區（PlayerGeneral）
- **Widget**：`left: 16px, verticalCenter: 0`
- **佈局**：水平排列，間距 8px
- **內容**：
  - **肖像框**（PlayerPortrait）：56×56px 圓形，2px 金色邊框 `#D4AF37`
    - 內部載入主將頭像 SpriteFrame
  - **文字區**：
    - **名稱**（PlayerName）：字號 20px，粗體，顏色 `#FFE088`
    - **稱號**（PlayerTitle）：字號 12px，顏色 `#888`，如「蜀國上將：趙雲」

#### 中央：回合與資源區（CenterInfo）
- **Widget**：`horizontalCenter: 0, verticalCenter: 0`
- **佈局**：垂直排列，間距 4px，水平置中
- **內容**：
  - **回合標題**（TurnTitle）：字號 18px，粗體，顏色 `#D4AF37`
    - 格式：「第 {N} 回合：{我方行動/敵方行動}」
  - **陣地血量條區**（FortressBars）：
    - 水平排列，總寬 480px，高 16px
    - 左半藍色（我方 HP）：`#3A8FD9`，由左向右填充
    - 右半紅色（敵方 HP）：`#D94A3A`，由右向左填充
    - 中央 VS 圖示或分隔線
  - **糧草顯示**（DpLabel）：字號 14px，顏色 `#F0E68C`
    - 格式：「糧草: {N}」

#### 右側：敵方主將區（EnemyGeneral）
- **Widget**：`right: 16px, verticalCenter: 0`
- **佈局**：水平排列（反向），間距 8px
- **內容**：
  - **文字區**（右對齊）：
    - **名稱**（EnemyName）：字號 20px，粗體，顏色 `#f09090`
    - **稱號**（EnemyTitle）：字號 12px，顏色 `#888`
  - **肖像框**（EnemyPortrait）：56×56px 圓形，2px 紅色邊框 `#D94A3A`
    - **敵方 SP 條**：在肖像框下方或環繞肖像，寬 56px，高 4px
      - 填充色 `#9b6dff`（紫色），背景 `rgba(0,0,0,0.5)`
      - 用途：僅顯示敵方 SP 百分比

### 與原版差異
- ❌ 移除：ProgressBars 中的 PlayerSpBar / EnemySpBar（SP 不再放 TopBar）
- ❌ 移除：勝場數 / 關場數顯示
- ✅ 新增：雙方主將肖像 + 名稱 + 稱號
- ✅ 合併：TopBar (64px) + ProgressBars (24px) 合為單一區域 (88px)
- ✅ 保留：陣地血量條、糧草、回合計數

## Zone 3：虎符卡片列（重設計）

### 佈局參考：圖一左側卡片

```
┌──────────────┐
│  ⚔48    ❤47  │  ← 攻擊/生命數值（overlay）
│              │
│  [兵種美術圖] │  ← 全幅底圖
│   虎豹騎     │
│              │
│ [技能1][技能2]│  ← 技能 icon 行（可選）
│     消耗: 10  │
└──────────────┘
      ↑ 品質外框（金/紫/藍...）
```

### 精確規格

| 屬性 | 值 |
|---|---|
| **Widget** | `top: 88px, left: 0, bottom: 0` |
| **寬度** | 16%（≈307px @1920） |
| **背景** | `rgba(10, 10, 18, 0.85)` + 右側分隔線 `rgba(212, 175, 55, 0.25)` |
| **卡片數量** | 4 張垂直排列 |
| **卡片間距** | 8px |
| **內邊距** | `padding: 8px 10px` |

### 單張卡片結構（TigerTallyCard）

| 屬性 | 值 |
|---|---|
| **尺寸** | 寬：287px（列寬 -20px padding），高：按可用空間等分（≈228px @992px 可用高） |
| **圓角** | 8px |
| **底圖** | 兵種美術圖，`sprite-type: sliced` 或 `filled`，填滿整張卡片 |
| **品質外框** | 依稀有度（白/綠/藍/紫/金/紅/彩）套用不同邊框色，寬 2-3px |

#### 卡片內子節點（由上而下疊加 overlay）

1. **數值列**（StatsOverlay）：
   - **Widget**：`top: 4px, left: 8px, right: 8px`
   - **佈局**：水平排列，兩端對齊
   - **攻擊數值**（AtkLabel）：字號 16px，粗體，顏色 `#FF6B6B`（紅系）
     - 格式：「⚔{N}」或「{N}⚔」
     - 帶投影（shadow）確保在任何底圖上可讀
   - **生命數值**（HpLabel）：字號 16px，粗體，顏色 `#6BCB77`（綠系）
     - 格式：「❤{N}」或「{N}❤」
     - 帶投影（shadow）

2. **兵種名**（UnitName）：
   - **Widget**：`bottom: 28px, horizontalCenter: 0`
   - 字號 14px，粗體，顏色 `#FFFFFF`
   - 帶黑色描邊或底板確保可讀

3. **消耗標示**（CostBadge）：
   - **Widget**：`top: 4px, right: 4px` 或 `bottom: 4px, right: 4px`
   - 28×28px 圓形深色底 `rgba(0,0,0,0.7)`
   - 字號 14px，顏色 `#F0E68C`
   - 顯示糧草消耗值

4. **狀態遮罩**（DisabledMask）：
   - 灰度全覆蓋 `rgba(0,0,0,0.55)` + 「冷卻中」文字
   - 僅在 Disabled / Downgrade 狀態顯示

### 與原版差異
- ✅ 新增：攻擊 + 生命數值 overlay
- ✅ 改變：底圖由純色改為兵種美術圖（全幅填充）
- ✅ 保留：品質外框、消耗標示、狀態（Normal/Selected/Disabled/Downgrade）

| 屬性 | 值 |
|---|---|

| **鎖定狀態** | 條件不滿足時：灰度 + 🔒 圖示 + 不可點擊 |

### SP 填充環運作邏輯

```
SP 0%   → 外環全空白（僅底色環）
SP 50%  → 外環順時針填充至 180°（藍色）
SP 100% → 外環完整填滿 + 呼吸發光 + 文字切換為「發動」
點擊後  → SP 歸零，環清空，觸發奧義技能
```

- ✅ 結束回合 → 最大最顯眼按鈕

## 新增：WorldSpace 單位 HUD（棋盤格上方）

### 適用對象
- 所有棋盤上的單位（小兵 + 敵我主將）

### 小兵 HUD（Unit HUD）

```
┌─────────────┐
│ 999 步 999  │    ← 攻擊 兵種 生命
└─────────────┘
```

| 屬性 | 值 |
|---|---|
| **位置** | 單位模型頭頂（WorldSpace → Billboard，永遠面向攝影機） |
| **尺寸** | 約 120×28px（視距離自動縮放） |
| **背景** | `rgba(0, 0, 0, 0.6)` 圓角面板 |
| **文字格式** | `{攻擊值} {兵種字} {生命值}` |
| **兵種字** | 步 / 騎 / 弓 / 盾 / 槍（單字） |
| **攻擊值顏色** | `#FF6B6B`（紅色系） |
| **兵種字顏色** | `#D4AF37`（金色） |
| **生命值顏色** | `#6BCB77`（綠色系） |
| **字號** | 14px，粗體 |
| **陣營區分** | 我方 HUD 底端有 2px 藍色底線 `#3A8FD9`；敵方有 2px 紅色底線 `#D94A3A` |

### 範例

| 單位 | 顯示 |
|---|---|
| 我方步兵 | `999 步 999`（藍底線） |
| 敵方騎兵 | `777 騎 555`（紅底線） |
| 我方弓兵 | `666 弓 888`（藍底線） |

### 主將 HUD（General HUD）

```
┌─────────────┐
│ ■■■■■■░░░░  │  ← HP Bar
│   張飛       │  ← 名稱
│ 999 將 999  │  ← 攻擊 兵種 生命
│ [⬆][💨][🛡] │  ← Buff icons（可選）
└─────────────┘
```

| 屬性 | 值 |
|---|---|
| **HP Bar** | 寬 100px，高 8px，我方藍 `#3A8FD9`，敵方紅 `#D94A3A` |
| **名稱** | 字號 14px，粗體，顏色 `#FFE088`（金色） |
| **數值列** | 同小兵 HUD 格式，但兵種字顯示「將」 |
| **Buff 列** | 小方塊 icon 最多 4 個，16×16px |
| **重要性** | HP 歸零 = 戰敗，因此 HP Bar 需常駐顯示 |

### 顯示規則
- 小兵 HUD：預設顯示；可由設定選項隱藏（僅選中時顯示）
- 主將 HUD：**常駐顯示**（HP 歸零 = 輸贏條件）
- 受傷時：HP 數值閃紅 0.3s
- 陣亡時：HUD 灰度化 + 「陣亡」文字
