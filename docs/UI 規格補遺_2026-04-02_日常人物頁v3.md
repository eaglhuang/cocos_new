# UI 規格補遺 2026-04-02 日常人物頁 v3

> 對應 proof：`日常人物頁 v3`
> 主卡：`UI-2-0054`

## 版型母型

`General Detail` 最新母型採：

1. 左角色立繪
2. 右主資訊卡
3. 下方 6 格故事帶

這條線正式定義為 `日常人物頁`，不與 `血脈命鏡` 混用。

## 內容主次

1. 角色本體是第一主角。
2. 血脈區是第二層閱讀主題，不可壓過人物本體。
3. 故事帶是特色內容層，需保留足夠高度與閱讀節奏。

## 建議元件

1. `general-detail-v3-root`
2. `general-portrait-carrier`
3. `general-info-main-card`
4. `general-bloodline-subcard`
5. `general-bloodline-crest`
6. `general-story-strip`
7. `general-story-cell`

## 前端可行性

1. 高：整體結構清楚，適合目前 `layout / skin / screen` 拆分。
2. 高：人物、徽記、故事圖都能先用靜態圖填入。
3. 中：血脈區需要後續 family 化，避免每位角色都變成一次性手工拼裝。
4. 低：不建議在此頁強上動態光效與自動滾動故事。
