# 圖1 — 武將詳情 標註說明

目標：產出一張標註圖（PNG），在截圖上標示 9-slice、bleed、content-safe、CTA 三態區域與文字描邊需求。

標註項目：
- 9-slice 角區：標示 20~24 px 的角區範圍（用紅框）
- 9-slice border：標示 1~2 px 金屬邊（黃線）
- Bleed（暖色暈邊）：標示框外 2~4 px 的 glow 範圍（橙色半透明）
- Content safe：標示內容安全帶 4 px（綠線），避免文字/icon 貼邊
- 半透明蒙版：標示蒙版漸變方向與最小/最大 opacity（50%~80%）
- CTA：標示按鈕正常/pressed/disabled 三態區（含 1px 縮放差異）
- 文字描邊：註記深底上白色/金色文字需 1~2 px 黑描邊

輸出檔名建議：`圖1_武將詳情_annotated.png`

備註：若原始截圖解析度不是 1920×1080，請先等比放大/縮小到設計基準再標註。
