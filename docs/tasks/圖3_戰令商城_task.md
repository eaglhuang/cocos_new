# 任務卡 — 圖3：戰令商城（草案）

摘要：戰令介面為深淺嵌套示例，需確保大面積淺色面板與內嵌深色格子的視覺階層與互動一致性。

改善項目：
- 主內容面板（淺色羊皮紙）設置 9-slice 與圓角（4px），邊框 1px 金色
- 進度條樣式（容器與填充）需支援動態漸層（綠 → 金）與可視化動畫
- 免費/進階物品格樣式一致（深色底），鎖定狀態加鎖遮罩與灰化
- CTA（購買 / 領取）顯著且符合金色 CTA 規範（3~4px 邊框 + glow）

驗收條件：
- 主面板在縮放下無紋理重複或拉伸痕跡
- 進度條在不同 fill 比例下顏色過渡平滑且無斷層
- 鎖定物品顯示遮罩且無誤觸

交付物：
- 主面板 9-slice sprite + bleed
- 進度條 sprite / shader 範例
- 測試視訊或 GIF（進度條從 0→100）

預估工時：UI 美術 1~1.5 日；工程整合 1 日

負責人（建議）：UI 美術 / UI 工程 / QA
