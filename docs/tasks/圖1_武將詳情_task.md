# 任務卡 — 圖1：武將詳情（草案）

摘要：依據 `docs/UI參考圖品質分析.md (doc_ui_0051)` (doc_ui_0051) 的拆解，修正武將詳情頁的框體、9-slice、文字描邊與 CTA 行為，確保在不同背景明暗下可讀性與層次感一致。

改善項目：
- 深色金屬框（左側固定戰法、右側數值面板）：建立 9-slice（角區 20~24px）、bleed 2px、內容安全帶 4px
- 半透明暗色蒙版（英雄立繪後方）：確認 opacity 為 50~80% 並在左側做漸變羽化
- Tab（數值/將魂）：啟用態為金色填充、閒置為深灰 + 細金邊（高 44~48px）
- 試玩角色金色 CTA：實作 pressed/disabled 三態（pressed 縮 1px，disabled opacity 0.5）
- 文字描邊：深色底上的白/金字加入 1~2px 黑色描邊，檢查在 hero 立繪明亮區是否斷字

驗收條件：
- 在 1920×1080 預覽，左/右面板角落無拉伸異常，金屬角裝飾完整
- Tab 切換時啟用態明顯（金色），高度與間距一致
- CTA 在按下時有視覺回饋；disabled 狀態灰化且無互動
- 長文字段落在淺/亮背景下仍可讀（無字被背景亮斑覆蓋）

交付物：
- 9-slice sprite（deep metal frame）+ bleed sprite
- skin JSON 範例（示範金邊、描邊參數）
- 預覽截圖（正常 / pressed / disabled）

預估工時：UI 美術 0.5~1 日；工程整合 0.5~1 日

負責人（建議）：UI 美術 / UI 工程 / QA
