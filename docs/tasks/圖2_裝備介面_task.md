# 任務卡 — 圖2：裝備介面（草案）

摘要：針對裝備詳情頁的物品格、淺色統計區與菱形 Tab 做可拉伸、可讀性與互動性的驗證。

改善項目：
- 裝備詳情卡使用 deep metal frame（9-slice 20~24px）；內部分割線顏色統一為 `#2A2A2A`
- 裝備 icon 格（SSR 等級示例）：固定 80~100px，邊框依稀有度變色，badge 大小與位置一致
- 裝備提供能力區（淺色羊皮紙）：確認不使用描邊，文本用深褐色並加 12~16px padding
- 菱形 Tab：以獨立 sprite 實作，啟用態帶外 glow（gold）且圖標色調為深色
- 「全部卸除」破壞按鈕：實作 confirm 流程與視覺紅色邊框 2~3px

驗收條件：
- 所有格子在不同解析度下 icon 無失真，稀有度邊框顏色正確
- 淺色統計區在長段文字下無可讀性問題
- 菱形 Tab hover/active 行為一致且無拉伸

交付物：
- icon sprite（含 badge）
- skin JSON（格子、淺色區）
- 測試截圖（選中/未選中/鎖定）

預估工時：UI 美術 1 日；工程整合 1 日

負責人（建議）：UI 美術 / UI 工程 / QA
